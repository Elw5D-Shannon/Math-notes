# 分式环

## 基本概念

- **乘法封闭集**：对环乘法封闭的环子集（$a,b\in S\red\Rt ab\in S$）
  - **实例**：
    - 幺环的非零因子全集（如整环上的非零元）
    - 幺环的单位全集 $\U$
    - 交换环上的素理想全集 $\P$ 及其补集 $\I\j\P$
- **（定理4.2）约分等价**：
  - 设 $S$ 是交换环 $R$ 的乘法封闭集
  - 则 $R\times S$ 上存在如下的等价关系：$$\exists s_0\in S\ 使得\ s_0(rs' - r's) = 0 \red\Rt (r,s)\sim (r',s')$$
    - **本质**：相对于零因子可约
    - **实例**：
  - **简化定理**：若 $R$ 是整环，$0\notin S$，则 $(r,s)\sim (r',s') \LR rs'-r's=0$
  - **约分等价类**：$r/s$
  - **全体约分等价类**：$S^{-1}R$
    - **唯一性条件**：若 $0\in S$，则约分等价类只有一个
    - **可约性**：$(tr)/(ts) = r/s\quad (\forall s,t\in S，r\in R)$
      - **证明**：定义中使用乘法分配律提公因式 $t$ 即可
    - **实例**：$\Q = \N\times \dkh{\N-\{0\}}$
      - 约分等价类：约分后相等的分数
- **（定理4.3）分式环**：
  - 设 $S$ 是交换环 $R$ 的乘法封闭集
  - 则全体约分等价类 $S^{-1}R$ 构成含幺交换环
    - 加法 $r/s+r'/s' = (rs'+r's)/(ss')$
    - 乘法 $(r/s)(r'/s') = (rr')/(ss')$
    - 幺元 $r/r$，零元 $0/r$
    - **证明**：
      - **加法封闭性**：由乘法封闭性，$rs' + r's\in R，ss'\in S$，
        - 再取两个另外的等价元。由等价定义，可设 $\begin{cases} s_2(rs_1-r_1s) = 0 \\ s_3(r's_1'-r_1's') = 0 \end{cases}$
        - 则此时 $s_2s_3\Big[ (rs'+r's)s_1s_1' - (r_1s_1'+r_1's_1)ss' \Big] = 0$
        - 因此，$(rs' + r's)/ss' = (r_1s_1'+r_1's_1)/s_1s_1'$
        - 综上，加法封闭
      - 同上，乘法也封闭
- **分式整环**：若 $R$ 无零因子，$0\notin S$，则 $S^{-1}R$ 是整环
  - **证明**：
    - 由无零因子性，$r/s = 0/s \LR \exists s_1，s_1s(r-0) = 0 \LR r=0$
    - 从而 $(r/s)(r'/s') = 0 \LR rr' = 0$，当然也无零因子
  - **反例**：逆命题不成立
    - $S = \set{2,4}，R = Z_6$，则 $S^{-1}R\cong Z_3$ 是整环，但 $R$ 有零因子 $3$ 
- **完全分式环**：若 $S\neq\varnothing$ 是所有无零因子的非零元，则 $S^{-1}R$ 称为完全分式环
  - <font color='chartreuse'>若 $R$ 无零因子，则其完全分式环是域</font>
- **分式域**： $R$ 无零因子，$S$ 是非零元全集，则 $S^{-1}R$ 是域
  - **逆元**：$(r/s)^{-1} = s/r$

## 分式环的同态

- **（定理4.4）分式同态**：
  - 设 $S$ 是交换环 $R$ 的乘法封闭集
  - 则 $\forall s\in S，\p_S: R\to S^{-1}R，r\mapsto (rs)/s$ 都是环同态，且 $\p_S(s)$ 是可逆元
    - **证明**：
      - **乘法公式**：易得
      - **良定义**：对于 $\forall s,s'\in S$，都有 $\p_S(r) = (rs)/s = (rs')/s' = \p_{S}(r)$，从而同态唯一，是良定义的
      - **可逆性**：易得 $\p_S(s)^{-1} = s/s^2$，从而可逆
  - **单射条件**：若 $S$ 不含零元和零因子，则其为单同态
    - **证明**：
      - 不为单同态时，$\exists \p_S(r) = (rs)/s = 0$，即存在 $s_1\in S$ 使得 $rs^2s_1 = 0$
      - 再由 $S$ 无零因子性，$s^2s_1 \neq 0$，从而只能是 $r = 0$
    - **推论**：整环可嵌入其分式域<font color='chartreuse'>（整环可看作分式域的子环）</font>
      - **实例**：$\Z$ 可嵌入 $\Q$
  - **双射条件**：若 $R$ 含幺元，$S$ 是可逆元全集，则其为同构
    - **证明**：只需证明满射性即可。
      - 任取 $r/s$，由于 $s$ 可逆，得存在 $\p_S(rs^{-1}) = r/s$。由任意性即得满射性
    - **推论**：域 $F$ 同构于其完全分式环<font color='chartreuse'>（域的分式域是自身）</font>
- **（定理4.5）整环表示定理**：设 $S$ 是交换环 $R$ 的乘法封闭子集，$T$ 是含幺交换环
  - 若环同态 $f:R\to T$ 满足像集为均可逆元<font color='chartreuse'>（$f$ 将环嵌入到单位群中）</font>
  - 则存在唯一环同态 $\ol f:S^{-1}R\to T$ 使得 $\ol f\circ \p_S = f$
  - **证明**：
    - **存在性**：取 $\ol f(r/s) = f(r)f(s)^{-1}$ 即可
    - **唯一性**：反设还有 $g$，则由同态性，$g(\p_S(s)^{-1}) = g(\p_S(s))^{-1}$
      - 故 $g(r/s) = g\Big(\p_S(r)\p_S(s)^{-1}\Big) = g\Big( \p_S(r) \Big)g\Big( \p_S(s) \Big)^{-1} = f(r)f(s)^{-1} = \ol f$，从而得到唯一性
  - **理解**：$\ol f$ 是分子映射，$\p_S$ 是分式映射，$f$ 是原映射
    - 也就是说，把原像除上分母，然后将分子映射，本质和原映射相同
  - **推论**：分式环 $S^{-1}R$ 可被该性质完全表出<font color='chartreuse'>（其也可作为分式环的定义）</font>
    - **证明**：易得以上面的 $(f,T)$ 为对象，可组成范畴，设为 $\mc C$ 
      - 其中态射为 $g:T_1\to T_2，gf_1 = f_2$
        - 当 $g$ 是同构时，$(f_1,T_1) \sim (f_2,T_2)$
      - 易得 $(\p_S,S^{-1}R)$ 是泛对象，即任意对象都是其在某个态射下的像。再由泛对象唯一性，当知道 $f$ 和 $\ol f$ 时，即可唯一得到 $\p_S = \ol f^{-1}f$，从而可唯一得到 $S^{-1}R = \p_S(R)$
    - **理解**：
- **（推论4.6）域表示定理**：
  - 设 $R$ 是其分式域 $F$ 上的整环
  - 若 $E$ 是域，$f:R\to E$ 是单同态
  - 则存在唯一的域单同态 $\ol f:F\to E$ 使得 $\ol f|_R = f$
  - **证明**：
    - 设 $S$ 是 $R$ 非零元的集合，则 $S^{-1}R = F$，即存在 $\ol f:F\to E$
    - 易得其为单同态，则 $\p_S(R) = R$，故 $\ol f|_R = \ol f\circ \p_S = f$
  - **推论（夹逼性）**：任何包含 $R$ 的域 $E_1$ 都包含一个 $F$ 的同态 $F_1$，满足 $R\subset F_1\subset E$
    - **证明**：特例而已

## 分式环的理想

- **（定理4.7）分式环的理想**：设 $S$ 是交换环 $R$ 上的乘法封闭集
  - **理想传递性**：若 $I$ 是 $R$ 上理想，则 $S^{-1}I = \{a/s\}$ 是分式环上的理想
  - **理想保单调性**：若 $I\subset J$，则 $S^{-1}I\subset S^{-1}J$
  - **理想运算**：
    - **加法公式**：$S^{-1}(I+J) = S^{-1}I + S^{-1}J$
    - **乘法公式**：$S^{-1}(IJ) = (S^{-1}I)(S^{-1}J)$
    - **交公式**：$S^{-1}(I\cap J) = (S^{-1}I) \cap (S^{-1}J)$
  - **证明**：
    - 任取 $c_i/s\in S^{-1}R$
    - **加法公式**：$\sum\limits^n_{i=1} (c_i/s) = (\sum\limits^n_{i=1} c_i)/s$
    - **乘法公式**：$\sum\limits^m_{j=1} (a_jb_j/s) = \sum\limits^m_{j=1} (a_j/s)(b_js/s)$
    - **交公式**：$\sum\limits^t_{k=1} (c_k/s_k) = \Big(\sum\limits^t_{k=1} c_k\prod\limits^t_{i\neq k}s_i \Big)/(\prod\limits^t_{i=1}s_i)$
- **（定理4.8）分式理想无幺性**：
  - 设 $S$ 是含幺交换环 $R$ 上的乘法封闭集，$I\in\I_R$
  - 则 $S^{-1}I = S^{-1}R \LR S\cap I \neq \varnothing$
  - **证明**：
    - **充分性**：任取 $s\in S\cap I$，则 $1_{S^{-1}R} = s/s\in S^{-1}I$，从而由理想乘法封闭性，$S^{-1}R\subset S^{-1}I$。再互包即得 $S^{-1}I = S^{-1}R$
    - **必要性**：此时 $\p_S^{-1}(S^{-1}I) = R$，从而可设 $\p_S(1_R) = a/s$
      - 再由定义，$\p_S(1_R) = 1_Rs/s$。由等价性，$\exists s_1$ 使得 $s^2s_1 = ass_1$
      - 再由 $s^2s_1 \in S，ass_1\in I$，即得 $S\cap I \neq \varnothing$
  - **理解**：任意真理想均与 $S$ 不相交。否则由两个乘法封闭性，交集直接进化为整个环，也就不可能是真理想
    - 或者说，若真理想与 $S$ 相交，则交集在分式同态下的像是幺元。从而像是含幺的分式理想，其进化为整个分式环
  - **本质**：分式理想不含幺元的条件等价于 $S\cap I = \varnothing$
- **理想的收缩**：设 $J\in\I_R$，则 $\p_S^{-1}(J)$ 称为其收缩
  - <font color='chartreuse'>因为分式同态是嵌入（分式环是原环的扩张），故分式逆同态当然就是收缩了
  - 将 $R$ 看作分式环，此时只有 $rs/s$ 形式的元素在 $\p^{-1}_S$ 的支集中，即将所有 $r/s$ 形式的元素变为 $0$。环当然会变小
  - 当然了，选取不同的 $S$，分式的定义也会不同，所以仅仅只给出 $R$ 时，我们找不到一个固定标准来判断 $R$ 中哪些是分式，哪些是整数，也就无法写出分式逆同态的显式形式。
  - 但是呢，如果给出了 $R = S^{-1}I$，也即规定了 $R$ 中哪些元素是整数，此时就可以讨论一下它的性质</font>
  - **实例**：$\p^{-1}_{\N-0}(\Q) = \N$，消灭其中所有真分数

## 环的局部化

- **$R$ 在 $P$ 上的局部化**：已知含幺交换环 $R$ 和其上素理想 $P$，取乘法封闭集 $S = R-P$，则 $R_P = S^{-1}R$ 称为 $R$ 在 $P$ 上的局部化
  - **局部化理想 $I_P$**：$R_P$ 上的理想 $S^{-1}I$
  - **实例**：
    - $R = \Z，P = (2)，S = \set{2k+1}，R_P = \set{\dfrac{m}{2k+1}\mid m,k\in\N}$，其中可取某个 $I_P = \dfrac{3k}{2k+1}$
- **（定理4.11）局部化的性质**：设 $P$ 是含幺交换环 $R$ 上的素理想
  - **限制对应性**：存在（$P$ 中素理想）到（$R_P$ 上的素理想）的双射 $Q\mapsto Q_P$
    - **证明**：由于 $P$ 中素理想也是 $R$ 上素理想，由素理想一一对应性可得双射 $R\to R_P$，再取其限制，易得也是双射
    - **实例**：
      - 在上面的整数环局部化中，$(6)$ 是 $P = (2)$ 中素理想，故存在双射 $f:(6)\to \dfrac{(6)}{2k+1}，6m\mapsto \dfrac{6m}{2k+1}$
  - **局部最大性**：$P_P$ 是 $R_P$ 上唯一的最大理想
    - **证明（最大性）**：反设 $P_P\subset I_P$，即 $\dfrac{P}{R-P} \subset \dfrac{I}{R-P}$
      - 也即 $\exists a/s\in I_P-P_P$。但由于 $a\in I-P$，故 $a\in S$，即 $a\in S\cap I$，即 $I_P$ 含幺，退化为 $R_P$，矛盾
    - **证明（唯一性）**：
      - 反设 $M$ 也是 $R_P$ 上最大理想，则由含幺交换性。其为素理想。再由 $P$ 最大性，存在理想 $Q\subset P$ 使得 $M = Q_P$
      - 由分式理想单调性，$Q_P\subset P_P$。再由 $Q_P$ 最大性，只能是 $Q_P = P_P$，从而唯一
    - **理解**：
    - **本质**：这是我们人为地选取素理想 $P$，取人为定义的局部化分式环，使得 $P$ 的分式理想在其上最大，从而达到某些研究某些性质的目的

- **（定理4.13）局部环与可逆性**：含幺交换环 $R$ 的下列条件等价
  - $R$ 是局部环
  - 不可逆元均包含于某个真理想 $M$ 中
  - 不可逆元全集组成一个理想
  - **证明**：
    - $(3)\to (1)$：由于含可逆元的理想会退化为 $R$，得若不可逆元全集是理想，则其为唯一最大理想，从而是局部环
    - $(2)\to (3)$：同上得此时 $M$ 就是唯一最大理想

      - 反设不可逆元 $a\notin M$，由不可逆性 $(a)\neq R$，再由最大理想的包含性，$(a)\subset M$，从而 $a\in M$，矛盾
  - **本质**：因为可逆和理想存在使之退化的关系，故当唯一最大理想存在时，某些性质会暴露出来
