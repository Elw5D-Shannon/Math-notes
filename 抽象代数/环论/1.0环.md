# 一般环

- 最好只看看环的基本概念，然后学模和域，再回头从交换代数的角度学环
- 不少教材是先学域再学环的。实际上域的学习只需要环的概念，用不到多少环的定理，里面的向量空间知识也可以用高代的线性空间去类比。所以也可以先简单了解下环、模的相关概念后，直接去看域的部分。看完后再读环的部分，对环的认识会更清晰。当然，按顺序读也是可以的。

## 基本概念

- **环**：一个集合 $R$ 与两个封闭二元运算（加法、乘法），满足 $(R,+)$ 是阿贝尔群
  - **加法**：具有可逆性、交换律、结合律的运算
  - **乘法**：具有结合律、左右分配律的运算
    - **左右分配律**：$a(b+c) = ab + ac$、$(a+b)c = ac + bc$
    - <font color='chartreuse'>$(R,\cdot)$ 不一定是群，因为不一定有乘法幺元和逆元。而且，很有意思的是，对 $c\in R$，只有其不是素元时才有 $ba = c$（详见[欧几里得环](./7.交换环.md)）</font>
- **实例**：
  - **平凡环**：只有一个元素的环，等价于 $1_R = 0$
  - **整数环**：$\Z$ 是环<font color='red'>（自然数 $\N$ 本质是由归纳法定义的结构，整数 $\Z$ 本质是由环定义的结构）</font>
  - **同余类环**：$\Z_n$ 是环
  - **倍数环**：$n\Z$ 是环
  - **倍数同余类环**：$n\Z/m\Z$ 是环
- **反例**：
  - **全体奇数**：没有加法零元，不是环

### 交换性和含幺性

- **交换环**：满足乘法交换律的环
  - **反例**：
    - **矩阵环**：任意数域上的方阵环 $F^{n\times n}$ 都非交换
    - **上三角矩阵环**
- **含幺环**：
  - **幺元**：$1_Ra = a1_R = a$
  - **零元**：$0 + a = a + 0 = a$
  - <font color='red'>加法零元必定存在，幺元可能不存在
  - 实际上很多教材都只关心含幺环</font>
  - **反例**：
    - **倍数环**：$n\Z$ 没有幺元
- **运算法则**：<font color='chartreuse'>都是分配律的推论</font>
  - 零元乘法性：$0a = a0 = 0$
    - <font color='chartreuse'>反设 $0a = b$，则由分配律 $a = (0+1)a = b+a$，矛盾</font>
  - 负数交换律、结合律：$(-a)b = a(-b) = -(ab)$
    - <font color='chartreuse'>由分配律，$ab + (-a)b = (a-a)b = 0b = 0$，故是加法逆元</font>
  - 数乘交换律、结合律：$(na)b = a(nb) = n(ab)$
    - <font color='chartreuse'>由分配律，$(na)b = (a+\cdots+a)b = ab+\cdots+ab = n(ab)$</font>

### 可逆性

- **左零因子**：已知 $a$，若 $\exists (b\neq 0)\in R$ 使得 $ab = 0$，则 $a$ 是 $b$ 的左零因子
  - **零因子**：同时为左右零因子的元素
    - **实例**：$\Z_6$ 中，$2$ 与 $3$ 互为零因子
  - <font color='red'>（实际上纠结这种死板定义没有意义。具体情况中取临时定义也没有问题。只需知道有提出零因子的必要）</font>
  - **乘法消去律**：若 $b$ 无左零因子，则 $ab = cb \red\Rt a = c\pad (\forall a,c\in R)$
- **左可逆**：已知 $a$，若 $\exists c\in R$ 使得 $ca = 1_R$，则 $a$ 左可逆
  - **左逆**：$a^{-1}_L = c$
    - **不唯一性**：
      - **反例**：无穷实数列对分量加法构成阿贝尔群，其上自同态构成环。
        - 设 $l,r$ 分别是左右推移算子，$l_0:(a_0,...) \mapsto (a_0+a_1,a_2,...)$
        - 则 $lr = \text{id} = l_0r$，即 $r$ 左逆不唯一
        - 同理，$l$ 右逆不唯一<font color='chartreuse'>（？？$l$ 非满射，$r$ 非单射，详见泛函分析的满线性算子群）</font>
    - **唯一性条件**：若 $a$ 不是右零因子，则其左逆唯一
      - **证明**：反设 $b_1a = b_2a = 1$，则 $(b_1-b_2)a = 0$，矛盾
      - <font color='chartreuse'>$r$ 是 $(l-l_0):(a_0,...)\mapsto (a_0,0,0,...)$ 的右零因子
        - 同时也是 $p:(a_0,...)\mapsto(a_0,a_0,a_0,...)$ 的右零因子
      - $l$ 是 $(r-r_0):(a_0,...)\mapsto (a_0,0,0,...)$ 的左零因子
        - 同时也是 $q:(a_0,...)\mapsto (a_0,0,0,...)$ 的左零因子</font>
  - **可逆元（单位）**：左右均可逆的元
  - **单位群 $\U_R$ 或 $\R^\times$**：含幺环中的所有可逆元构成的乘法群
    - **实例**：复平面整数点 $\Z[i] = \set{a+bi\mid a,b\in \Z}$ 的单位群为 $\lang i \rang$
      - $\Z[3i]$ 的单位群为 $\lang -1 \rang$ 
- **整环 $R$**：无零因子且 $1_R \neq 0$ 的交换环<font color='red'>（无零因子、含幺、交换）</font>
  - **实例**：整数环 $Z$
  - **反例**：
    - **不含幺**：偶数环
    - **不交换**：矩阵环，连续函数环
    - **含零因子**：合数同余类环，直积环，多项式商环
- **可除环 $D$（体）**：非零元均可逆 且 $1_D \neq 0$ 的环<font color='red'>（可逆、含幺）</font>
  - **有限情况**：无零因子的非平凡有限环是可除环
    - **证明**：无零因子得左右逆唯一，从而可逆，
  - **实例**：实四元数环
- **域 $F$**：交换的可除环<font color='red'>（可逆、含幺、交换）</font>
  - **有限情况**：整环 $\LR$ 域
    - **证明**：$R-\{0\}$ 对乘法满足消去律，且为有限幺半群。易得其为乘法群。
      - **含幺性**：由整环定义
      - **交换性**：由整环定义
      - **可逆性**：整环得无零因子性，再由有限性得可逆性
  - **实例**：$\Z_p$ 是有限域
  - **反例**：$\Z$ 是无限整环，但不是可除环，从而不是域
  - **总结**：有限情况下，除环 $\subset$ 整环 $\subset$ 域

### 例子


- **含幺非交换环**：
  - 矩阵环 $\Q^{n\times n}、\R^{n\times n}、\C^{n\times n}$，其中可逆元是非奇异矩阵
- **含零因子有限环**：$Z_{kr}$，零因子为 $\bar k，\bar r$
  - $Z_m$ 的子环为 $\set{Z_r\cong \dfrac{m}{r}Z_m : r\mid m}$
  - $\Aut Z_m = \{1\}$
- **有理数域 $\Q$**：下面结论都是域论知识
  - $\Q$ 的子域只有 $\Q$（详见[有限域](15.域的分类.md)）
  - $\Q[\sqrt{2}] = \set{a+b\sqrt{2}\mid a,b\in\Q}$ 是 $\R$ 的子域，其子域为 $\Q$ 和 $\Q[\sqrt{2}]$
  - $\Aut\Q[\sqrt{2}]\cong Z_2$
    - **证明**：易得 $\sigma(1) = 1$，故在 $\Q$ 上限制为恒等映射。再设 $\sigma(\sqrt{2}) = a$，已知 $\sigma(2) = a^2 = 2$，故 $a = \pm\sqrt{2}$
- **整数域 $\R$**：
  - $\Aut \R$ 具有单调性
    - **证明**：只需证明保号性。设 $a > 0$，则存在 $b$ 使得 $a = b^2$，从而 $f(a) = f(b)^2 > 0$
  - $\Aut \R = \{1\}$
    - **证明**：易得 $f$ 在 $\Q$ 上限制为恒等映射
      - 设 $r$ 是正无理数，由完备性，可取单增收敛列 $r_n\to r$，易得 $f(r)-f(r_n) < f(\e)$。再由单调性易得 $f(\e)\to 0$，从而 $f(r) = \lim\limits_{n\to\infty}r_n = r$，即得结论
- **复数域 $\C$**
  - **矩阵表示**：取嵌入 $\sigma:\C\to GL_n(\R)，a+bi\mapsto \tvec{a & b \\ -b & a}$，本质是复数特征值的一一对应性
- **自同态群** $\End A$ 是环
  - **交换条件**：若 $A$ 是交换群，则 $\End A$ 是交换环
  - **反例**：$A = Z\oplus Z$ 时，$\End A$ 是非交换环
- **群与环的组合**：设 $G$ 是 $n$ 阶乘法群，$R$ 是环
  - **群环**：$R[G] = \sum\limits_{g\in G} R$
    - <font color='chartreuse'>（为每个 $g$ 赋予一个环 $R$ 的副本，本质上是环的直和）</font>
    - 元素：$x = \sum\limits^n_{i=1} r_{g_i}g_i$<font color='chartreuse'>（形式和，类似形式幂级数）
      - 意思是说，$r_{g_i}$ 的下标是为了区分直和中不同的 $r$，但它无法表示该项在直和中所处的序号。故要引入右边的因子 $g_i$，它没啥实际意义，就是用来表示在直和中的序号。实际中的运算我们关心的只有 $g_i$ 的系数而已</font>
    - 加法：$x+y = \sum\limits^n_{i=1} r_{g_i}g_i + \sum\limits^n_{i=1} s_{g_i}g_i = \sum\limits^n_{i=1} (r_{g_i}+s_{g_i})g_i$
    - 乘法：$R(r_is_j) = G(g_ih_j) = \dkh{\sum\limits^n_{i=1} r_ig_i}\dkh{\sum\limits^m_{j=1} s_{g_i}h_j} = \sum\limits^n_{i=1}\sum\limits^m_{j=1} (r_is_j)(g_ih_j)$
    - 幺元：$1_R\cdot e$
  - **实例**：
    - 当 $G$ 为 $n$ 阶循环群时，$R[G] = R[x]/(x^n-1)$ 是干域（详见[有限域](15.域的分类.md)）
    - 当 $G$ 为无限循环群（$\Z$）时，$R[G] = R[x]$ 是多项式环
  - **父环性**：$R$ 是 $R[G]$ 的子环
  - **交换性**：
    - 若 $G$ 有限，$R$ 交换，则 $R[G]$ 的中心是
    - $R,G$ 均交换 $\LR$ $R[G]$ 交换
  - **零因子**：
    - $G\neq \{e\}$ 时，$\Big( \sum\limits_{g\in G} g \Big)(1-h) = (1-h)\sum\limits_{g\in G} g = 0$，它们互为 $R[G]$ 的零因子
    - $G = \{e\}$ 时，$R[G]$ 无零因子 $\LR R$ 无零因子
  - **单位群**：
    - 若 $G = \set{1,g}$，则 $n+mg\in R[G]$ 可逆 $\LR \begin{cases} nx+my = 1 \\ mx+ny = 0  \end{cases}$ 有整数解，即 $n^2-m^2 = \pm 1$，从而 $R[G]$ 的单位群为 $K_4 = \set{1,-1,g,-g}$
- **实四元可除环**：[群的习题](2.群的映射.md)中也有出现
  - **实数表示**：$\set{1,-1,i,-i,j,-j,k,-k}$
  - **复矩阵表示**：所有矩阵 $\tvec{z & w \\ -\ol w & \ol z}$ 构成的环
    - 生成元系：$\tvec{1 & 0 \\ 0 & 1},\tvec{i & 0 \\ 0 & -i},\tvec{0 & 1 \\ -1 & 0},\tvec{0 & i \\ -i & 0}$


## 环同态

- **（定理1.6）二项式定理**
- **环同态**：两个环之间的映射 $f:R\to S$ 满足 $\begin{cases} f(a+b) = f(a)+f(b) \\ f(ab) = f(a)f(b) \end{cases}$
  - **实例**：
    - **平凡同态（恒等）**
    - **满同态（收缩）**：$\Z\to \Z_m，k\mapsto \ol k$
    - **单同态（嵌入）**：$\Z_3 \to \Z_6，\bar k\mapsto \ol{4k}$
    - **群环同态**：
      - 设 $f:G\to H$ 是群同态
      - 则 $\bar{f}:R[G]\to R(H)，\sum\limits^n_{i=1} r_ig_i\mapsto \sum\limits^n_{i=1} r_if(g_i)$ 是环同态
  - **核**：同态像是 $0$ 的元素
    - <font color='red'>同态一般不保乘法幺元，所以 $1$ 的原象意义不大</font>
  - **保含幺性**：若 $R$ 含幺，则陪域 $S$ 必须含幺
    - <font color='chartreuse'>$\p:\Z\to 2\Z，1\mapsto 2$ 不是环同态，不满足乘法公式</font>
  - **无保幺性**：$1_R\to 1_S$ 不必总成立
    - **反例**：
      - **陪域无幺元**
        - $R\to 2\Z$
      - **零同态（零映射）**
      - **陪域存在零因子**
        - 矩阵同态：$\p:R\to R^{2\times 2}，x\mapsto \tvec{x & 0 \\ 0 & 0}$，其像集无幺元（是 $R^{2\times 2}$ 的子环，但幺元不相等）
      - **不保可逆元**：
        - 泛函分析中，满线性映射构成群，其有右逆，无左逆
- **保幺条件**：设陪域含幺
  - **满条件**：满同态保幺元
    - **证明**：$\p(1_R)s \overset{满射}{=} \p(1_R)\p(r) = \p(1_R\cdot r) = \p(r) = s$
  - **无零条件**：同态非零，且陪域无零因子
    - **证明**：
      - 反设 $\p(1_R) \neq 1_S$，则由同态运算得 $\forall a\in R，\p(a)\ddkh{\p(1_R)-1_S} = 0$，与无零因子性矛盾
  - **可逆条件**：在单位群上的限制是群同态
    - **证明**：
      - 设 $\p:R\to S$，取样法易得此时 $\p(R^\times) \subset S^\times$
      - 设 $\p(1_R) = s$，则 $s^{-1} = \p(1_R)^{-1} = \p(1_R)$。由幺元唯一性得 $s = 1_S$
- **保可逆定理**：保幺元的同态必定保逆元
  - **证明**：显然

### 特征

- **环的特征**：若 $\exists n\in \Z$ 使得 $\forall a\in R，na = 0$，则称 $\char R = n$
  - <font color='red'>幺元的加法阶（讨论特征时，一般默认是含幺环）</font>
  - **幺元阶定义法**：特征 $n$ 是满足 $n1_R = 0$ 的最小正整数
    - **证明**：
      - 反设 $\char R = n$，但存在 $m<n$ 使得 $m1_R = 0$
      - 则此时存在 $a$，使得 $n$ 是使其为 $0$ 的最小系数，则 $n$ 也是 $a\cdot 1_R$ 的最小系数
        - 但是由乘法分配律，$ma\cdot 1_R = a\cdot m1_R$
        - 由 $1_R$ 性质，右式为0，但由 $a$ 性质，左式不为0，矛盾
  - **实例**：$\Z_p$ 的特征为 $p$
- **零特征环**：若不存在有限特征，则 $\char R = 0$
  - **实例**：
    - 整数环 $\Z$
    - 倍数环 $n\Z$
    - 矩阵环 $\Z^{n\times n}$
- **（定理1.9）特征的本质**：<font color='red'>$\Z_n$ 可嵌入特征为 $n$ 的环（包括零特征环）</font>
  - **特征嵌入**：$\a_R:\Z_n\to R，k\mapsto k1_R$ 是嵌入
    - <font color='red'>仅 $R = \Z_n$ 时是满射</font>
  - **同余性**：若 $R$ 不含零因子，则 $\char R$ 是素数
    - **证明**：
      - 反设 $\char R = mn$，则 $mna = m(na) = 0$
        - 由于 $m<mn$，故由特征定义得 $na$ 只能是 $0$
        - 再由 $n<mn$ 得要么 $a=0$（无任意性，矛盾），要么 $R\cong \Z_{mn}$（含零因子，矛盾）
  - **特征定理**：$\p:R\to S$ 是保幺同态 $\LR \p\circ \a_R = \a_S$
    - <font color='red'>也就是说，环的特征实际上指的是特征嵌入的特征性质</font>
    - **证明**：显然
- **（定理1.10）分裂扩张定理**：每个环 $R$ 都可以保幺嵌入到某个含幺环 $S$ 中
  - 若 $\char R = 0$，则 $\char S = 0$
  - 若 $\char R = n$，则 $\char S = 0$ 或 $n$
  - <font color='red'>保幺嵌入只能是等特征的，但非保幺嵌入就多种多样</font>
  - **证明**：
    - **零特征嵌入**：
      - 设 $S$ 是阿贝尔群 $R\oplus \Z$
      - 定义乘法 $(r_1,k_1)(r_2,k_2) = (r_1r_2+r_1k_2+r_2k_1，k_1k_2)$
      - 则 $S$ 是含有幺元 $(0,1)$ 的环且特征为0，环嵌入为 $r\mapsto (r,0)$
    - **等特征嵌入**：
      - 设 $S$ 是阿贝尔群 $R\oplus \Z_n$（其中 $n = \char R$）
      - 定义乘法 $(r_1,\ol k_1)(r_2,\ol k_2) = (r_1r_2+r_1\ol k_2+r_2\ol k_1，\ol k_1\ol k_2)$
      - 则 $S$ 是含有幺元 $(0,\ol 1)$ 的环且特征为 $n$，环嵌入为 $r\mapsto (r,0)$
  - <font color='red'>不难发现这两个群（Dorroh扩张）跟群的半直积扩张是完全相同的。嵌入就是半直积的规范逆投影</font>

### 环范畴

- **环范畴**：所有环构成的范畴，态射为环同态
- **含幺环范畴**：所有含幺环构成的范畴，态射为保幺同态
  - **始对象**：$\Z$
- **特征含幺环范畴**：所有特征为 $n$ 的含幺环构成的范畴，态射为保幺同态
  - **始对象**：$\Z_n$

### 习题

- **环的直和**：设 $\{R_i\mid i\in I\}$ 均为含幺环，则 $\sum\limits_{i\in I} R_i$ 是含幺环，加法和乘法为各分量相加相乘。
- **$R$ 的反环**：$R^{\circ p}$ 的底层集合是 $R$，加法和乘法都换序（$a\circ b = ba$）
  - **含幺性相同**：两环幺元相同
  - **可除性相同**：由互逆可换性，两环逆元相同
  - **自反性**：反环的反环是原环
  - **同构性**：定义易得
- **广义可逆性**：若 $R$ 中任意非零元 $a$，都存在唯一的 $b$ 使得 $aba = a$，则其满足
  - **对称性**：$bab = b$
    - **证明**：易得此时 $(ab)^2 = ab$，即 $ab$ 是幂等元素
      - 故 $(ab)^3 = (aba)(bab) = ab$，再由 $aba = a$ 即得 $bab = b$
  - **含幺性**：
    - **证明**：此时 $(ab)a = a$，是左幺，$b(ab) = b$，是右幺
  - **无零因子性**：
    - **证明**：反设 $a$ 有右零因子，则其右逆不唯一。设 $ac_1 = ac_2 = 1$，则 $ac_1a = a，ac_2a = a$，与题设唯一性矛盾，故无零因子
  - **可逆性**：
    - **证明**：由含幺性得 $(ab-1)a = 0$，再由无零因子性，只能是 $ab = 1$，同理 $ba = 1$，从而 $b = a^{-1}$
- **新生之梦（蚌埠住了这名字）**：设 $R$ 是交换幺环，若 $\char R = p$，则 $(a\pm b)^{p^n} = a^{p^n} \pm b^{p^n}$
  - **证明**：二项式定理易得
- **幂零元素**：$\exists n$ 使得 $a^n = 0$
  - **加法封闭性**：交换环中两幂零元素的和 $a+b$ 也幂零
  - **反例**：非交换环中不成立
    - $GL_2(\R)$ 中，上升阵 $N$ 和下降阵 $N^T$ 幂零，但 $N+N^T$ 可逆
  - **推论**：环 $R$ 中不存在非零的幂零元素 $\LR a^2 = 0 \blue\Rt a = 0$
    - **证明**：
      - **必要性**：定义易得
      - **充分性**：设 $a^n = 0$
        - 若 $n$ 是偶数，则 $a^n = (a^m)^2 = 0$，由充分性条件即得 $a^m = 0$
        - 若 $n$ 是奇数，则 $a^n\cdot a = 0$，从而 $a^{\dfrac{n+1}{2}} = 0$
        - 不断重复即得结论
- **Frobenius同态**：若交换环 $R$ 满足 $\char R = p$，则 $F:R\to R，r\mapsto r^p$ 是同态
  - **证明**：
    - 由交换性，$F(ab) = (ab)^p = a^pb^p = F(a)F(b)$
    - 由新生之梦，$F(a+b) = (a+b)^p = a^p+b^p = F(a)+F(b)$
- **可逆共轭性**：幺环中，$1-ab$ 可逆 $\LR 1-ba$ 可逆
  - **证明**：设 $(1-ab)^{-1} = c$，则 $(1-ba)^{-1} = 1+bca$

### 例子

- **全零环（交换无幺环）**：加法阿贝尔群 $G$ 上，定义 $\forall ab = 0$，则其构成环
  - **交换性**、**无幺性**、**不可逆性**
  - **零因子**：所有元素均互为零因子
- **幂集环（交换幺环）**：幂集 $\mc P(U)$ 中，$A+B = A\D B$，$AB = A\cap B$，则其构成环
  - **交换性**：易得
  - **含幺性**：幺元为全集，零元为空集
  - **零因子**：不相交的集合互为零因子
- **布尔环（幂等环）（交换幺环）**：$\forall a\in R，a^2 = a$
  - **自逆性**：$a+a = 0$
    - **证明**：已知 $(-a)^2 = (-1)a(-1)a = a^2 = a$，再由 $(-a)^2 = -a$，即得 $a = -a$
  - **交换性**：$(a+b)^2 = a+ab + ba + b = a+b$，故 $ab = -ba$，自逆性即得 $ab = ba$
  - **本质**：元素可以只用 $0,1$ 表示的环
  - **实例**：
    - 幂集环（因为 $A\cap A = A$）
    - 幂等矩阵 $\set{A\in F^{n\times n}\mid \forall n\in N^+，A^n = A}$
- **映射环**：设 $R$ 是环，$S$ 是非空集合，则映射群 $M(S,R) = \set{f:S\to R}$ 是环
  - 加法 $(f+g)(s) = f(s)+g(s)$，乘法 $(fg)(s) = f(s)g(s)$

- **同态环 $\End A$（非交换幺环）**：阿贝尔群的自同态
  - **幺元**：幺元是恒等映射，零元是零映射

- **连续函数环 $C(R)$（非交换幺环）**
  - **无交换性**
  - **幺元**：幺元是恒等映射，零元是零映射
