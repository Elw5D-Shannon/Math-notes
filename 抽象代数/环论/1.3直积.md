# 环与直积

## 直积

- **（定理2.22）直积性质**：设 $R_i$ 是环族，$\prod\limits_{i\in I} R_i$ 是加法阿贝尔群直积
  - 加法为各元相加，乘法为各元相乘 $\{a_i\}_{i\in I}\{b_i\}_{i\in I} = \{a_ib_i\}_{i\in I}$，则其构成环
  - 含幺性、交换性可传递
  - 规范投影是满同态
  - 规范逆投影是单同态
- **环的直和**：环的弱直积是直积的子环，同时也是环的直和
- **（定理2.24）弱内直积**：
  - 若 $R$ 的理想列 $\{A_i\}^{n}_{i=1}$ 满足
    - **覆盖性**：$\sum\limits^n_{i=1} A_i = R$
    - **不相交性**：$A_k\cap \sum\limits_{i\neq k} A_i = 0\quad (\forall 1\leq k\leq n)$
  - 则存在环同构 $R\cong \prod\limits^n_{i=1} A_i$
  - **证明**：已知 $(a_1,...,a_n) \mapsto \sum\limits^n_{i=1}a_i$ 是加法阿贝尔群同构，只需证明其为环同构即可

## 同余

- **环同余**：若 $a,b\in R$ 满足 $a-b\in A$，则称为**模理想 $A$ 同余**
- **（定理2.25）中国剩余定理**：
  - 设 $\{A_i\}^n_{i=1} \in \I_R，\{b_i\}^n_{i=1}\in R$
  - 若满足
    - **平方不太小性**：$R = R^2+A_i$
    - **两两覆盖性**：$R = A_i+A_j\pad (\forall i\neq j)$
  - 则存在 $b\in R$ 使得 $b\equiv b_i \pmod {A_i}$
  - **引理**：$R = A_1 + (\mathop{\bigcap}\limits_{i\neq 1}A_i)$
    - **初始条件**：$R = A_1 + (A_2\cap A_3)$
      - 由题设，$R^2 = (A_1+A_2)(A_1+A_3) = A_1^2 + A_1A_3 + A_2A_1 + A_2A_3 $
      - 由乘积缩小性，原式可改写为 $\subset A_1+A_2A_3 \subset A_1+(A_2\cap A_3)$
      - 再由题设，$R = A_1 + R^2 \subset A_1 + (A_2\cap A_3)\subset R$
        - 综上即得**初始条件**
    - **归纳假设**：$R = A_1 + (A_2\cap\cdots A_{k-1})$
      - 依然有 $R^2 = \Big(A_1 + A_2\cap\cdots\cap A_{k-1}\Big)(A_1 + A_{k}) = \cdots$
  - 同理也可得 $R = A_k + (\mathop{\bigcap}\limits_{i\neq k}A_i)$
  - 综上，$\forall k，\exists a_k\in A_k，r_k\in (\mathop{\bigcap}\limits_{i\neq k}A_i)$ 使得 $b_k = a_k + r_k$
    - 也即 $\begin{cases} r_k\equiv b_k \pmod{A_k} \\ r_k\equiv 0\pmod{A_i} & i\neq k\end{cases}$
    - 最后取 $b = r_1 + \cdots + r_n$ 即可
  - **推论**：$b$ 由模 $\bigcap A_i$ 的同余类唯一决定
    - **证明**：若 $\forall i$，都有 $c \equiv b_i \pmod{A_i}$，则也有 $b\equiv c\pmod A_i$
      - 从而由任意性，$b-c\in \bigcap A_i$，也即 $b\equiv c\pmod{\bigcap A_i}$
- **（推论2.27）商积定理**： <a id = "商积定理"></a>
  - 若 $A_1,...,A_n \in \I_{R}$
  - 则存在单同态 $\t:R/(A_1\cap \cdots \cap A_n)\to R/A_1\times \cdots \times R/A_n$
  - **证明**：由环范畴的范畴直积定理，规范商映射 $\pi_k$ 可导出商积环同态 $\t_1: R\to \prod\limits^n_{i=1} R/A_i，r\mapsto (r+A_i)^n_{i=1}$
    - 易得 $\ker \t_1 = \mathop{\bigcap}\limits^n_{i=1} A_i$。再由环同构定理，存在环单同态 $\t:R/(\cap) \to \prod R/A_i$
  - **推论**：满足中国剩余条件，则 $\t$ 是环同构
    - **证明**：
      - 由中国剩余定理可得同余关系
      - 此时 $\t(b+\cap A_i) = (b+A_i)^n_{i=1} = (b_i+A_i)^n_{i=1}$，从而是满射
    - **反例**：
      - 设 $R = \Z，A_1 = (6)，A_2 = (4)$
      - 则 $\t:R/(A_1\cap A_2)\to R/A_1\times R/A_2$ 不是满射
      - **证明**：此时 $\t:Z_{12}\to Z_6\times Z_4$，前者是12阶，后者是24阶

### 习题

- **性质传递**：设 $R = \prod R_i$，则
  - 交换性等价
  - 含幺性等价
  - 可逆性等价：$x = (x_i)$ 是 $R$ 的可逆元 $\LR x_i$ 均为可逆元
  - 理想直积性：若 $R$ 含幺，$I$ 是有限理想，则 $I = \prod A_i$
- **直积因子**：两个理想 $I_1\times I_2 = R$
- **幂等元素**：$e^2 = e$
- **正交基引理**：含幺环 $R$ 中，若中心元素 $e$ 幂等，则
  - $1_R-e$ 是中心幂等元素
    <!-- - $(1_R-e)^2 = 1_R - 2e + e^2 = 1_R-e$ -->
  - $R \cong eR \times (1_R-e)R$
  - **证明**：第一结论计算易得
    - 第二结论：首先已证是陪集理想，$\forall r \in R$，都有 $er + (1_R-e)r = r$，故 $R\subset eR\oplus (1_R-e)R$
      - 再由于 $\forall er +(s-es)\in R$，故 $eR\oplus (1_R-e)R \subset R$。综上，两者相等
- **正交元素族**：一列幂等元素 $\{e_i\}^n_{i=1}$ 满足 $e_ie_j = 0$
- **正交积定理**：设 $R，R_1,...,R_n$ 是含幺环，则下列条件等价
  - $R\cong \prod R_i$
  - $R$ 含有中心幂等的正交元素族，满足 $\sum\limits^n_{i=1} e_i = 1_R$，且 $e_iR\cong R_i$
  - $R = \prod A_i$，其中 $A_i\cong R_i$ 是 $R$ 的理想
  - **证明**：
    - $(2)\to (1)$ 由正交基引理归纳直得
    - $(1)\LR (3)$ 由内直积定义直得
  - **实例**：向量空间（域）的正交基
