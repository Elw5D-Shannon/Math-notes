# 理想与商环

- **符号约定**：
  - $R$ 上全体理想的集合 $\I(R)$

## 理想

- **子环**：环 $R$ 的对加法乘法封闭的子集，同时也是一个环
  - **实例**：
    - **中心**：含幺环作为乘法群的中心是子环
  - **幺不等性**：子环不一定含幺，且幺元不一定和原环 $R$ 相等
    - **反例**：上面矩阵环同态的例子
  - <font color='red'>也有书规定子环必须和母环的幺元相同</font>
- **左理想**：满足 $r\in R，x\in I \red\Rt rx \in I$ 的子环
  - <font color='red'>对 $R$ 中元素乘法封闭，对 $I$ 中元素加法封闭</font>
- **理想**：同时是左理想和右理想
  - **实例**：
    - **平凡理想**：$R$、$\{0\}$
    - $\lang n \rang$ 是 $\Z$ 上的理想<font color='chartreuse'>（数论定理）</font>
    - 环同态的核是理想
  - **反例**：
    - 环的中心不是理想<font color='chartreuse'>（外部元素乘中心元素，积不是中心元素）</font>
      - $GL_n(\R)$ 的中心 $aI_{n\times n}$ 不是理想（$A\cdot aI = aA$，不是对角阵）
        <!-- - 除环上的二阶矩阵环，其中心为 -->
    - 环同态的像是子环，但不一定是理想<font color='chartreuse'>（同态只能保证像集自身运算封闭性）</font>
    - $n$ 阶矩阵可除环：
      - （仅第 $k$ 列有非零元的矩阵集合 $I_k$）是左理想，不是右理想
      - （仅第 $k$ 行有非零元的矩阵集合 $J_k$）是右理想，不是左理想
- **真理想**：非平凡的理想
  - **无幺性**：$1_R\in I \LR I = R$
  - **无逆性**：$I$ 包含 $R$ 中的可逆元 $\LR I = R$
    - **证明**：显然
    - <font color='red'>真理想都能嵌入 $n\Z$</font>
  - **推论**：
    - 除环没有真理想
    - 除环上的 $n$ 阶矩阵环 $D^{n\times n}$ 有真左理想和真右理想，但无真理想
      - **证明**：显然不可能存在一个特殊矩阵集，满足运算封闭 + 左右可逆

### 主理想

- **$X$ 生成的理想**：
  - 设 $X\subset R$，则 $\hkh{A_i\Bigm| i\in I，A_i\in\I(R)，X\subset A_i}$ 是 $R$ 上所有包含 $X$ 的理想。此时 $\mathop{\bigcap}\limits_{i\in I} A_i = (X)$ 即为子集生成的理想
  - **本质**：由理想的交封闭性，我们可以用闭包来定义被生成的理想
- **主理想**：<font color='red'></font>$$(a) = \hkh{ra + as + na + \sum\limits^{m}_{i=1} r_ias_i \Bigm | r,s,r_i,s_i\in R,\ m\in \N^+,\ n\in \Z}$$
  - **本质**：<font color='red'>该元素的全体乘法的线性组合
    - 单向乘法的线性组合可以合并为 $r$ 和 $s$，但双向乘法不行</font>
  - **实例**：倍数环 $n\Z$ 是整数环 $\Z$ 的主理想
- **（定理2.5）主理想的性质**：
  - **幺环上的主理想**：$(a) = \biggm\{\sum\limits^{m}_{i=1} r_ias_i\mid r_i,s_i\in R,\ m\in N^+ \biggm\}$
    - <font color='red'>具有幺元后，单向乘法可用含幺元的双向乘法代替</font>
  - **环中心上的主理想**：$(a) = \Big\{ra+na\mid r\in R,\ n\in Z\Big\}$
    - <font color='red'>具有交换性后，双向乘法和单向乘法可合并为左乘</font>
  - **陪集性**：$R$ 上的左理想都可写为 $Ra$ ，右理想都可写为 $aR$
    - **证明**：显然
  - **中心元素的主理想**：
    - 若 $a\in C(R)$，则 $(a) = Ra = aR$
    - 若 $X\subset C(R)$，则 $(X) = \hkh{\sum\limits^n_{i=1}r_ia_i \Bigm | n\in \N^+，r_i\in R，a_i\in X}$
- **主理想环**：理想均为主理想的环
  - **实例**：$\Z、\Z_n$
  - **反例**：
    - 一元多项式环 $\Z[x]$ 中，设 $n\neq 0,\pm 1$，则 $(n,z)$ 是理想，但不是主理想
    - 二元多项式环 $\Q[x_1,x_2]$ 中，$(x_1,x_2)$ 是理想，但不是主理想
- **（定理2.6）理想性质**
  - 交封闭性
  - 加法乘法封闭性
    - 理想的乘积也是理想，且 $I_1I_2 \subset I_1\cap I_2$
      - **实例**：$\Z_p\Z_q = \Z_{pq}$，不互素时则真含于
    - 理想的和也是理想，且 $I_1+I_2$ 是包含两者的最小理想
  - 加法乘法结合律
  - 乘法左右分配律
  - 无并集封闭性
    - $\Z$ 中，$(2)\cup (3)$ 不是理想

## 商环

- **商环**：
  - 设 $R$ 是环，$I$ 是其上理想
  - 则定义以下运算后，$R/I = \{a+I\mid a\in R\}$ 称为商环
    - 加法 $(a+I) + (b+I) = a+b+I$
    - 乘法 $(a+I)(b+I) = ab + I$
  - <font color='red'>只有理想才能满足（加法平移元对乘法封闭），子环不行</font>
- **（定理2.7）商环的性质**：
  - **商群性**：商环是加法的商群
    - **证明**：环是交换群，故子群均正规
  - **传递性**：含幺性和交换性可传递
    - **证明**：显然
- **（定理2.8）商映射**：$\pi:R\to R/I，r\mapsto r+I$ 是满同态，核为 $I$
  - **证明**：显然
  - **本质**：将元素 $r$ 的理想 $I$ 下的平移等价类 $\ol r$ 都合并在一起的映射
  - **推论**：群的同构定理均可应用到环中，理想充当正规子群的角色

- **主理想运算封闭性**：
  - 加法：$\forall a,b\in R，\exists c$ 使得 $(a)+(b) = (c)$
    - **证明**：$na$
  - 乘法：
- **同态传递定理**：设 $f:R\to S$ 是同态
  - **理想反射性**：理想的原像是包含 $\ker f$ 的理想
    - **证明**：定义易得
  - **满射保理想性**：若 $f$ 是满同态，则理想的像还是理想
    - **证明**：定义易得
    - **反例**：
  - **保主理想性**：主理想的同态像还是主理想
    - **证明**：设 $f(a) = b$
      - 则 $f(ra+ as + na + \sum r_ias_i) = f(r)b + bf(s) + nb + \sum f(r_i)bf(s_i)$
      - 由 $r,s,r_i,s_i,n$ 的任意性易得像也是主理想
  - **保主理想环性**：主理想环的同态还是主理想环
    - **证明**：上个定理易得
- **无遗传性**：理想的理想不一定是理想
  - **反例**：类似同余类中正规子群无遗传性
- **右逆无限性**：若幺环中某元素右逆不唯一，则其右逆无限
  - **证明**：设 $a$ 右逆不唯一，则易得其右零因子可组成右理想 $I$
    - 反设 $I$ 不是无限集，设为 $I = \set{x_0,...,x_n}$，其中 $x_0 = 0$
      - 由右乘封闭性得 $\forall x_ia\in I$，再由 $a$ 有右逆，故其为右因子时满足消去律，故 $x_ia\neq x_ja \neq 0$，从而 $\set{x_1a,...,x_na}$ 是 $I-\{x_0\}$ 的一个置换。
        - <font color='chartreuse'>若 $a$ 右逆不唯一，则其无逆</font>
    - 设 $b$ 是 $a$ 的右逆<font color='chartreuse'>（此时对其它右逆 $b'$，都有 $b-b'\in I$）</font>
      - 由于 $a$ 是左零因子，其不可能为某元素的右幺，故 $ba\neq 1$
      - 再由于 $a(ba-1) = (ab)a-a = 0$，得 $ba-1$ 为右零因子，即 $ba-1\in I$
      - 可设 $ba-1 = x_i$，则 $x_ib = (ba-1)b = 0$
      - 再由 $x_i = x_ja$，得 $x_ib = x_jab = x_j\neq 0$，矛盾，只能是 $I$ 无限，即右零因子无限。再由于 $b+x_i$ 也是右逆，故右逆也无限
  - **本质**：
  - **理解**：
    - 当 $I$ 是有限集时，存在雷点 $ba-1\in I$，某个置换中会爆雷
      - 若 $b$ 是右逆，则 $ba-1$ 是 $a$ 的右零因子
      - 再易得 $(ba-1)b = 0$，但这与其为 $a$ 的右零因子矛盾
    - 而当 $I$ 为无限集时，可以不存在置换关系（$x_i = x_ja$）
  - **推论**：有限幺环中若有零因子，则该元素不可逆
    - 比如 $Z_6$ 中 $3$ 不可逆
- 设 $(m,n) = 1$，若整环中 $a^m = b^m，a^n = b^n$，则 $a=b$
  - **证明**：
    - 由互素性，裴蜀等式得 $ms+nt = 1$
    - 则 $bb^{ms} = ba^{ms} = ba^{1-nt} = ab(a^n)^{-t} = ab(b^n)^{-t} = ab^{ms}$
    - 由整环消去律即得 $a=b$

### 理想实例

- **乘法幂零理想 $N$**：交换环的所有幂零元素构成理想
  - **证明**：$(rx)^n = r^nx^n = 0$，故 $rx$ 也幂零，乘法封闭
    - 再由幂零加法封闭性，即得是理想
  - **推论**：$R/N$ 无幂零元素
- **幂理想**：设 $I$ 是交换环 $R$ 的理想，则 $\Rad I = \set{x\in R\mid \exists n，x^n\in I}$ 是理想
  - **证明**：$(rx)^n = r^nx^n \in I$，乘法封闭。
    - 再由 $(x_1+x_2)^{n_1n_2}\in I$，自身加法封闭
  - **本质**：
  - **实例**：$Z_8$ 中，$I = (4)$，则 $\Rad I = (2)$
- **零因子理想**：设 $R$ 是环，$a\in R$，则 $J = \set{x\in R\mid xa = 0}$ 是左理想，$K = \set{x\in R\mid ax = 0}$ 是右理想
  - **证明**：$(rx)a = r(xa) = 0$，乘法封闭
    - $(x_1+x_2)a = 0+0 = 0$，自身加法封闭
  - **本质**：左右零因子集是左右理想
  - **理解**：所有零因子，必定都是某个零因子的积
- **零衍生理想**：若 $I$ 是 $R$ 的左理想，则 $A(I) = \set{x'\in R\mid \forall x\in I，x'x = 0}$ 也是 $R$ 的理想
  - **证明**：$(rx')x = r(x'x) = 0$，乘法封闭
    - $(x_1'+x_2')x = 0+0 = 0$，自身加法封闭
  - **本质**：理想的左零因子集也是理想
- **陪集**：设 $I$ 是 $R$ 的理想，则 $[R:I] = \set{x\in R\mid \forall r\in R，rx\in I}$ 是大于 $I$ 的理想
  - **证明**：$r(r'x) = (rr')x \in I$，乘法封闭
    - $r(x_1+x_2) = rx_1+rx_2\in I$，自身加法封闭
    - 元素分析法易得 $I\leq [R:I]$
  - **本质**：只满足乘法封闭性的集合也是理想
- **嵌入理想**：设 $R$ 是无幺无零因子的环，$S$ 是其嵌入 $R\times\Z$，则
  - $A = \set{(r,n)\in S\mid \forall x\in R，rx + nx = 0}$ 是理想
    - **证明**：
  - $S/A$ 是商环
    - 含幺性
    - 存在同构于 $R$ 的子环
    - 无零因子
- **矩阵理想**：设 $R$ 是幺环，$S$ 是 $R$ 上所有 $n$ 阶方阵构成的环<a id = "矩阵理想"></a>
  - 则 $J\in\I_S \LR \exists I\in\I_R$，使得 $J$ 是 $I$ 上所有 $n$ 阶方阵构成的环
  - **证明**：
    - **充分性**：此时 $J$ 的内部加法封闭性、$S$ 内乘法封闭性均易得
    - **必要性**：
      - **单元矩阵引理**：
        - 设 $E_{rs}$ 是 $(r,s)$ 位置为 $1_R$，其余位置为 $0_R$ 的 $n$ 阶矩阵
        - 则对任意矩阵 $A$，$E_{pr}AE_{sq}$ 是 $(p,q)$ 位置上为 $a_{rs}$，其余为 $0$ 的 $n$ 阶矩阵。
      - 给定 $J\in\I_S$，设 $I$ 是 $R$ 中所有出现在 $J$ 的某个矩阵 $j$ 的 $(1,1)$ 位置的元素全集。
      <!-- - 易得 $\forall j\in J，s\in S$，$\begin{cases} (js)_{(1,k)} = \set{\sum\limits^n_{i=1}j(1,i)s(i,k)} \\ (sj)_{(k,1)} = \set{\sum\limits^n_{i=1}s(k,i)j(i,1)} \end{cases}$ -->
      - **内部加法封闭性**：由 $J$ 的内部加法封闭性易得
      - **整体乘法封闭性**：由 $J$ 的整体乘法封闭性得，$\forall s_1,s_2\in S，j\in J，s_1js_2\in J$
        - 故取 $s_1 = r_1E_{11}，s_2 = r_2E_{11}$ 时也有 $s_1js_2 = r_1r_2j_{11}E_{11} \in J$
        - 由 $I$ 定义，$r_1r_2j_{11}\in I$，再由 $r_1,r_2$ 的任意性即得 $I$ 的乘法封闭性
      - 综上，$I\in \I_R$，再证明其上的所有矩阵全集为 $J$ 即可
        - 由于 $S$ 是所有 $R$ 上方阵，故初等矩阵也包含其中（将 $1$ 替换成 $1_R$ 即可）。从而 $J$ 中矩阵的行与列经过换序后依然在 $J$ 中，也即 $J$ 中矩阵所有位置上的元素都在 $I$ 中（**证毕**）
  - **理解**：（某个元素环 $R$ 生成的矩阵环 $S$）中的理想 $J$ 是（$R$ 上理想 $I$ 生成的矩阵环）
  - **推论**：设 $S$ 是可除环 $D$ 上所有 $n$ 阶矩阵构成的环，则
    - $S$ 没有真理想
      - **证明**：已知可除环没有真理想，结合定理即得结论
    - $S$ 中存在零因子
      - **证明**：两个退化矩阵即可构成零因子
    - $S\cong S/(0)$ ，且不是除环
      - **证明**：存在零因子，从而存在不可逆元素（也就是不可逆矩阵），当然不是除环
    - $(0)$ 是素理想
