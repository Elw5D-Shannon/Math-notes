# 模论

## 基本概念

- **左 $R$ 模**：
  - 设 $A$ 是加法交换群
  - 若对于环 $R$，存在映射（数乘作用） $R\times A\to A$ 满足
    - 数乘左右分配律：$r(a+b) = ra + rb$，$(r+s)a = ra + sa$
      - <font color='chartreuse'>推论：$r(na) = n(ra)$，这里 $n\in\Z$ 表示项数</font>
    - 数乘结合律：$r(sa) = (rs)a$
  - 则 $A$ 是左 $R$ 模
- **右 $R$ 模**：$A$ 和 $A\times R\to A$
- **酉/幺 $R$ 模**：$R$ 是含幺环，且 $1_Ra = a$
  - <font color='red'>一般来说极少讨论含幺环的非幺模，所以后面只要 $R$ 含幺，就默认它的模是幺模</font>
- **向量空间**：若 $R$ 是除环，则左/右 $R$ 模称为 $R$ 上的左/右向量空间
  - <font color='red'>元素均可逆的R模</font>
- **作用定理**：模就是被环作用的加法交换群
- **实例**：
  - **整数模（阿贝尔群）**：加法交换群都是幺 $\Z$ 模，作用为 $n\times a = na$
  - **子环模（子环作用）**：若 $S$ 是环，$R$ 是子环，则 $S$ 是 $R$ 模<font color='red'>（反之不成立）</font>
    - **实例**：多元多项式环、形式幂级数环都是R模
    <!-- - **多项式模**：若 $K$ 是环，$X$ 是子集，则 $K[X]$ 是 $K$ 模，作用为 $k\times \sum\limits^n_{i=1}k_ix^i = \sum\limits^n_{i=1}(kk_i)x^i$ -->
  - **理想模（理想作用）**：设 $I\in \I_L(R)$，则 $I$ 是左R模，作用为 $r\times a = ra$
    - **实例**：$(0),R$ 都是R模
  - **商环模**：设 $I\in\I(R)$，则 $R/I$ 是R模，作用为 $r\times (r_1+I) = rr_1+I$
      - 若 $I$ 是双向理想，则 $R/I$ 必须是环
  - **回撤模**：若 $\p:R\to S$ 是环同态，则S模都是R模，作用为 $r\times x = \p(r)x$
    - <font color='chartreuse'>同态都可看作一种收缩，收缩后（S）的模在收缩前（R）当然也是模</font>
  - **自同态模**：交换群 $A$ 是自同态环 $\End A$ 的模，作用为 $f\times a = f(a)$
  - **平凡模（全零模）**：任意交换群，任意环，都存在平凡模，作用为 $r\times a = 0$

## 模同态

- **R模同态**：R模之间的映射 $f:A\to B$，满足加法公式和数乘公式<font color='chartreuse'>（线性组合公式）</font>
  - **实例**：
    - 零映射是任何模的同态
    - 阿贝尔群同态是 $\Z$ 模同态
    - 升幂映射 $X:R[x]\to R[x]，f\mapsto xf$ 是R模同态，但不是环同态
      - <font color='chartreuse'>不满足环乘法公式：$X(fg) = xfg \neq xfxg = X(f)X(g)$
      - 但满足模乘法公式：$X(rf) = xrf = rxf = rX(f)$</font>
- **线性变换**：若 $R$ 是除环，则R模同态等价于线性变换
  - **实例**：
    - **实线性变换**：$R = \R^{n\times n}，A = \R^n$，则 $r\times a = R_{n\times n}(a_1,...,a_n)^T$
- **R模范畴**：全体 $R$ 模构成范畴，态射是 $R$ 模同态

## 子模

- **子模**：设 $A$ 是模，若加法子群 $B$ 关于原作用封闭，则称为 $A$ 的子模
  - **实例**：
    - **回撤模的子模**：$\ker f$ 是定义域的子模，$\Im f$ 是陪域的子模，子模的同态逆像是子模
    - **理想模的子模**：若 $I\in \I_{LR}$，$S\subset A$，则 $IS$ 是 $A$ 的子模
  - **反例**：
    - $A = \Z$ 是 $(2)$ 模，但 $(3)$ 不是 $(2)$ 模
  - **传递性**：
    - **交传递性**：两个R模的交也是R模
    - **遗传性**：子模的子模是子模
- **子空间**：向量空间的子模

### 生成子模

- **子集生成的子模**：
  - **闭包定义**：所有包含 $X$ 的 $A$ 子模之交
  - **集合定义**：$\lang X \rang = \hkh{\sum\limits^s_{i=1} r_ia_i + \sum\limits^t_{j=1} n_jb_j \Bigm| a_i,b_j\in X，r_i\in R，n_j\in \Z}$
  - **乘积定义**：$\lang X \rang = RX + \Z x$
- **含幺生成子模**：若 $R$ 是含幺环，则 $\lang X \rang = RX$
- **循环子模**：单元素生成的模
  - **集合表示法**：$\lang a \rang = \hkh{ra+na\mid r\in R，n\in \Z}$
  - **乘积表示法**：$\lang a \rang = Ra + \Z a$
  - **实例**：
    - $\Z$ 是 $\Z$ 模，由 $\lang 1 \rang$ 生成
    - $\Q$ 是 $\Q$ 模，由 $\lang 1 \rang$ 生成<font color='chartreuse'>（$\Z$ 不是 $\Q$ 的 $\Q$ 子模，因为有理数乘运算不封闭）</font>
- **含幺循环子模**：若 $R$ 是含幺环，则 $\lang a \rang = Ra$
  - **实例**：线性代数的循环子空间
- **生成并**：$\ip{\mathop{\bigcup}\limits_{i\in I}B_i} = B_{i_1} + B_{i_2} + \cdots$
  - <font color='red'>（不保证无交，故不是直和）</font>

## 商模

- **（定理1.5）诱导同态**：设 $A$ 是R模，则 $R\to Ra，r\mapsto ra$ 是R模满同态
  <!-- - 子模的和就是元素和的全集 -->
- **（定理1.6）商模**：
  - 设 $A$ 是R模，$B$ 是 $A$ 的子模
  - 则商群 $A/B$ 是R模，作用为 $r\times (a+B) = ra+B$
- **商映射（规范同态）** $\pi:A\to A/B，a\mapsto a+B$ 是R模满同态，$\ker \pi = B$

### 同构基本定理

- **（定理1.7）第一同构定理**：
  - 设 $R$ 是环，$f:A\to B$ 是R模同态，$C$ 是 $\ker f$ 的R子模
  - 则
    - 存在唯一的 $R$ 模同态 $\ol f:A/C\to B，a+C\mapsto f(a)$
    - $\ol f$ 是R模同构 $\LR f$ 是R模满同态，且 $C = \ker f$
  - **证明**：和群相同
  - **推论**：$A/\ker f\cong \Im f$
- **（推论1.8）第二同构定理**：
  - 设 $R$ 是环，$A',B'$ 分别是两个R模 $A,B$ 的子模，$f:A\to B$ 是R模同态，$f(A')\subset B'$
  - 则
    - 存在R模同态 $\ol f:A/A'\to B/B'，a+A'\mapsto f(a)+B'$
    - $\ol f$ 是同构 $\LR \Im f + B' = B，f^{-1}(B')\subset A'$
  - **证明**：和群相同
  - **推论**：
- **（定理1.9）第三同构定理**：
  - 设 $B,C$ 是环 $B$ 上模 $A$ 的子模
  - 则
    - 存在R模同构 $B(B\cap C)\cong (B+C)/C$
    - 若 $C\subset B$，则 $B/C$ 是 $A/C$ 的子模，且存在R模同构 $(A/C)/(B/C) \cong A/B$
  - **证明**：和群相同
- **（定理1.10）子模的商传递性**：
  - 设 $R$ 是环，$B$ 是R模 $A$ 的子模，$C$ 是包含 $B$ 的子模
  - 则存在双射 $\p: \{A\}\to \{A/B\}，C\mapsto C/B$
  - **证明**：
  - **本质**：每个 $A/B$ 的子模都具有 $C/B$ 形式

## 直积

- **（定理1.11）直积**：设 $R$ 是环，$\{A_i\}$ 是非空 $R$ 模族，$\prod\limits_{i\in I} A_i$ 是群直积，$\sum\limits_{i\in I}A$ 是群直和
  - **模传递性**：$\prod\limits_{i\in I} A_i$ 是R模，$r\{a_i\} = \{ra_i\}$
  - **弱性**：$\sum\limits_{i\in I} A_i$ 是 $\prod\limits_{i\in I} A_i$ 的子模
  - **规范投影**：$\pi_k:\prod A_i\to A_i$ 是满同态
  - **规范逆投影**：$\iota_k: A_k\to \sum A_i$ 是单同态
- **（定理1.12）范畴R模直积**：
  - 设 $R$ 是环，$\{A_i\}$ 是非空 $R$ 模族，$C$ 是R模，$\p_i:C\to A_i$ 是R模同态族
  - 则存在唯一的R模同态 $\p:C\to \prod\limits_{i\in I} A_i，\pi_i\p = \p_i$，也即直积由同构 $\p$ 唯一决定
  - **证明**：和群相同
- **（定理1.13）范畴R模余积**：
  - 设 $R$ 是环，$\{A_i\}$ 是非空 $R$ 模族，$D$ 是R模，$\psi_i:A_i\to D$ 是R模同态族
  - 则存在唯一的R模同态 $\psi:\sum\limits_{i\in I} A_i\to D，\iota_i\psi = \psi_i$，也即直积由同构 $\psi$ 唯一决定
  - **证明**：和群相同
- **（定理1.14）直和判定**：设 $R$ 是环，$A$ 是R模，$\{A_i\}^n_{i=1}$ 是R模族
  - 则 $A\cong \sum\limits^n_{i=1} A_i \LR \forall i，\exists \pi_i，\iota_i$ 满足以下条件
    - 互逆性：$\pi_i\iota_i = 1_{A_i}$
    - 正交性：$\pi_i\iota_j = 0\pad (i\neq j)$
    - 组合性：$\sum \pi_i\iota_i = 1_A$
  - **证明**：由范畴R模余积定理，取规范投影和规范逆投影验证即可
  - **本质**：类似前面环的正交性习题
- **（定理1.15）弱内直积**：
  - 设 $R$ 是环，$\{A_i\}^n_{i=1}$ 是R模 $A$ 的子模族，满足
    - $A$ 是 $A_i$ 的普通和<font color='chartreuse'>（$\forall a\in A，\exists \{a_i\in A_i\}，a = \sum a_i$）</font>
    - $\forall k\in I，A_k\cap \sum\limits_{i\neq k}A_k = 0$
  - 则 $A \cong \sum\limits_{i\in I} A_i$
  - **证明**：直和判定直得
