# 尺规作图

## 基本概念

- **域 $F$ 平面**：设 $F$ 是实数域的子域，则称 $P_F = \hkh{(c,d)\Bigm | c,d\in F}$ 是域 $F$ 平面
  - **$P_F$ 上的线**：以 $P,Q\in P_F$ 为两端点的直线
  - **$P_F$ 上的圆**：以 $P\in P_F$ 为圆心，$PQ$ 为半径的圆
- **基本作图法**：
  - **线段 $PQ$ 的垂直平分线**
    - 分别以 $P,Q$ 为圆心，$PQ$ 为半径画两个圆，连接两圆交点即可
  - **过点 $A$ 的 $PQ$ 平行线**：
    - 作出垂直平分线 $\a$
    - 以 $A$ 为圆心，随便画一个与 $\a$ 交于两点 $C,D$ 的圆
    - 再作 $CD$ 的垂直平分线，即为平行线
  - **过点 $A$ 的 $PQ$ 垂线**：
    - 作 $PQ$ 的垂直平分线 $\a$
    - 过 $A$ 作 $\a$ 的平行线

## 代数结构

- **交点定理**：$P_F$ 的线、圆等几何图形的交点，均位于某个 $(u\in F)\geq 0$ 对应的 $F(\sqrt{u})$ 平面中
  - **证明**：直线等价于一次多项式方程，圆等价于二次多项式方程
  - **理解**：一个平方根数即可表出所有平方根数，而所有有理方程都
- **规矩数 $c$**：若点 $(c,0)$ 可被整数坐标下的点经过有限次尺规构造得出，则 $c$ 称为规矩数
  - <font color='red'>尺规作图本质就是用"线"和"圆"进行运算，从而生成元素
  - 规矩数帮助我们把它翻译成代数语言</font>
  - **构造性**：$(c,d)$ 可尺规作图 $\LR c,d$ 均是规矩数
    - **证明**：
      - **充分性**：先作出 $(c,0)$，再画出 $(c,d)$ 即可
      - **必要性**：已知 $(c,d)$，那么分别作与两个坐标轴垂直或平行的线，其交点即为 $(c,0)，(0,d)$
  - **根号传递性**：若 $c\geq 0$ 是规矩数，则 $\sqrt{c}$ 也是
    - **作图**：设 $P$ 为原点，$Q$ 是 $(c,0)$，则分别以 $P,Q$ 为圆心，$PQ$ 为半径任意画两个相交的圆。将两圆交点连起来，就是垂直平分线。将两圆交点分别与 $P,Q$ 连起来，形成的正方形边长即为 $\sqrt{c}$
  - **四则运算传递性**：
    - **加减法**：首先作出 $P = (c,0)$，然后以 $P$ 为原点再作出 $Q = (c+d,0)$ 和 $R = (c-d,0)$ 即可
    - **乘法**：不断作 $d$ 个 $P$，即得 $(cd,0)$
    - **除法**：首先作 $P = (c,0)$，以 $P$ 为圆心，$d$ 为半径作圆。其与 $x = 1$ 的交点即为 $(1,\dfrac{d}{c})$
  - **实例**：
    - 整数是规矩数
    - 有理数是规矩数
      - **证明**：
        - 首先，$x = \dfrac{1}{2}$ 是 $(1,0)$ 的中垂线，其为规矩数
        - 再由有理数可通过四则运算得到，结论易得
  - **反例**：
    - 有理数域上既约多项式的根不是规矩数
- **（命题1.16）规矩定理**：若 $c\in\R$ 是规矩数，则 $c$ 是 $\Q$ 上的度为 $2^k\ ( k\in\N)$ 的代数元素
  - **证明**：直接以 $\Q$ 坐标平面为基础讨论即可
    - 由交点定理，第一次构造的点全部在 $\Q(\sqrt{u})$ 上
      - 此时 $u$ 的次数为 $2\ (u\notin\Q)$ 或 $1\ (u\in\Q)$
    - 再在第一次基础上进行第二次构造，则点全在 $Q(\sqrt{u},\sqrt{v})$ 上。
    - 最终有限次构造出 $[\Q(u_1,...,u_n):\Q] = 2^n$，其为有限维扩张，从而由维度代数定理，其为代数扩张。从而 $c\in \Q(u_1,...,u_n)$ 是代数元素

## 简单结论

- **（推论1.17）角等分**：60度角不可被尺规三等分
  - **证明**：反设可三等分
    - 首先，等边三角形是可构造的，故 $60\degree$ 的角也是可构造的，由假设此时 $20\degree$ 的角也是可构造的
    - 那么由规矩数的除法封闭性，此时 $20\degree$ 的三角函数也是可构造的
    - 再由三倍角公式得 $\dfrac{1}{2} = 4x^3 - 3x$ 存在实数根，与其既约矛盾
  - **推论**：实际上由多项式可直接得出，任意角都不可三等分
- **（推论1.18）立方复制**：单位立方体不可被尺规复制（不可作出体积为2的立方体）
  - **证明**：
    - 此时新立方体的边长 $s$ 是 $x^3 = 2$ 的解。该多项式在有理数域上既约，从而不是规矩数
- **化圆为方**：不可作出一个和给定圆 $C$ 面积相等的矩形
  - **证明**：
    - 该正方形的边长为 $\sqrt{\pi}r$，由于 $\pi$ 是 $\Q$ 的超越元素，故其不是规矩数，从而 $\sqrt{\pi}$ 也不是
- **分圆多项式的既约性**：若 $p$ 是素数，则分圆多项式 $f(X) = \sum\limits^{p-1}_{i=0} X^i$ 是既约的
  - **证明**：
    - 高代知识易得 $f(X) = \cfrac{X^p-1}{X-1}$
    - 取 $f(X+1)$，由艾氏判别法，其在 $\Q$ 上既约，从而 $f(X)$ 也既约
  - **推论**：
    - $\Q\fkh{e^{\dfrac{2\pi i}{p}}}$ 是 $\Q$ 的 $p-1$ 次扩张
