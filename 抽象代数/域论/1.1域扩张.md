# 代数扩张和超越扩张

- **符号约定**：
  - 用 $u$ 表示具体元，用 $x$ 表示不定元
    - 即 $K[x]$ 表示多项式环，$K[u]$ 表示单扩张环（虽然两者同构，但仍是不同概念）
  - 多项式 $f$ 默认是首一既约的
- **思想**：
  - 域为什么和多项式的根联系紧密呢？
    <!-- - 首先，多项式本身就是环中元素组合的方式，即环都是多项式环的子环
    - 通过构造干域，我们可将环作商得到可逆性，从而成为域。而决定可逆性的就是极小多项式的根 -->
    - 研究根本质是研究代数元素，而代数元素的环扩张就是域扩张

## 基本概念

- **域扩张的代数元素**：设域 $F\geq K$，若 $u\in F$ 是 $K[x]$ 上某多项式的根，则称为 $K$ 上的代数元素
  <!-- - **传递性**：子域传递到扩张 -->
- **域扩张的超越元素**：设域 $F\geq K$，若 $u\in F$ 不是 $K[x]$ 上某多项式的根，则称为 $K$ 上的超越元素
  <!-- - **传递性**：扩张传递到子域 -->
- **代数扩张**：每个 $F$ 元素都是 $K$ 上代数元素
  <!-- - **传递性**：扩张传递到子域 -->
- **超越扩张**：存在 $F$ 元素是 $K$ 上超越元素
  - **纯超越扩张**：扩张元素都是超越元素
  - **混合型扩张**：扩张元素既存在代数元素，又存在超越元素
  <!-- - **传递性**：子域传递到扩张 -->
- **实例**：
  - **自扩张**：已知 $\forall u\in K$ 都是多项式 $x-u\in K[x]$ 的根，从而是 $K$ 上代数元素
  - **经典数域**
    - $\Z$ 不是域
    - $\Q$ 是 $\Q$ 的代数扩张
    - $\R$ 及其父域是 $\Q$ 的超越扩张（$x^2 - 1 = 0$ 的根 $\sqrt{2}\notin\Q$，且 $\pi,e$ 是超越数）
    - $\R$ 及其父域是 $\R$ 的超越扩张
    - $\C = \R(i)$ 及其含R子域是 $\R$ 的有限维扩张（有限扩张、代数扩张）
  - **二次整数域**：
    - $\Q(\sqrt{m})\pad (m\in \Z)$ 是 $\Q$ 的代数扩张
  - **有限域的扩张**
  - **有理函数域**：环 $K[x_1,...,x_n]$ 的分式域就是 $K(x_1,...,x_n)$，元素为 $\dfrac{f}{g}$
    - $K\subsetneqq K(x_1,...,x_n)$，且差集为 $K$ 上超越元素


## 干域（系数域的代数单扩张）

- **极小多项式**：
  - 设 $\a$ 是域 $K$ 的代数元素
  - 则 $K[x]$ 中（以 $\a$ 为根的）（次数最小的）首一既约多项式称为（$\a$ 在 $K$ 上的极小多项式）
  - <font color='red'>环上的极小多项式未必能化为首一的，所以环中没有该概念</font>
- **干域定理**：
  - 设
    - $K$ 是域，$\a$ 是代数元素
    - $f$ 是 $\a$ 在 $K$ 上的极小多项式
  - 则 $F = K[\a]$ 是域，称为 $f$ 的干域 $(F,\a)$
  - <font color='red'>这里的 $f$ 由 $\a$ 决定，所以干域是不唯一的</font>
  - **证明**：
    - 只需证明 $\a$ 可逆即可
    - 设 $f(x) = x^n + c_{n-1}x^{n-1}... + c_0$
      - <font color='chartreuse'>由既约性，$c_0\neq 0$</font>
    - 易得 $\a^{-1} = -\cfrac{1}{c_0}(\a^{n-1} + c_{n-1}\a^{n-2} + ... + c_1)$
  - **理解**：
    - <font color='red'>实际上我们证明了代数元素满足 $K[\a] = K(\a)$
    - 本质是将 $f\in K[x]$ 商为 $0$，从而在环 $K[\a]\cong K[x]$ 上定义可逆性</font>
- **标准干域**：
  - 设 $f\in K[x]$ 是 $m$ 次首一既约多项式，$(f)$ 是 $f$ 生成的主理想
  - 若取 $F = K[x]/(f)，\a = x+(f)$，则 $(F,\a)$ 称为标准干域
  - <font color='red'>标准干域不依赖于外部元素 $\a$ 的选取，只依赖于 $K$ 和 $f$ 本身</font>
  - **证明**：
    - **$\a$ 是根**：$f(\a) \in (f)$，从而在 $F$ 中等价于0
    - **商环**：$\a = x+(f) \in K[x]/(f)$
    - **商映射**：$\p:K[X]\to K[\a]，f_0(X)\mapsto f_0(\a)$，其中 $\ker\p = f(X)$
      - **证明**：
        - $K[\a] \LR f_0(\a) = \sum\limits^{m-1}_{i=0} a_i\a^i =  \sum\limits^{m-1}_{i=0} a_i\Big(X+(f)\Big)^i$
        - 由于 $nX^k(f) = nX^k\Big[ K[X]中元素乘上(\sum\limits^{m-1}_{i=0} b_iX^i) \Big] \in (f)$
          - <font color='chartreuse'>也就是说，二项式的中间项均等于 $(f)$</font>
        - 故 $原式 = \sum\limits^{m-1}_{i=1} a_iX^i + (f) = f_0(X) + (f)  \LR K[X]/(f)$，从而是商映射
          - <font color='chartreuse'>也就是说，当 $f(\a) = 0$ 时，$K[\a] = K[X]/(f)$</font>
        - 由 $\p$ 定义，$\p(f(X)) = f(\a)$。再由 $\a$ 定义即得结论
    - **可逆性**：
      - **证明**：设 $f(X)$ 既约
        - 由裴蜀等式，$\forall g\in K[X]$，都有 $a(X)f(X) + b(X)g(X) = d(X)$
        - 由既约性，$d(X) \equiv 1$
        - 再由 $f(\a) = 0$，取 $X = \a$ 即得 $b(\a) = g(\a)^{-1}$
        - 再由 $g$ 任意性，即得 $K[\a] = K[X]/(f)$ 可逆
    - 综上，此时 $\Big(K[X]/(f), x+(f)\Big)$ 是域
  - **理解**：代数元素生成的环，由裴蜀等式可得可逆性
  - **本质**：
    - 将任意多项式环化为多项式域（干域），只需要将一些多项式视为等价，那么幺元 $1$ 就等价于无穷个任意次数的多项式，在此基础上，对任意多项式 $g$，总有一个适合它的 $(f)+1 = gh$，从而可构建出等价逆 $h$
     - 我们所做的只是规定某个既约的 $f$ 为0，它等价于用其根 $\a$ 代替自由未知数 $x$ 来组成多项式环（根的约束条件被转化为“相差 $f$ 的多项式等价”）。
    - 在后面的[代数扩张结构公式](#结构的性质)中，还会进行详细说明和总结

### 干域的性质

- **干域维度公式**：若 $f$ 是 $K$ 上的 $m$ 阶首一既约多项式，则 $K(\a) = K[x]/(f)$ 是 $m$ 维域扩张
  - **证明**：
    - 已知多项式均可被某个既约的 $f$ 用带余除法削至低于 $f$ 次数
    - 在干域中，我们规定某个既约的 $f$ 为0，则在带余除法过程中始终保持等价关系，最终每个多项式都等价于次数比 $f$ 小的多项式
    - 从而此时用 $m$ 个单项式 $\set{1_K,\a,...,\a^{m-1}}$ 线性组合，即可表出所有等价多项式
- **标准干域唯一性**：任意干域均关于 $K$ 扩域同构于标准干域
  - **证明**：
    - 扩域同构映射为 $\sigma:K[x]/(f)\to K[\a]，f_0(x) + (f) \mapsto f_0(\a)$
      - <font color='chartreuse'>实际上它就是 $\sigma:K[x]\to K[\a]，g\mapsto g(\a)$ 在商域上的限制</font>
    - **单射性**：
      - 反设不是单射，则存在 $f_1(\a) = f_2(\a)$，但 $f_1\neq f_2 + (f)$
      - 此时只能是 $f_1$ 和 $f_2$ 相差一个以 $\a$ 为根的多项式 $g$。但由 $f$ 最小性，此时 $f\mid g$，故 $g\in (f)$，矛盾
    - **满射性**：易得

### 干域实例

- **实多项式环的干域**：已知 $\R[x]$ 不是域
  - **求解**：易得 $\R$ 上的最小既约多项式为 $f(X) = X^2 + 1$，其根为 $i$，故干域为 $\R(i) = \R[X]/(f) = \C$
    - **代数单扩张**：$\R(i) = a_0 + a_1i + a_2 i^2+\cdots = a+bi$
      - <font color='chartreuse'>$i^2=-1$，变回实数，故合并后只剩下两项</font>
    - **标准干域**：$\R[X]/(f) = (a_0 + a_1X + a_2X^2 + \cdots )/(X^2+1) = a+bX$
      - <font color='chartreuse'>带余除法消去2次以上单项式，故最终只剩下两项</font>
    - 显然，取 $X = i$，即标准同构 $\sigma:\R[X]/(f)\to \R[i]$ 时，两者相等
  - **实例**：原不可逆多项式 $X+1 \red\Rt i+1$ 的逆为 $\dfrac{1}{i+1} = \dfrac{i-1}{2} \red\Rt \dfrac{1}{2}X-\dfrac{1}{2}$
- **复数域多项式的实数表示**：已知一个复系数多项式，求相同根的实系数多项式
  - **求解**：首先将根求出来，$\a_1,...,\a_n$
    - 然后将共轭根分别复合，若 $n$ 为偶数，则根均共轭。若 $n$ 为奇数，则存在一个根无共轭。这时我们添加其共轭根因子 $x-\ol\a_n$
    - 最终得到的就是实系数多项式
  - **实例**：$\Big[ x-(1+i) \Big]\Big[ x-(2+i) \Big] = x^2-(3+2i)x+(1+3i)$
    - 添加两个共轭因子 $x-(1-i)$ 和 $x-(2-i)$
    - 原式变为 $(x^2-2x+2)(x^2-4x+5)$
  - <font color='chartreuse'>可以应用到复相似矩阵求特征值上</font>
- **欧几里得算法求逆**：
  - **题设**：设 $f(X) = X^3-3X-1\in \Q[X]$，已知其既约
  - **分析**：$\Q[x] = \Q[X]/(f)$ 是 $\Q$ 上向量空间，基为 $\set{1,x,x^2}$
  - **求解**：
    - 设 $\b = x^4+2x^3+3\in \Q[x]$，用带余除法约去 $f(X)$ 后，得余项 $r = 3x^2+7x+5$。再用欧几里得算法，得 $f$ 与 $r$ 的裴蜀等式，从而得到 $r$ 的逆</font>


## 结构定理

- **（定理1.5）纯超越扩张结构自由性**：
  - 设域 $F\geq K$，$u\in F$ 是 $K$ 上超越元素，$x$ 是未知量
  - 则存在同构 $K(u)\cong K(x)$
  - <font color='red'>纯超越扩张就是普通的域生成扩张，代数扩张是可被简化的域生成扩张</font>
  - **证明**：
    - 取 $\p:K(x)\to F，f/g \mapsto f(u)/g(u)$
    - 由超越性，$\forall g(u)\neq 0$，从而是良定义的
    - 易得是域单同态。再由域的生成定理即得 $\Im\p = K(u)$，是满同态
  - **理解**：
    - 用超越元素取代未知数 $x$，可以达到一样的多项式环效果。比如在 $\Z[x]$ 或 $\Q[x]$ 中，$ax^2 + 1 = a\pi^2+1$
    - 或者说，若取 $\a$ 为超越元素，则取干域不会商去任何等价的多项式
- **（定理1.6）代数扩张结构公式**：
  - 设域 $F\geq K$，$u\in F$ 是 $K$ 上代数元素，$f$ 是极小多项式
  - 则 $K[u] = K(u)\cong K[x]/(f)$
  - **证明**：
    - 详见[干域部分](#干域)，或取商映射 $\p:K[x]\to K[u]，g\mapsto g(u)$ 即可

### 代数元素的次数

- **代数元素 $u$ 在 $K$ 上的次数**：
  - **扩张定义法**：$n = [K(u):K] = \deg f$
  - **基定义法**：$\{1_K,u,u^2,...,u^{n-1}\}$ 是 $K$ 作用的向量空间 $K(u)$ 的基
  - **表出定义法**：$K(u)$ 中元素均可唯一表为 $\sum\limits^{n-1}_{i=0} a_iu^i$
  - **证明**：
    - 已知 $\forall g\in K[x]$，由带余除法可设 $g = qf + r\pad (\deg r <\deg f)$
    - 两边取值得 $g(u) = q(u)f(u) + r(u)$。再由 $u$ 是 $f$ 的根，得 $g(u) = r(u)$，故 $\deg g(u) < \deg f$，再由 $g$ 任意性即得结论
  - **理解**：
    - <font color='red'>$K$ 中元素看作常数，可全部由基 $1_K$ 生成。$F\j K$ 中元素看作由基 $\{u,u^2,...\}$ 生成
    - 但若存在一个 $f$，使得 $f(u) = 0$，即基在线性组合 $f$ 下会归零，即在次数大于 $\deg f$ 的线性组合必定等价于某个次数小于 $\deg f$ 的线性组合。也就是说，此时（线性无关向量）基只能是 $\deg f$ 个，即扩张被限制成了有限维的</font>
- **根非基**：极小多项式 $f$ 的全部根 $\{u_1,...,u_n\}$ 不一定构成基
  - **反例**：
    - $\Q(\sqrt[3]{2})$ 中，$u = \sqrt[3]{2}$ 的极小多项式为 $x^3-2$，其根为两个 $2$ 次本原单位根，不是 $\Q(\sqrt[3]{2})$ 的基
  - **实例**：详见[有限域习题](15.域的分类.md)
- **根的共轭性**：既约多项式的根在分裂域中成对共轭出现
  - **理解**：以 $\C = \R(i)$ 为例（实际上在高代中已解释过一遍了）
    - $\R$ 中的既约多项式可写为 $a_nx^n+\cdots + a_1x = -a_0$，两边都没有含 $i$ 的系数，也就是说 $x$ 中的 $i$ 被左边一番运算后，最终结果为 $0i$，故将 $x$ 的 $i$ 全都取负后为 $-0i$，不改变结果。但将 $x$ 中的实数取负后会改变结果（除非 $a_0 = 0$），从而不是解。但是当 $a_0 = 0$ 时，存在解 $x=0$。将其约去后又转化为 $a_0 \neq 0$ 的情况。
    - 把 $i$ 替换成 $u$，就能把这个思想推广到所有单扩张中
- **根的本原性**：既约多项式的根在分裂域中都可被本原单位根构成的基表出
  - **证明**：根的共轭性 + 棣莫弗公式即可
  - **实例**：$x^2+3x+3 = 0$ 的根为 $\sqrt{3}\zeta_3，\sqrt{3}\zeta_3^2$

## 同构定理

- **（定理1.8）同构的域扩张延拓**：
  - 设 $\sigma:K\to L$ 是域同构，$u,v$ 分别是 $K,L$ 的扩域元素
  - 若满足两种情况之一
    - $u,v$ 都是超越元素
    - $u$ 是既约多项式 $f\in K[x]$ 的根，且 $v$ 是 $\sigma f\in L[x]$ 的根
  - 则 $\sigma$ 可延拓为域扩张同构 $K(u)\to L(v)，u\mapsto v$
  - <font color='red'>同构都可延拓到超越扩张中
  - 保根同构可延拓到代数扩张中</font>
  - **证明**：
    - 由同构易得 $\sigma$ 可延拓为 $\sigma':K[x]\cong L[x]$
    - 若 $u$ 是超越元素，则由结构性质，$\sigma'':K(x)\to L(x)，h/g\mapsto \sigma h/\sigma g$ 也是同构。从而 $\sigma(u)$ 是超越元素
    - 若 $u$ 是 $f$ 是首一既约多项式的根，同构式得 $\sigma f$ 也是首一既约多项式
      - 取 $\sigma'$ 在干域 $\Big(K[x]/(f),u\Big)$ 上的限制，即转化为超越元素情况，易得是同构。取复合即可
  - **推论**：环也成立
- **（推论1.9）干域同构定理**：
  - 设域 $E,F\geq K$，$u,v$ 分别是 $E,F$ 上的代数元素
  - 则 $u,v$ 都是既约多项式 $f\in K[x]$ 的根 $\LR K(u)\cong K(v)$
  - <font color='red'>同一多项式的干域在同构意义下唯一</font>
  - **证明**：
    - **必要性**：取 $\sigma:K(u)\to K(v)$ 为恒等映射即可
    - **充分性**：
      - 设 $\sigma:K(u)\cong K(v)，\sigma(u) = v$ 是K同构
      - 任取以 $u$ 为根的 $f\in K[x]$，设为 $\sum\limits^n_{i=0}k_ix^i$
      - 则 $0 = \sigma(0) = \sigma(f(u)) = \sum k_i\sigma(u)^i = f(v)$，从而 $v$ 也是 $f$ 的根
  - **本质**：取适当的同构，应用根的同构不变性定理即可

## 生成定理

- **（定理1.10）标准干域存在唯一性**：
  - 设 $K$ 是域
  - 则任取 $f\in K[x]$，都存在单扩张 $K(u)$ 满足以下三个条件
    - $u$ 是 $f$ 的根
    - $[K(u):K] \leq \deg f$，当 $f$ 既约时等式成立
    - 若 $f$ 在 $K[x]$ 上既约，则 $K(u)$ 可由某个K同构唯一决定
  - **证明（存在性）**：
    - 不妨设 $f$ 是最小既约多项式（若不是，则取其既约因式），则 $(f)$ 是 $K[x]$ 的最大理想，从而 $K[x]/(f)$ 是域
    - 取规范商映射 $\pi:K[x]\to K[x]/(f)$，只需存在 $u$ 使其为干域 $K$
      - 由于 $K$ 中只有常数多项式，故 $\pi(K)\cong K$，从而 $\pi(K[x])$ 是 $K$ 的扩张，平凡地取 $u = \pi(x)$（标准干域），就有 $f(u) = 0$
        - <font color='chartreuse'>（实际上，取任意 $u\in K[x]/(f)$ 均可达到目的。可以证明它们和标准干预同构。当然，这些并不是所有的干域，将 $K[x]/(f)$ 替换为其它的含有 $f$ 根的域，就可构造出另外的干域）</font>
        <!-- - <font color='chartreuse'>（详细证明步骤和[干域的商映射部分](#干域)相同）</font> -->
  - **证明（同构性）**：前面已证干域在同构下唯一
    <!-- - 详见[干域部分](#干域) -->
  - **理解**：
    - 条件中只给了 $f$，不确定它在 $K$ 中有没有根，即使有 $K$ 以外的根，也不清楚到底在哪个扩张域 $F$ 中
    - 我们这里实际是用了商映射 $\pi$ 来构造一个商等价意义下的根 $u \in F$
      - 为了方便讨论，先平凡地取一个扩张 $F = K[x]$
      - 构造商等价后有 $f=0$，也就是说 $\forall u\in F/(f)$ 都有 $f(u) = 0$。即在 $F/(f)$ 中总可找到一个 $f$ 的根 $u$
      - 再证明 $F/(f) = \pi(K[x])$ 也是 $K$ 的扩张，此时我们就得到了定义干域所需要的扩张，即包含一个 $f$ 的根 $u$ 的扩张
      - 那么，此时就可取干域 $\Big[ F/(f) \Big]/(f) = F/(f)$
  - **本质**：干域并不依赖于提前给出的扩张 $F\geq K$，我们有一套标准的构造方法
  - <font color='chartreuse'>代数基本定理进一步将 $\R$ 的扩张确定为 $\R(i)$</font>
- **（定理1.11）有限结构定理**：有限维扩张 $\LR$ 有限个代数元素生成的扩张 $\red\Rt$ 代数扩张
  - **证明**：
    - 设 $[F:K] = n$
    - 任取 $u\in F$，易得 $n+1$ 个元素 $\set{1_K,u,...,u^n}$ 必定线性相关，则 $\sum\limits^n_{i=0} a_iu^i = 0\pad (a_i\in K)$ 的系数可以不全为0，从而 $u$ 是它的根。再由 $u$ 任意性得是代数扩张
    - 由于 $F$ 是 $K$ 向量空间，故存在 $n$ 个基 $\{v_i\}^n_{i=1}$，从而 $F = K(v_1,...,v_n)$，是有限生成扩张
  - **理解**：实际上是把[代数扩张结构公式](#代数扩张结构公式)中的理解又叙述了一遍
  - **本质**：代数扩张结构公式的逆命题
  - **实例**：
    - $\C = \R(i)$ 是二维扩张，基为 $\{1,i\}$，且是代数扩张
      - <font color='chartreuse'>$i$ 是 $x^2+1$ 的根，详见前面[干域的例子](#干域)</font>
    - $\Q(\sqrt{2}) \geq \Q$ 是二维扩张，基为 $\{1,\sqrt{2}\}$，且是代数扩张
      - <font color='chartreuse'>$\sqrt{2}$ 是 $x^2-2$ 的根
      - $p+q\sqrt{2}$ 是 $\Big[ x-(p+q\sqrt{2}) \Big]\Big[ x-(p-q\sqrt{2}) \Big] = x^2-2px+(p^2-2q)$ 的根

- **（定理1.12）代数性的生成传递**：
  - 设域 $F\geq K$
  - 若存在 $X\subset F$ 满足 $F=K(X)$，且 $X$ 上元素均为 $K$ 中代数元素
  - 则 $F$ 是代数扩张
  - <font color='red'>无穷个代数元素生成的扩域也是代数扩张</font>
  - **证明**：
    - 任取 $v\in F$，可设 $v\in K(u_1,...,u_n)$
    - 由题设，$u_i$ 是代数元素，从而是 $K(u_1,...,u_{i-1})$ 的代数元素，设次数为 $r_i$
    - 由生成域的复合公式，$K(u_1,...,u_{i-1})(u_i) = K(u_1,...,u_i)$，存在扩张关系，由代数元素次数定义，$[K(u_1,...,u_i):K(u_1,...,u_{i-1})] = r_i$
    - 归纳得 $[K(u_1,...,u_i):K] = \prod r_i$，维度有限。则由维度代数关系，其为 $K$ 的代数扩张，再由任意性得 $F$ 也是
  - **理解**：由生成域的复合公式，可层层降阶。由代数扩张结构公式，降阶过程始终保持维度有限。由维度代数关系，是代数扩张
  - **本质**：就是上面的共轭法
- **（定理1.13）代数性的扩张传递**：若 $F$ 是 $E$ 的代数扩张，$E$ 是 $K$ 的代数扩张，则 $F$ 是 $K$ 的代数扩张
  - **证明**：
    - 设 $u\in F$ 是某个多项式 $\sum\limits^n_{i=0}b_iu^i\pad (b_i\in E)$ 的根
      - 显然其为 $K(b_1,...,b_n)$ 的代数元素，从而 $[K(b_1,...,b_n)(u):K(b_1,...,b_n)]$ 有限
    - 和生成传递性的证明过程相同，可得 $[K(b_1,...,b_i):K]$ 有限
    - 综上由维数公式得 $[K(b_1,...,b_i)(u):K]$ 也有限，由维度代数关系即得 $u$ 是代数元素。再由任意性即得结论

## 习题

- **多项式环是超越扩张**：$\forall f\in K(x_1,...,x_n)$，若 $f\notin K$，则是 $K$ 的超越元素
  - **证明**：利用超越元素结构自由性，归纳即可
- **中间环定理**：设 $K\leq F$ 是代数扩张，则任意中间环 $R$ 都是域
  - **证明**：
    - 由于 $R\subset F$，故易得交换性和无零因子性，从而是整环。只需证明 $R$ 的可逆性
    - 此时 $R$ 中元素均为代数元素，即 $K(\a)$ 是干域。再由 $\a\in R$ 即得 $K(\a)\in R$
    - 不断归纳即得 $K(\a) \leq K(\a,\b)\leq ... \leq R$，即 $R$ 是域
  - **反例（超越扩张不成立）**：
- **代数整环进化性**：设 $F$ 是域，$R\supset F$ 是整环。若 $R$ 是 $F$ 的代数元素组成的F代数，则其为代数扩张
  - **证明**：可设 $R = F[u_1,...,u_n]$。由代数元素性，可设 $u_i^{m_i} = \sum\limits^{m_i-1}_{i=0} a_i u_i^i$，则 $R$ 是F向量空间 $\span\{u_1^{i_1}\cdots u_n^{i_n}\mid \forall i_j < m_j，1\leq j\leq n\}$，其为有限维的F代数，从而由维度代数关系，$R$ 是代数扩张
- **超越退化的对称性**：
  - 设 $u,v$ 是 $K$ 的超越元素
  - 若 $v$ 是 $K(u)$ 的代数元素，则 $u$ 是 $K(v)$ 的代数元素
  - **证明**：
    - 此时存在 $f\in K(u)[x]$ 使得 $f(v) = 0$
      - 由 $v$ 超越性，$f$ 的系数必须包含 $u$，故可将 $f(v)$ 看作 $u$ 的多项式 $g$（此时 $v$ 变成系数），那么就有 $g(u) = 0$ 且 $g\in K(v)$
  - <font color='red'>由代数扩张的传递性，$u,v$ 必须都是超越元素，否则都是代数元素</font>

  - 设 $E_1,E_2\leq F$
  - 若 $E_1(X)$ 中元素均为 $E_2(X)$ 的代数元素，则 $E_1,E_2$ 同理
  - <font color='red'>域收缩不改变代数元素</font>
  - **反例（逆命题不成立）**：
- **多元维数公式**：设 $u,v$ 分别是 $K$ 上 $m,n$ 次代数元素，则 $[K(u,v):K]\leq mn$，当次数互素时取等
  - <font color='red'>一元代数扩张总取等，但多元代数扩张不一定取等</font>
  - **证明**：
    - 已知 $K(u,v) = K(u)(v)$，由中间域维数公式得 $[K(u,v):K] = \ffkh{K(u)(v):K(u)}\ffkh{K(u):K}$
      - 虽然 $v$ 在 $K$ 上是 $n$ 次代数元素，但在 $K(u)$ 上，多项式系数得到扩展，故 $v$ 的次数可能小于 $n$，故 $[K(u)(v):K] \leq n$
      - <font color='chartreuse'>或者我们可换一个视角，$K(u)$ 中 $\deg v < n \LR \lang v \rang \cong \lang u^i \rang$，那么互素时显然不可能有同构关系，故取等</font>

### 基的互素置换

- **互素置换引理**：若 $u$ 是 $K$ 上奇数次代数元素，则 $K(u) = K(u^2)$，从而 $u^2$ 也是奇数次代数元素
  - **证明**：
    - 设 $u$ 的最小多项式为 $f$，则 $\deg f = 2k+1$ 是奇数，$K(u)$ 有奇数个新基 $\{u,...,u^{2k+1}\}$
    - 由奇数与 $2$ 互素，故将 $u$ 替换成 $u^2$ 时基不变（置换），从而 $K(u) = K(u^2)$
- **互素置换定理**：设 $u$ 是 $K$ 上 $m$ 次代数元素，若 $m,n$ 互素，则 $K(u) = K(u^n)$
  - **证明**：同上
