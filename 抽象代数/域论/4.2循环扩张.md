# 循环扩张

## 范数和迹

- **代数元素的范数和迹**：设 $F\geq K$ 是有限维扩张，$\ol K\geq F$ 是代数闭包
  - 设 $\sigma_1,...,\sigma_r$ 是所有K单同态 $F\to \ol K$
  - 对 $u\in F$
    - **范数** $N_K^F(u) = (\sigma_1(u)\sigma_2(u)...\sigma_r(u))^{[F:K]_I}$
    - **迹**：$T_K^F(u) = [F:K]_I \Big(\sigma_1(u) + \sigma_2(u) + \cdots + \sigma_r(u) \Big)$
  - **等价定义**：$\ol K$ 换成正规扩张也成立
  - **实例**：$F = \C，K = \R$，则 $\R$ 单同态 $\C\to \C$ 只有恒等和共轭，此时 $N^{\C}_{\R}(a+b) = \Big[ (a+bi)(a-bi) \Big]^{1} = a^2+b^2$
  - **相似性**：迹是范数的加法形式，故证明其中一个的性质，另一个也可以类似证明
    - **反例**：若 $F$ 不是可分扩张，则 $\char K = p$，$[F:K]_I = p^t$。此时 $\forall u\in F，T^F_K(u) = 0$，但 $N^F_K(u)$ 不一定
- **（定理7.2）乘法加法公式**：
  - 设 $F\geq K$ 是有限维伽罗瓦扩张，$\aut_K F = \set{\sigma_1,...,\sigma_n}$
  - 则 $\forall u\in F，\begin{cases} N_K^F(u) = \sigma_1(u)\sigma_2(u)...\sigma_r(u) \\ T_K^F(u) = \sigma_1(u) + \sigma_2(u) + \cdots + \sigma_r(u) \end{cases}$
  - **证明**：
    - 由伽罗瓦扩张的正规性，$F$ 是 $K$ 的正规扩张，从而也是 $\ol K$ 的稳定中间域，从而K单同态 $F\to \ol K$ 都在 $\aut_K F$ 中
    - 由伽罗瓦扩张的可分性，$[F:K]_I = 1$
    - 综上，由定义可直得结论
- **（定理7.3）其它公式**：设 $F\geq K$ 是有限维扩张，则 $\forall u,v\in F$
  - **同构公式**：$\begin{cases} N_K^F(u)N_K^F(v) = N_K^F(uv) \\\\ T_K^F(u) + T_K^F(v) = T_K^F(u+v) \end{cases}$
    - **证明**：定义直得
  - **指数乘法公式**：若 $u\in K$，则 $N_K^F(u) = u^{[F:K]}，T_K^F(u) = [F:K]u$
    - **证明**：定义直得
  - **根式公式**：
    - 设 $f = x^n + a_{n-1}x^{n-1} + ... + a_0\in K[x]$ 是 $u$ 的既约根式
    - 则 $\begin{cases} N_K^F(u) = \Big((-1)^n a_0\Big)^{[F:K(u)]}\in K \\\\  T_K^F(u) = -[F:K(u)]a_{n-1}\in K \end{cases}$
    - **证明**：
      - 设 $E = K(u)$，则 $\ol K$ 也是 $E$ 的代数闭包，则K单同态 $F\to \ol K$ 都可写成 $\sigma_i\tau_j$ 的形式，故由正规扩张的可分次数性 $t = [E:K]_S$，则 $n = t[E:K]_I$
      - 由可分维数公式，再由 $\sigma_i:K(u)\cong K(\sigma_i(u))$ 的像均为不同根，此时既约根式为 $f = \Big[ x^t - (\sum\limits^t_{k=1}\sigma_k(u))x^{t-1} + ... + (-1)^t\sum\limits^t_{k=1}\sigma_k(u) \Big]^{\large [E:K]_I}$
      - 若 $[E:K]_I = 1$，则 $n = t$，结论易得
      - 若 $[E:K]_I > 1$。此时容易计算 $a_0，a_{n-1}$，从而得到结论
  - **中间域传递公式**：设 $E$ 是中间域，则 $\begin{cases}  N_K^E\Big( N^F_E(u) \Big) = N_K^F(u) \\\\ T_K^E\Big( T^F_E(u) \Big) = T_K^F(u) \end{cases}$
    - **证明**：
- **（引理7.5）自同构线性无关性**：
  - 设 $S\subset \Aut F$
  - 则 $\forall a_i\in F，\sigma_i\in S，u\in F$，非零式 $\sum\limits^n_{i=1} a_i\sigma_i(u) = 0\red\Rt\forall a_i = 0$
  - **证明**：

## 循环扩张

- **循环扩张**：$F\geq K$ 是代数伽罗瓦扩张，且 $\aut_K F$ 是循环群
  - **$n$ 次循环扩张**：循环群阶数为 $n$，当然也有 $[F:K] = n$
- **阿贝尔扩张**：$F\geq K$ 是代数伽罗瓦扩张，且 $\aut_K F$ 是阿贝尔群
- **（定理7.6）平凡条件**：设 $F\geq K$ 是 $n$ 次循环扩张，$\sigma$ 是 $\aut_K F$ 的生成元，$u\in F$，则
  - **迹平凡条件**：$T_K^F(u) = 0 \LR \exists v\in F，u = v-\sigma(v)$
  - **范数平凡条件**：$N_K^F(u) = 1_K \LR \exists v\in F，u = v\sigma(v)^{-1}$（Hilbert）
  - **证明**：
- **（命题7.7）循环扩张的中间域**：
  - 设 $F\geq K$ 是 $n$ 次循环扩张，$\char K = p，n = mp^t，(m,p) = 1$
  - 则存在中间域链 $F\geq E_0\geq \cdots \geq E_t = K$，其中 $F$ 是 $E_0$ 的 $m$ 次循环扩张，$E_{i-1}$ 是 $E_i$ 的 $p$ 次循环扩张
  - **证明**：
- **（命题7.8）循环扩张的分裂性**：设 $\char K = p$，则
  - $F$ 是 $p$ 次循环扩张 $\LR F$ 是 $K$ 上 $x^p-x-a\in K[x]$ 的分裂域
  - 此时 $F = K(u)$，$u$ 是它的根
  - **证明**：
- **（推论7.9）循环多项式**：设 $\char K = p，x^p-x-a\in K[x]$，则其要么既约，要么分裂
  - **证明**：
- **（引理7.10）单位根**：设 $n$ 是正整数，$K$ 是包含 $n$ 阶本原单位根的域
  - 若 $d\mid n$，则 $\z^{n/d} = \eta$ 是 $K$ 上的本原 $d$ 次单位根
  - 若 $d\mid n，u$ 是 $x^d-a = 0\in K[x]$ 的根，则 $x^d-a$ 有 $d$ 个不同的根 $u,\eta u,...,\eta^{d-1}u$
    - 此时 $K(u)$ 是分裂域和伽罗瓦扩张
- **（定理7.11）本原单位根的循环性**：设 $n$ 是正整数，$F\geq K$，$\zeta\in K$ 是本原单位根，则下列命题等价
  - $F$ 是 $d$ 度循环扩张，$d\mid n$
  - $F$ 是 $K$ 上 $x^n-a\in K[x]$ 的分裂域，单根扩张
  - $F$ 是 $K$ 上 $x^d-a\in K[x]$ 的分裂域，单根扩张
  - **证明**：

## 习题

- 范数和迹的新定义
- 设 $F$ 是有限域 $K$ 的有限维扩张，则范数 $N_K^F$ 和迹 $T_K^F$ 是满射
- 设 $v\in \ol \Q-\Q$，$E$ 是使得 $v\notin E$ 的最大 $\Q$ 子域，则其有限维扩张均为循环扩张
- 设 $\sigma\in \aut_K \ol K$，$F = \{u\in\ol K\mid \sigma(u) = u\}$，则 $F$ 是域，且有限维扩张均循环
  - **证明**：
- 设 $F\geq K$ 是 $p^n$ 次循环扩张，$L$ 是中间域
  - 若 $F = L(u)，L\geq K$ 是 $p^{n-1}$ 次循环扩张，则 $F = K(u)$
  - **证明**：
- 若 $\char K = p$，设 $K_p = \{u^p-u\mid u\in K\}$
  - 存在 $p$ 次循环扩张 $F$ $\LR K\neq K_p$
    - **证明**：
  - 若存在 $p$ 次循环扩张，则 $\forall n\geq 1$，存在 $p^n$ 次循环扩张
    - **证明**：对 $n$ 归纳
      - 若 $E\geq K$ 是 $p^n$ 次循环扩张，$\aut_K E = \lang \sigma \rang$，则存在 $u,v\in E$ 满足 $T_K^E(v) = 1_K$，且 $\sigma(u) - u = v^p-u$
      - 则 $x^p-x-u\in E[x]$ 是既约的
      - 若 $w$ 是根，则 $K(w)$ 是 $p^n$ 次循环扩张
- 若 $\char K \neq 2$，且 $K$ 有奇数 $n$ 次本原单位根，则其也有 $2n$ 次本原单位根
  - **证明**：
- 若 $F\geq \Q$ 是有限维扩张，则 $F$ 只有有限个本原单位根
  - **证明**：
- 若 $K$ 满足以下条件之一，则 $f(x) = x^p-a\in K[x]$ 要么既约，要么分裂
  - $\char K = p$
  - $\char K \neq p$ 但含有 $p$ 次本原单位根
    - **推论**：此时对任意 $f(x)$ 的根 $u$，都有 $K(u)\neq K(u^p) \LR [K(u):K] = p$
