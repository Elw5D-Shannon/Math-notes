## 基本概念

### 域扩张

- **域 $K$ 的扩张**：设域 $K$ 是域 $F$ 的子域，则称 $F$ 是 $K$ 的扩域
- **扩张的维数**：若 $F\geq K$，则 $F$ 是 $K$ 作用的向量空间，维数写为 $[F:K]$
  - <font color='chartreuse'>即加法群的陪集指数
  - 基中的 $1_K$ 就可以表出全部的 $K$，其它基元素都是用来表出 $F\j K$ 的</font>
- **有限扩张**：维数有限的扩张
  - **实例**：$\C$ 是 $\R$ 的 $2$ 维有限扩张
- **无限扩张**
  - **实例**：
    - $\R$ 是 $\Q$ 的无限扩张
      - **证明**：$\Q$ 可数，故其有限扩张均是可数集的有限直积，当然也可数
    - $\forall X\subset K，K(X),K[X]$ 都是无限扩张
      - **证明**：基为 $1,X,X^2,...$，从而无穷
- **（定理1.12）中间域维数公式**：$[F:K] = [F:E][E:K]$
  - **证明**：模的维数公式直得


### 域代数

- **域 $K$ 的代数**：设域 $K$ 是环 $R$ 的子环，则称 $R$ 是 $K$ 域代数
  - **环代数性**：其也是环意义下的代数（易得模性、数乘交换性）
  - **进化性**：整环 + 有限的域(作用)代数 = 域
    - **证明**：只需可逆性，定义易得
    - 有限扩张的子环是域


### 域同态

- **域同态**：域作为交换除环的环同态
- **扩域同态**：设域 $E,F\geq K$，若同态 $\sigma:E\to F$ 在 $K$ 上恒等，则称为关于 $K$ 的扩域同态
  - **模定义法**：域同态 + $K$ 模同态
    - **证明**：
      - 域同态保幺元，K模同态可提取 $k\in K$ 元素
      - 故 $\sigma(k) = k\sigma(1_E) = k1_F = k$
  - **代数定义法**：域 $K$ 作用的代数的同态
    - **证明**：定义易得
  - **实例**：
    - 设 $K$ 是域，$F = K(x)$，则对任意非零元 $a\in K$
      - **数乘自同构**：$\sigma_a:F\to F，f(x)/g(x)\mapsto f(ax)/g(ax)$ 是K自同构
      - **加法自同构**：
    - $\C$ 中共轭映射是 $\R$ 自同构
    - $\C(x)$ 中，$\sigma(x) = \dfrac{ax+b}{cx+d}$ 是 $\C$ 自同构
      - **推论**：$\aut_\C \C(x) \cong GL_2(\C)$
        - **理解**：已知黎曼球面 $\P^1_\C$ 上的亚纯函数 $\C(x)$ 同构于 $z$ 的有理函数 $\C(z)$，故自然映射 $\aut(\P^1_\C) \to \aut_\C \C(x)$ 是同构
    - **Cremona群**：


### 域生成

- **子集生成的子域**：$F$ 上所有包含 $X$ 的域的交
  <!-- - **子集生成的子环**：$F$ 上所有包含 $X$ 的环的交 -->
- **子集在扩域上生成的子域**：设域 $F\geq K，X\subset F$，则 $K\cup X$ 生成的子域写为 $K(X)$
- **子集在扩域上生成的子环**：设域 $F\geq K，X\subset F$，则 $K\cup X$ 生成的子环写为 $K[X]$
    - <font color='chartreuse'>易得其为整环</font>
- **有限生成扩张**：若 $X= \{u_1,...,u_n\}$，则 $K(u_1,...,u_n)$ 称为 $K$ 的有限生成扩张
  - **单扩张**：若 $X = \{u\}$，则 $K(u)$ 称为单扩张
- **（定理1.3）生成环【域】的多项式性**：
  - 设域 $F\geq K$，$u,u_i\in F$，$X\subset F$
  - 则<a id = "生成域的多项式性"></a>
    - 子环 $K[u]$ 由所有 $K$ 上多项式的像 $f(u)$ 组成<font color='chartreuse'>（即 $f\in K[x]$）</font>
    - 子环 $K[u_1,...,u_m]$ 由所有 $K$ 上m元多项式组成<font color='chartreuse'>（即 $g\in K[x_1,...,x_m]$）</font>
    - 子环 $K[X]$ 由所有 $K$ 上 $X$ 中元素的所有多项式组成<font color='chartreuse'>（即 $h\in K[x_1,...,x_n]$）</font>
    <br>
    - 子域 $K(u)$ 由所有 $K$ 上单项式的分数组成 <font color='chartreuse'>即（$f(u)/g(u) = f(u)g(u)^{-1}$）</font>
    - 子域 $K(u_1,...,u_m)$ 由所有 $K$ 上多项式的分数组成
  - **证明（仅以最后一个举例）**：
    - 易得每个包含 $X,K$ 的域都包含多项式的分式集 $E$，即 $K(X)\supset E$
    - 易得多项式的分式集 $E$ 是加法群和乘法群。再由可逆性和交换性得是域，再由 $K,X\subset E$ 得 $K(X)\subset E$
  - **理解**：
    - <font color='red'>用向量空间来理解域生成，就是把原域 $K$ 中的元素看作常数，基为 $1_K$，把新元素 $u\in X$ 看作基 $\{u,u^2,...\}$。基通过四则运算来生成所有新元素，最终可写成有理分式的形式
    - 域中的多项式 $f(u)\in K[u]$ 相当于基的线性组合结果，多项式函数 $f\in K[x]$ 相当于基的线性组合方式</font>
- **有限生成覆盖性**：任取 $v\in K(X)$ 或 $K[X]$，都存在有限集 $X'\subset X$ 使得 $v\in K(X')$ 或 $K[X']$
  - <font color='red'>（全体有限集生成的域）可覆盖整个域</font>
  - **证明**：由于 $v$ 的次数有限，显然其系数集是有限集，从而也是有限生成集
- **生成域的复合公式**：$K(X,x) = K(X)(x)$
  - **证明**：
    - 左边是 $\sum\limits^n_{k=1} a_{k_1k_2}X^{k_1}x^{k_2}$ 相除，右边是 $\sum\limits^m_{i=1}(\sum\limits^n_{j=1} a_jX^j)x^i$ 相除<font color='chartreuse'>（$m,n\in\N^+$）</font>，显然它们展开后相等，故相除得到的分式也相等

### 生成并

- **生成并**：设 $L,M$ 是 $F$ 的子域，则 $LM$ 是 $L\cup M$ 生成的子域
  - <font color='red'>（当生成集 $X$ 是子域时，生成关系就是相乘关系）</font>
- **生成并的对称性**：$LM = L(M) = M(L)$
  - **证明（互包）**：
    - 易得 $L(M) = \hkh{\cfrac{\sum\limits^n_{i=0} l_{i_1,...,i_m} m_1^im_2^i...m_m^i}{\sum\limits^n_{i=0} l_{i_1,...,i_m} m_1^im_2^i...m_m^i}\biggm| l_{i_1,...,i_m}\in L，m_{j}\in M}$
      - 由域的加乘除幂的封闭性，可得 $L(M)\subset LM$
    - $LM = \set{lm\mid l\in L，m\in M}$
      - 显然 $L(M)$ 中取单项式即可表出 $lm$，从而 $LM\subset L(M)$
    - 另一个同理
  - **本质**：<font color='red'>由于域的良好封闭性，多个元素线性组合（右）等价于两个元素相乘（左）</font>

### 习题

- **生成并的传递性**：若 $K\leq L\cap K(S)，S\subset K(S)$，则 $L\cdot K(S) = L(S)$
  - **证明**：本质是 $LKS = LS$
  - **实例**：$K(u,v) = K(u)(v) = K(u)K(v)$
  
### 生成并的性质

- $[LM:K]$ 有限 $\LR [L:K],[M:K]$ 有限
  - **证明**：
    - **必要性**：显然左边比右边两个都大，故结论易得
    - **充分性**：
      - 由题设，在 $K$ 作用下，$L,M$ 的基均有限，易得这两个有限基的多项式组合也是有限的，只需证明是 $LM$ 在 $K$ 作用下的基
      - 任取 $\sum\limits^n_{i=0} l_im^i\in LM$，表为基形式即可
- **维数公式**：$[LM:K]\leq [L:K][M:K]$，
  - **证明**：
    - 转化为基公式 $|B_L\cup B_M| \leq |B_L||B_M|$ 即可
    - 若互素，反设 $L,M$ 有共同中间域 $E$，即 $L\cap M = E\geq K$
      - 则 $[L:K] = [L:E][E:K]，[M:K] = [M:E][E:K]$，与互素矛盾，故只能是 $L\cap M = K$
    - 此时可设 $L = K(x_1,...)，M = K(y_1,...)$，由无共同中间域性，易得基集合不交，故可取等
  - **推论1**：两维数互素时取等，且 $L\cap M = K$
  - **推论2**：若 $L\cap M = K$，则仅当其中一个维数为 $2$ 时等号成立
  - **反例**：
    - $2$ 次本原单位根中，$\Q(\sqrt[3]{2})$ 和 $\Q[\zeta]$ 都是 $3$ 次扩张，交集为 $\Q$，但乘积 $\Q(\sqrt[3]{2},\z) = \Q(\z)$，是 $3$ 次扩张
- **代数扩张传递性**：若 $L,M$ 都是代数扩张，则 $LM$ 也是
  - **证明**：
    - 基都是根，故基的并集也是根
