# 求根公式

- **符号约定**：
  - $p$ 默认是素数，$n$ 默认是正整数

## 有限域

- **有限域（伽罗瓦域）**：元素数量有限的域
  - **表示方法**：$n$ 个元素的有限域用 $F_n$ 或 $GF(n)$ 表示
  - **素同构性**：$n$ 为素数 $\LR F_n \cong \Z_n$
    - **证明**：由数论知识，同余类中只有缩系才有逆，只有素数同余类的缩系为其本身

### 素域

- **素域**：不含真子域的域
  - **分类**：素域都同构于 $\Z_p$ 或 $\Q$
  - **因子性**：每个域都包含一个素域
  - **特征性**：$F$ 的特征为 $p$ 【$0$】 $\LR F$ 中存在同构于 $\Z_p$ 【$\Q$】 的素域
  - **新生之梦**：域的Frobenius自同态是单射，有限域则进化为自同构
  - <font color='chartreuse'>素域就是域的素理想(最大理想)的商环</font>
- **（定理5.1）素域最小性**：设 $F$ 是域，$P$ 是其所有子域的交，则 $P$ 是域，且没有真子域
  - 若 $\char F = p$，则 $P\cong \Z_p$
  - 若 $\char F = 0$，则 $P\cong \Q$
  - **证明**：易得 $P$ 的生成元只能是 $0,1_F$
    - 再由域的运算封闭性
      - 若 $\char P = p$，则 $P = \{m1_F\mid 0\leq m\leq p\}$
      - 若 $\char P = 0$，则 $P = \set{\dfrac{m1_F}{n1_F}\mid m,n\in\Z}$
    - 此时环同态 $\p:\Z\to P，m\mapsto m1_F$ 中，易得 $\ker\p$ 是 $\char P$ 生成的主理想
      - 若 $\char P = p$，则 $\Z_p \cong \Z/\ker\p \cong \Im\p \subset P$。由于 $\Z_p$ 是域，且 $P$ 没有真子域，则只能是 $\Z_p\cong P$
      - 若 $\char P = 0$，则其为单同态，已知其可延拓到分式域 $\Q$ 上，易得此时为满射，即 $\Q\cong P$
  - **理解**：大概就是先阐述一下 $P$ 的集合写法，发现 $\Z_p$ 和 $\Q$ 都只能嵌入 $P$。再用环同态的单射性证明同构性
  - **本质**：最小的有特征域是 $\Z_p$，最小的零特征域是 $\Q$

### 有限域的结构

- **（推论5.2）有限域的结构**：设 $F$ 是有限域
  - 则存在素数 $p$ 满足 $\char F = p$，且 $|F| = p^n$，$F\cong \sum\limits^n_{i=1} \Z_p$
  - **证明**：
    - 已知整环的特征为素数，故 $\char F = p$，即 $P\cong \Z_p$
    - 再由 $F$ 是 $P$ 的有限维向量空间，由向量空间自由性得 $F \cong \sum\limits^n_{i=1} \Z_p$
  - **理解**：
    - 有限域 $F$ 必定是代数扩张，从而其结构是明晰的，找到一个最小的素域 $P$，然后逐步扩张即可得到 $F$ 的结构
    - 素域 $P$ 可看作平凡域 $\Z_2 = \{0,1\}$ 关于 $p$ 原根的代数扩张，而有限域 $F_{p^n}$ 是素域 $P$ 的直和
- **（定理5.3）有限域乘法结构**：有限域上所有非零元构成的乘法群 $G$ 是循环群
  <!-- - 设 $F$ 是域，$G$ 是 $F$ 上乘法群的有限子群，则 $G$ 是循环群 -->
  - **证明**：
    - 易得 $G$ 是有限交换群，则由不变分解 $G \cong \sum\limits^k_{i=1} \Z_{m_i}$，且 $m_i$ 构成整除链
      - <font color='chartreuse'>交换的乘法群同构于加法群</font>
    - 故 $\forall u\in G，u^{m_k} = 1$，即 $G$ 中元素均为 $x^{m_k}-1_F \in F[x]$ 的根
    - 再由该多项式的根最多为 $m_k$ 个，而 $|G| = \sum\limits^k_{i=1}m_i$，故只能是 $k=1$，从而 $G$ 是循环群
  - **理解**：
    - 若 $F_p = \Z_p$。已知 $p^n$ 有原根，故 $\Z_{p^n}$ 的乘法群是原根生成的循环群。
  - **本质**：有限域乘法群的元素同构于本原单位根，而后者具有循环性

### 有限域的性质

- **（推论5.4）有限域单扩张性**：若 $F$ 是有限域，则 $F$ 是其素子域 $\Z_p$ 的单扩张
  - **证明**：取 $u$ 是乘法循环群 $G$ 的生成元即可
- **（引理5.5）新生梦**：
  - 设 $\char F = p$，$r\in\N^+$
  - 则 $\p:F\to F，u\mapsto u^{p^r}$ 是 $\Z_p$ 同态
    - 若 $F$ 有限，则是同构
  - **证明**：由新生之梦直得同态的加法乘法公式
- **（命题5.6）有限域分裂性**：$F$ 是 $p^n$ 阶有限域 $\LR F$ 是 $\Z_p$ 上 $x^{p^n}-x$ 的分裂域
  - **证明**：
    - **必要性**：此时 $|G| = p^n-1$，从而每个 $u\in G$ 都是 $x^{p^n-1}-1_F$ 的根。再想要使 $0$ 为根的话，只需变为 $x(x^{p^n-1}-1_F)$
    - **充分性**：
      - 易得 $f' = -1$，从而 $(f,f') = 1$，从而 $f$ 无重根，即有 $p^n$ 个不同根
      - 设 $\p$ 是新生梦单同态，则 $u\in F$ 是 $f$ 的根等价于 $u$ 是 $\p$ 的不动点
      - 由根的四则运算封闭性，$f$ 的根集 $E$ 是 $p^n$ 阶域。再由 $\Z_p$ 是最小域得 $E\geq \Z_p$。再由 $F$ 是分裂域，定义即得 $F = \Z_p(E) = E$
  - **理解**：
    - 分裂域就是所有根生成的域。而有限域的循环结构使得其等价于（本原多项式的分裂域）
    - 有趣的是，分裂域的维数和根的数量无关，但其阶数却等于根的数量。
- **（推论5.7）有限域存在唯一性**：对任意素数 $p$ 和正整数 $n$，都存在 $p^n$ 阶有限域，且在同构意义下唯一
  - **证明**：
    - 由分裂域存在性，$\Z_p$ 上 $x^{p^n}-x$ 的分裂域总存在
    - 再由分裂域等价于 $p$ 域 + 有限次多项式的分裂域唯一性即得结论
- **（推论5.8）有限维扩张存在唯一性**：
  - 设 $K$ 是有限域
  - 则对 $\forall n\in\N^+$，总存在有限 $n$ 维扩张，且是单扩张，且在K同构意义下唯一
  - **证明（存在性）**：
    - 设 $K$ 的阶为 $p^r$，设 $F$ 是 $K$ 上 $f_{nr}(x) = x^{p^{nr}}-x$ 的分裂域
      - 由有限域分裂性，$F$ 为 $p^{nr}$ 阶有限域
      - 由 $\Z_p$ 的最小性，$K,F$ 分别是 $\Z_p$ 上 $f_r,f_{nr}$ 的分裂域，由 $p^r,p^{nr}$ 个不同的本原单位根组成
      - 再由分裂域的定义，扩域本原单位根的数量 $=$ 基域本原单位根的数量 ** 基域中基的次数<font color='chartreuse'>（什么垃圾描述，直接看下面的理解得了）</font>
      - 综上，$p^{nr} = |F| = |K|^{[F:K]} = (p^r)^{[F:K]}$，即 $[F:K] = n$
      - 再由有限域的单扩张性即得 $F$ 是 $K$ 的单扩张
  - **证明（唯一性）**：
    - 反设存在另一个 $F_1$，则 $[F_1:\Z_p] = n[K:\Z_p] = nr$，从而 $|F_1| = p^{nr}$
    - 由有限域分裂性 + 有限次多项式分裂域唯一性，即得 $F_1$ 和 $F$ 在同构下等价
  - **理解**：本质是两个分裂域的阶具有指数关系
    - 另一种想法是寻找本原单位根的空隙。因为 $K,F$ 分别对标 $\z_r, \z_{nr}$，而本原单位根满足 $\z_{nr}^n = \z_r$，从而可立刻得到 $[F:K] = n$
- **（推论5.9）有限域多项式存在性**：设域 $K$ 有限，则对 $\forall n\in\N^+$ 均存在 $f\in K[x]$ 是 $n$ 次既约多项式
  - **证明**：由有限维扩张存在唯一性 + 代数扩张结构公式直得
- **（命题5.10）有限维扩张循环性**：
  - 设 $F\geq K$ 是有限域的有限维扩张
  - 则 $F$ 有限，且是伽罗瓦扩张，且 $\aut_K F$ 是循环群
  - <font color='red'>即有限域的有限维扩张都是循环扩张</font>
  - **证明**：
    - 设 $\Z_p$ 是 $K$ 的素子域，则 $F\geq \Z_p$ 是有限维扩张，维数设为 $n$，则 $|F| = p^n$
