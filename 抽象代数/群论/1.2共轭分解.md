# 共轭

## 共轭等价

- **子群的共轭**：
  - 设 $A,B\leq G$
  - 若 $\exist g\in G$ 使得 $g^{-1}Ag = B$，则称 $A$ 与 $B$ 共轭
  - <font color='red'>陪集类似高代的商空间，$A$ 是商平面，$g$ 是商平面的平移向量
  - 共轭类似高代的相似矩阵，$A$ 是矩阵，$g$ 是相似</font>
- **元素的共轭**：
  - 设 $a,b\in G$
  - 若 $\exists g\in G$ 使得 $g^{-1}ag = b$，则称 $a$ 与 $b$ 共轭
- **共轭等价**：
  - **自反性**：$e^{-1}ae = a$
  - **交换性**：$gbg^{-1} = a$
  - **传递性**：$(g_1g_2)^{-1}a(g_1g_2) = c$
- **共轭等价类**：$\bar x = \{gxg^{-1}\mid g\in G\}$
- **阶数相等性**：$|B| = |g^{-1}Ag| = |A|$
  - <font color='chartreuse'>详细解释见[群作用](./5.有限群的结构.md)章节</font>
- **自同构性**：共轭作用是群的一种自同构
  - **实例**：
    - 矩阵的相似
- **无遗传性**：
  - 设 $G$ 是有限群，$P,Q\leq H \leq G$
  - 若 $P,Q$ 在 $G$ 中共轭，其不一定也在 $H$ 中共轭
  - **反例**：
    - $G$ 是对称群 $S_3$，$H$ 是交错群 $A_3$
    - 则置换群 $P = \{(1),(12)\}，Q = \{(1),(13)\}$ 不满足遗传性

## 共轭不变量

### 共轭不变的子群

- **正规子群**：$N = \{N\subset G\mid \forall g\in G，g^{-1}Ng = N\}$ 称为正规子群
  - **符号**：$N\lhd G$
  - <font color='red'>（作为一个整体进行运算时可交换）的子群
  - 子群共轭等价类的幺元</font>
  - **唯一性**：显然
  - **性质**
    - **自共轭性**
    - **整体可交换性**：$\forall K\leq G，NK = \lang N\cup K \rang = KN$
      - **证明**：定义直得
      - **推论**：正规子群的左右陪集相等
    - **商群性**：若 $N_1\cap N_2 = \lang e \rang$，则运算可交换，但彼此不互为子群
      - **证明**：见[本节习题]()
      - **理解**：这其实就是后面介绍的[商群](./2.群的映射.md)等价类
  - **传递性**：
    - **无遗传性**：
      <!-- - **反例**：$Z_4$ 中，$H=\{0,2\}\lhd K=\{0,1,2\} \lhd Z_4$
        - $2+3+2 = 3\notin H$
      - **理解**：H是空隙群，而K是退化群，在退化群中空隙不会显露出来。而在更大的群中运算会进入空隙（有点像拓扑基） -->
      - **反例**：$D_4$ 中，$\lang a^2 \rang\lhd\lang a \rang\lhd D_4$
        - $a^2 = (24)(13)$，其中 $ba^2b^{-1} = (13)(24)\notin \lang a^2 \rang = \{(24)(13),(1)\}$
    - **扩张性**：设 $K\leq H\leq G$，若 $K\lhd G$，则 $K\lhd H$
      - **证明**：定义显然
    - **限制传递性**：若 $N\lhd G，K\leq G$，则 $(N\cap K)\lhd K$
      - **理解**：
    - **夹逼性**：设 $N\leq K$，若 $N,K\lhd G$，则 $N\lhd K$
      - **证明**：限制传递性直得
    - **交传递性**：正规子群的任意交也正规
  - **实例**：
    - $2$ 阶子群均为正规子群
    -  $SL_n(\R)$ 是 $GL_n(\R)$ 的一个正规子群
- **正规化子**：设 $M\leq G$，则 $N_G(M) = \{g\in G\mid g^{-1}Mg = M\}$ 称为 $M$ 的正规化子
    - <font color='red'>使得 $M$ 在其中正规的最大子群</font>
  - **包含关系**：$M\lhd N_G(M) \leq G$
  - **弱交换性**：正规化子与 $M$ 本身可交换，但不可和 $M$ 的元素交换
  - **子群性**
    - **证明**：
      - **含幺性**：显然
      - **可逆性（可交换性）**：
        - 设 $g$ 是 $M$ 的正规化子，则 $gM = Mg$
        - 添项得 $gMg^{-1} = M$，即 $g^{-1}$ 也是 $M$ 的正规化子
- **范群**：全体子群的正规化子的交集
  - **正规性**：$\mc N(G)\lhd G$

### 共轭不变的元素

- **中心**：$C(G) = \{g\in G\mid ga = ag,\forall a\in G\}$
  - <font color='red'>$G$ 中可交换的全体元素
  - 元素共轭等价类的幺元</font>
  - **子群性**
    - **证明**：
      - **含幺性**
      - **可逆性**
    - <font color='red'>中心的大小反映了群的可交换程度</font>
    - **实例**：$S_n$ 的中心只有恒等映射（[置换不可交换性](2.群的映射.md)）
  - **中心元素**：
    - **自共轭性**：中心元素只共轭于自身
    - **实例**：$GL_n(\R)$ 中的 $kI_n$
- **中心化子**：$G$ 的子群 $C_G(M) = \{g\in G\mid g^{-1}ag = a,\forall a\in M\}$
  - <font color='red'>使得 $M$ 是其中心的最大子群</font>
  - **子群性**
    - **证明**：
      - **含幺性**：显然
      - **可逆性（可交换性）**：
        - 设 $g$ 是 $M$ 的中心化子，则 $gm = mg$
        - 显然 $gmg^{-1} = m$，即 $g^{-1}$ 也是 $M$ 的正规化子
  - **交集定理**：$C(G) = \mathop{\bigcap}\limits_{g\in G} C_G(g)$
    - <font color='red'>中心是全体中心化子的交</font>
    - **证明**：显然
  - **比较定理**：中心化子是正规化子的子群
    - **证明**：定义易得

## 共轭的性质

- **子群共轭指数**：$M$ 的共轭子群的个数为 $[G:N_G(M)]$
  - <font color='red'>$M$ 的共轭子群数 $=$ 使 $g^{-1}Mg$ 结果不同的 $g$ 的数量 $= N_G(M)$ 的陪集数量</font>
  - **证明**：
    - 设 $g^{-1}Mg = g'^{-1}Mg'$，移项得 $g'g^{-1}Mgg'^{-1} = M$，即 $gg'^{-1}\in N_G(M)$（**核心**）
      <!-- - 由子群封闭性得 $N_G(M)g = N_G(M)g'$ -->
      - 即 $G$ 在 $N_G(M)$ 的陪集分解下，$g$ 和 $g'$ 同余
      - 即 $N_G(M)$ 中不同余的 $g_i$ 均使得 $g_i^{-1}Mg_i$ 结果不同
      - 即 $N_G(M)$ 的代表元系就是 $M$ 的共轭映射类（**证毕**）
- **元素共轭指数**：$a$ 的共轭类的阶为 $[G:C_G(a)]$
  - **证明**：
    - 显然与 $a$ 可交换则与 $\lang a \rang$ 也可交换。故沿用上面结论即可
- **共轭轨道方程**：
  - **一般写法**：设 $\{\bar x_i\}^n_{i=1}$ 是互不相同的共轭类，则 $$|G| = \sum\limits^n_{i=1} [G:C_G(x_i)]$$
    - <font color='red'>注意互不相同的前提，这里的 $x_i$ 不能取全部元素</font>
    - **证明**：
      - 由共轭类的等价分拆性即得结论
  - **中心写法**：设 $\{\bar g_i\}^n_{i=1}$ 是中心以外的互不相同共轭类，则 $$|G| = |C(G)| + \sum\limits^n_{i=1} [G:C_G(x_i)]$$
    - <font color='red'>实际上啥也没改变</font>
    - **证明**：易得中心的共轭轨道即为本身，再由上面结论即可

### 素阶群

- **$p$ 群中心非平凡性**：素数正整数幂 $p^n$ 阶群 $G$，其中心非平凡，即 $|C(G)| > 1$
  - **证明**：
    - 构建共轭分拆，中心元素共轭于自身，其它每个等价类则有 $[G:C(a)]$ 个元素，数量整除 $|G| = p^n$，可设为 $p^k$
    - 所以 $p^n = \sum\limits_{中心元素} 1 + \sum\limits_{非中心元素} p^k$
    - 由整除性的等式传递，中心元素个数只能为 $np$（**证毕**）
  - **本质**：详见[p群定义](5.有限群的结构.md)
- **简单 $p$ 群的结构**：素数 $p、p^2$ 阶群均是阿贝尔群
  - **证明**：
    - 由拉格朗日结果，子群 $A$ 的指数/阶整除 $|G|$，故只能为 $p$ 或 $p^2$。再由元素的阶均整除 $|A|$，故只能为 $p$ 或 $p^2$
    - 若存在 $p^2$ 阶元素，则构成循环群，其为阿贝尔群
    - 若元素阶均为 $p$，已知素幂群存在非平凡中心元素 $a$，设其构成的循环子群为 $A$
      - 则对 $\forall b\notin A$，$\{A,Ab,...,Ab^{p-1}\}$ 构成G对A的陪集分解，写成标准形式 $G = \{a^nb^m\mid 0\leq n<m \leq p-1\}$
      - 再利用a的可交换性，即易得 $G$ 为阿贝尔群
  - **本质**：若 $G/C(G)$ 是循环群，则 $G$ 是阿贝尔群

### 共轭

- **轨道定理**： （有限群 $G$ 的真子群）的（全部共轭子群之并）不能覆盖 $G$
  - <font color='red'>共轭等价的代表元系不可能全在一个子群中
  - 子群的共轭类就如同一个个不相交的轨道</font>
  - **证明**：
    - 设全部共轭子群之并为 $\Sigma$，则由共轭指数公式 + 拉格朗日即得 $$\hspace{-1em}|\Sigma| \leq (|H|-1)\ffkh{G:N_G(H)} + 1 = \frac{|G|}{|N_G(H)|}\cdot|H| - \ffkh{G:N_G(H)} + 1$$
    - 若 $H$ 为正规子群，则正规化子为 $G$，则 $|\Sigma| = |H| < |G|$
    - 若 $H$ 不为正规子群，则 $[G:N_G(H)] > 1$，对子群阶进行放缩得 $|\Sigma| < |G|$
  - **理解**：由共轭数量公式来看，只要讨论 $|M|$ 和 $|N_G(M)|$ 的关系即可
  - **反例（无限群不成立）**：
    - $GL_n(\C)$ 中，$H$ 是上三角可逆复矩阵，则由高代知任意矩阵均相似于 $H$ 中的元，从而覆盖整个群
- 有限阿贝尔群 $G$ 中，$\p:g\mapsto g^k$ 是自同构 $\LR \gcd(k,|G|) = 1$
  - <font color='chartreuse'>用西罗定理更好</font>

### 双陪集分解

- **双陪集**：
  - 设 $G$ 是有限群，$H,K\leq G$
  - 则 $g'\in HgK$ 是等价关系 $g\sim g'$
  - **证明**：
    - **自反性**：$g = e_Hge_K$
    - **交换性**：设 $g' = hgk$，则 $g = h^{-1}g'k^{-1}$
    - **传递性**：设 $g_1 = h_0g_0k_0$，$g_2 = h_1g_1k_1$，则 $g_2 = h_1h_0g_0k_0k_1$
  - **理解**：$HgK$ 相当于 $HK$ 被 $g$ 偏移了某个距离，而 $HK$ 是 $G$ 的子群，当然也可陪集分解
  - **实例**：
    - $GL_n(\R)$ 中，$H$ 是下三角可逆矩阵，$K$ 是上三角可逆矩阵，$P$ 是置换矩阵，则双陪集均可写为 $HPK$ 形式
    - **证明**：
- **拉格朗日**：$|HgK| = |H|\cdot[K:K\cap g^{-1}Hg] = |K|\cdot[H:H\cap gKg^{-1}]$
  - **证明**：
    - 先按照 $H\leq HgK$ 进行陪集分解，得 $HgK = \mathop{\bigsqcup}\limits^t_{i=1} Ha_i$。此时 $|HgK| = |H|t$
    - 再由 $HgK$ 定义，得 $a_i = gk_i$
    - 此时还可以将 $k_i$ 看作 $Hg\leq HgK$ 的右陪集代表元系
    - 再易得 $k_i$ 是 $K\cap g^{-1}Hg \leq K$ 的右陪集代表元系
      - <font color='chartreuse'>此时 $Kk_i = K$，$\bigsqcup g^{-1}Hgk_i = g^{-1}\bigsqcup Hgk_i = g^{-1}HgK \geq K$，两者相交等于 $K$，由等度平移即得该结论</font>
    - 此时 $t = [K:K\cap g^{-1}Hg]$，即得第一式，用拉格朗日进行转化即得第二式
- **左右二象性**：若 $A\leq G$ 的指数有限，则存在 $g_1,...,g_m$ 同时是左右陪集的代表元系
  - **证明**：构造双陪集分解 $AgA$ 即可
  - **实例**：$S_3$ 中，三阶循环子群 $\bigm \lang (231) \bigm\rang$ 的左陪集均为右陪集
  - **反例**：$S_3$ 中，二阶循环子群 $\bigm \lang (213) \bigm\rang$ 不存在同时是左右陪集的集合

### 中心

- **中心不变性**：设 $A\leq G$，则 $C_GC_GC_G(A) = C_G(A)$
  - **证明**：
    - 设 $B = C_GC_G(A)$，则 $\forall y\in B$ 与 $x\in C_G(A)$ 可换，则 $\forall x$ 和 $y$ 也可换，从而 $x\in C_G(B)$ 且 $y\in C_G(A)$，由任意性得相等


### 正规子群


- **作业题**：若 $G$ 中存在两个阶 $3,5$ 的正规子群 $H,K$ ，则 $G$ 中至少有一个阶为 $15$ 的元素
  - **证明**：
    - 由于子群的交还是子群，故 $H\cap K\leq H,K$，故此时 $|H\cap K|$ 整除 $(m,n)$，只能是 $H\cap K = \lang e \rang$
    - 再由3阶和5阶群只有循环群，取 $|h| = 3，|k|= 5$，则 $|hk| = [3,5] = 15$
  - **反例（若替换成任意 $(m,n) = 1$，则结论不成立）**：
    - $K_4\times G_9$，其中 $K_4$ 是klein四元群，$G_9$ 是两个三阶元素生成的九阶非交换群 $\lang a,b \rang$
    - $K_4$ 中所有元素均为 $2$ 阶，$G_9$ 中所有元素均为三阶
- $Q_8$ 的子群均正规

#### 平凡正规子群

- **半分正规定理**：指数为 $2$ 的子群是正规子群
  - **证明**：
    - 右陪集分解 $H = He，G-H = Hg\pad (g\notin H)$
    - 由陪集分拆性，$Hg\cap H = \varnothing$，同理 $gH\cap H = \varnothing$，再由阶数相等，即得 $Hg = gH$，从而 $H$ 是正规子群
- **共轭同构推论**：设 $G$ 是有限群，若某个阶数的子群只有一个，则它是正规子群
  - **证明**：
    - 由[共轭同构](5.有限群的结构.md)，得 $g^{-1}Hg \cong H$，再由唯一性，只能是 $g^{-1}Hg = H$，从而正规

#### 传递性

- **循环正规子群的遗传性**：若循环群 $H\lhd G$，则其子群 $K\lhd G$
  - **证明**：
    - 设 $H = \lang a \rang$，则 $K = \lang a^k \rang$
    - 由 $H$ 正规性，$g^{-1}a^{nk}g \in H$，从而 $K$ 也正规
- **生成传递性**：若 $H\lhd G，K\lhd G$，则 $H\lor K\lhd G$
  - **证明**：
    - 设 $gh_1 = h_2g，gk_1 = k_2g$，则 $gh_1k_1 = h_2gk_1 = h_2k_2g$，其它情况讨论得同理
    - <font color='chartreuse'>还可以用平移视角、同态构造等证明</font>

#### 交换性

- **独立可换定理**：若两个正规子群 $M\cap N = \lang e \rang$，则彼此元素可交换
  - **证明**：
    - 任取 $m\in M，n\in N$，由正规性易得 $$mnm^{-1}n^{-1} = (mnm^{-1})n^{-1}\in N = m(nm^{-1}n)\in M$$
    - 再由题设即得 $mnm^{-1}n^{-1} = e$，从而可换


#### 小总结

- 可交换（正规化子、中心元素）（与其它元素可交换） $\neq$ 阿贝尔群（内部元素可交换） $\neq$ 左右陪集相等（正规子群）
