# 陪集

- **符号约定**：
  - $\bigsqcup$ 表示不交并
- **背景**：
  - 前面习题说明，子群的并不是群，从而不可直接对子群归纳
  - 但归纳法是重要的数学方法，所以我们提出了陪集
- **陪集归纳法**：
  - 首先证明性质对子群成立，再证明该子群的陪集成立，则对群也成立

## 陪集分解

### 同余

- **元素的同余**：
  - 设 $G$ 是群，元素 $g,h\in G$，子群 $A\leq G$
  - 若 $gh^{-1}\in A$，则称 $g$ 模 $A$ 右同余于 $h$
- **同余等价**：左/右同余是群上的等价关系
  - **证明**：
    - **自反性**：$g = eg$，故 $g\in Ag$
    - **交换性**：$g\in Ah$，$h\in A^{-1}g = Ag$
    - **传递性**：$g\in Ah$，$h\in Ai$，则 $g\in AAi = Ai$
- **同余等价类**：$g$ 的右同余等价类为 $Ag$
  - **不交性**：陪集是子群的不相交平移
    - **证明**：若 $g_1,g_2\notin A$，但 $c \in Ag_1\cap Ag_2$，则 $\exists c = a_1g_1 = a_2g_2$，即 $g_1 = Ag_2$，彼此等价
    - <font color='chartreuse'>故只有使（子群平移）不相交的元素才不等价</font>
  - **本质**：子群对自身的[轨道等价类](./5.有限群的结构)
  - **实例**：
    - $4$ 的同余类 $Z_4$ 中，子群为 $A = \{4k+3\}$
      - 若取 $g = 2$，则其等价类为 $Ag= 4k+2$
      - 分别取 $g = 0,1,2,3$，即可得到分拆
    - 单位圆基本群 $\pi_1(S^1,x_0)$ 的陪集是其它点上的环路 $\pi_1(S^1,x)$（见Munkres代数拓扑）
    - 同余类 $Z_n$ 的陪集是其它完系 $Z_n+kn\pad (k=1,...)$
    - 模1同余的有理数加法群 $(\Q_1,+)$（见stein Fourier分析）

### 陪集

- **子群的陪集**：
  - **子群 $A$ 以 $g$ 为代表元的右陪集**：取 $g\in G$，则陪集为 $Ag$
  - **子群 $A$ 以 $g$ 为代表元的左陪集**：取 $g\in G$，则陪集为 $gA$
    - <font color='red'>陪集是子群的平移，代表元指示平移的位置</font>
- **子群的代表元系**：
  - **子群 $A$ 的右陪集代表元系**：每个右陪集中取一个元素形成的集合 $R_A$
  - **子群 $A$ 的左陪集代表元系**：每个左陪集中取一个元素形成的集合 $L$
    - <font color='red'>代表元系</font>
- **陪集分解**：子群 $A$ 和其右代表元系 $R_A$ 构成群的右陪集分解
- **子群的指数**：陪集代表元的个数 $[G:A] = |R_A| = |L_A|$
  - **实例**：
    - 整数加法群 $\Z$ 中，$\lang 1 \rang = \Z$，故无限阶元 $1$ 的指数为 $1$
  <!-- - **证明**：左右陪集只有运算方向有区别，元素数量还是一一对应的。所以A只要大小不变，元素数量就不变 -->
- **拉格朗日结果**：若 $G$ 为有限群，$A$ 是子群，则 $|G| = |A|\cdot[G:A]$
  - **证明**：
    - 由于 $|Ag| = |A|$，且不同等价类中的 $Ag$ 不相交
    - 从而 $|G| = \abs{\mathop{\bigcup}\limits_{g\in R_A}Ag} = \sum\limits_{不同g\in R_A}|A| = |A|\cdot |R_A|$
  - **推论**：
    - 素数阶的群只有平凡子群
    - 如果是正规化子，则左右陪集相等

### 陪集分解公式

- **符号约定**：设 $G$ 是有限群，$A,B$ 是子群。
- **积的陪集分解公式**：$|AB| = \cfrac{|A|\cdot |B|}{|A\cap B|}$
  - <font color='red'>类似最大公因式和最小公倍式的关系</font>
  - **理解**：<font color='chartreuse'>灾难啊，早期我的证明怎么写的这么啰嗦又没用……</font>
    - 分母若为1，此时A和B是分拆中的两个集合，就是独立乘法公式
    - 举一个最简单的例子，$Z_p$ 内两个空隙群 $an、bn(n=1,2,...)$。此时就真的变成了最大公因数和最小公倍数的关系
    - 再考虑分母不为1，则
      - 法1：仿照最大最小夹逼性
      - 法2：化为分母为1的情况
      - 法3：变换为相似形式并比较（下述证法）
  - **证明（同构法）**：若 $A\cap B = \{e\}$，结论显然。下设 $A\cap B$ 至少有两个不同的项 $b、b'$
    - $AB$ 是陪集 $Ab$ 的并，$B$ 是陪集 $(A\cap B)b$ 的并
      - （这里 $A \cap B$ 完全是为了形式配凑而设置的）
    - $Ab = Ab'(b\neq b') \longleftrightarrow A = Ab'b^{-1} \longleftrightarrow b'b^{-1}\in A$
      - b和b'把A映射到一个相同的集合中（顺序不同）
      - 右乘 $bb^{-1}$ 是A的一个置换
    - 再由 $b\in B$ 得到 $b'b^{-1}\in (A\cap B)$
    - 因此 $(A\cap B)b = (A\cap B)b'$
    - 比较两个等式，交集和全集具有相同的依赖元素b或b'（它们在陪集分解意义下是等价的）
    - 此时 $(A\cap B)B在A\cap B、b分解下$，$AB在A、b分解下$，右陪集数量相同
    - 从而 $\frac{|AB|}{|A|} = \frac{|B|}{|A\cap B|}$（**证毕**）
  - **理解**：
    - 重点是寻找b和b'（需要满足封闭性乘积传递，不是任意的）
      - 封闭性乘积传递：依赖子群A和依赖元素b所属的群B，同时两个封闭性
    - 运用了陪集分解的子群任意性，把群的阶分别对两个子群进行陪集分解。从而实现群的阶的整除形式
  - **本质**：陪集数量对g的依赖关系
  - **证明（陪集分解）（Hungerford）**：
    - 设 $C = A\cap B$
      - 由交封闭性得 $C\leq B$，即 $B$ 可被 $C$ 分解为 $n = \cfrac{|B|}{|C|}$ 个陪集
      - 同理 $C\leq A$ 得 $AC = A$
      - 由乘积封闭性得 $B\leq AB$，即得 $AB$ 可被 $B$ 陪集分解
    - 综上，$AB = A(Ch) = Ah$，故 $|AB| = |A|[B:A\cap B] = \cfrac{|A|\cdot|B|}{|A\cap B|}$
      - <font color='chartreuse'>$h$ 是 $B$ 关于子群 $C$ 的代表元系</font>
  - **推论（交的陪集数量）**：同一个b分解下，群的陪集数量和其交的陪集数量相等
- **陪集的复合分解**：若 $A\leq B\leq G$，则 $[G:A] = [G:B][B:A]$
  - **证明**：
    - **复合分解（运算的结合律，通过群的封闭性推广到集合上）**
      - G对B的陪集分解：$G = \mathop{\bigsqcup}\limits^n_{j=1} Bg_j，n = [G:B]$
      - B对A的陪集分解：$B = \mathop{\bigsqcup}\limits^m_{i=1} Ab_i，m = [B:A]$
      - 从而 $G = \mathop{\bigsqcup}\limits^n_{j=1}\dkh{\mathop{\bigsqcup}\limits^m_{i=1}Ab_i} g_j$，由结合律，$G = \mathop{\bigsqcup}\limits^n_{j=1}\mathop{\bigsqcup}\limits^m_{i=1}\ddkh{Ab_ig_j}$
    - **互不相交性**
      - 反设 $Ag_jb_i = Ag_{j'}b_{i'}$
        - 则 $b_{i'}g_{j'}g^{-1}_jb^{-1}_i\in A$，移项得 $g_{j'}g^{-1}_{j}\in b^{-1}_{i'}Ab_i \subseteq B$，由B的封闭性得到 $Bg_j = Bg_{j'}$，由陪集的互不相交性得到 $j=j'$
        - 原先式子变为 $b_{i'}b^{-1}_i\in A$，重复得到 $i=i'$
      - 因此两个序号都不相等时，陪集才不同。类比全排列关系，满足复合陪集互不相交性，即可以进行复合分拆，得到题设等式（**证毕**）
    - **理解**：因为A和B都是群，所以才可以进行传递。和上例中A和 $A\cap B$ 都是群是类似的
- **陪集的交分解（三角不等式）**：$[G:A\cap B] \leq [G:A][G:B]$
  - **证明（分解法）**：
    - 取 $b、b'\notin (A\cap B)$（$b、b'\notin A$ 同样可以得到下面结果）
    - 此时 $(A\cap B)b \neq (A\cap B)b' \longrightarrow bb'^{-1}\notin (A\cap B)\subset A \longrightarrow \\ Ab\neq Ab'$
      - 即 $B/(A\cap B)$ 内可以出现在同一个陪集分解的元素，也一定可以出现在 $G/A$ 中
    - 因此 $[B:A\cap B] \leq [G:A]$，再利用复合分解得到题设不等式
      - **单向条件推出不等号，充要条件推出等号**
  - **证明（映射法）（Hungerford）**：
    - 设 $H$ 为 $A\cap B$ 在 $A$ 下的的右陪集，$K$ 为 $A$ 在 $G$ 下的右陪集，
    - 考虑陪集映射 $\varphi: H\to K，(A\cap B)g \mapsto Ag$
    - 证明映射为单射，从而得到阶的大小关系
      - 反设 $(A\cap B)g_1 \neq (A\cap B)g_2$ 且 $Ag_1 = Ag_2$
      - 则有 $g_1g_2^{-1}\in A$ 但 $\notin A\cap B$，矛盾
    - **本质**：也是分解，不过有点意思
  - **证明（计算法）**：
    - $左式= \dfrac{|G|}{|A\cap B|}，右式 = \dfrac{|G|}{|A|}\cdot\dfrac{|G|}{|B|}$
    - 从而 $左\leq 右 \LR |AB|\leq |G|$，显然成立
  - **推论**：
    - 若两指数互素，且 $AB = G$ 时取等号，则此时等式为 $|G| = |B|\cdot |A|$


## 习题

### 子群的乘积

- **子群无乘积封闭性**：设 $A,B\leq G$，则 $AB\leq G$ 不一定成立
    - **反例**：
      - <font color='chartreuse'>由封闭性，$AB$ 中不可能存在 $G$ 以外的元素，故只能是 $AB$ 不构成群
      - 由可逆性和含幺性，只能是 $AB$ 不封闭
      - 而 $aba'b' = a(b'a)b'$，即只需找到 $ba\notin AB$</font>
      - $|G| = p^km$，其中 $p$ 是素数，$(p,m) = 1$
        - 若 $|H| = p^k，|K| = p^d，0 < d\leq k$，且 $K\not\subset H$
        - 则 $HK$ 不是子群
      - **证明**：需要[西罗定理](5.有限群的结构.md)
        - 此时 $|H\cap K| < p^d$，由乘法公式得 $|HK| = p^l\cdot p^k$，从而 $HK$ 的阶不整除 $|G|$，不是子群
- **封闭可换定理**：$AB\leq G \LR AB = BA$
  - **证明**：
    - **充分性**：定义易得
    - **必要性**：$ba = (e_{AB}b)(ae_{AB}) \in (AB)(AB) = AB$，从而 $BA\leq AB$，反之同理
  - **本质**：若是直积，则结论易得。但这里把条件弱化，故稍微复杂
    <!-- - **必要性**：反设 $\exists a_0b_0\in AB$，使得 $\forall ba\in BA$，都有 $a_0b_0 \neq ba$
      - 则 $a_0 \neq bab_0^{-1}\pad (\forall b\in B，a\in A)$，即 $a_0\notin BAb_0^{-1}$ -->
- **子群单位定理**：
  - 设 $A,B\leq G$，且 $G = AB$
    - <font color='chartreuse'>即 $A$ 是子群，$B$ 是其代表元系</font>
  - 若 $A\leq C\leq G$，则 $C = A(B\cap C)$
    - <font color='chartreuse'>$B\cap C$ 实际上就是 $A$ 在 $C$ 内部的代表元系</font>
    - <font color='red'>这个定理实际上说明，任何包含 $A$ 的子群都是它的整数倍</font>
  - **证明**：此时 $\forall c = ab$，也即 $b = a^{-1}c\in B\cap C$，由任意性即得结论
- **有限群交集覆盖定理**：
  - 设 $G$ 是有限群，$A,B\subset G$
  - 若 $|A|+|B|>|G|$，则 $G = AB$
  - <font color='red'>两个相交子集的乘积可覆盖整个群</font>
  - **证明**： $\forall g\in G，A^{-1}g\cap B \neq \varnothing$，则存在 $b = a^{-1}g$，从而 $g = ab$，即得结论

### 陪集与平移

- **等距平移定理**：
  - 设 $A,B\leq G$
  - 则
    - **元素形式**：$g(A\cap B) = gA\cap gB$
      - <font color='red'>$g$ 代表平移距离，由于 $g$ 同时作用于 $A,B$，故它们的平移距离相等</font>
      - **证明（互包）**：
        - 设 $h\in A\cap B$，则其既在A中也在B中，故 $gh\in gA\cap gB$
        - 设 $k\in gA\cap gB$，则 $k\in gA$ 且 $k\in gB$，从而 $g^{-1}k\in g^{-1}gA$ 且 $g^{-1}k\in g^{-1}gB$，即得结论
    - **集合形式**：设 $H,K,N\leq G，H<N$，则 $H(K\cap N) = HK\cap N$
      - **证明**：
        - 由元素的等距平移易得结论
- **平移轨道定理**：
  - 设 $A,B\leq G$
  - 若 $\exists g,h\in G$，使得 $Ag = Bh$，则 $A = B$
  - <font color='red'>$A$ 和 $B$ 在经历两段不同的平移距离后，最终可以完全重合。简单运算即可，不难</font>
  - **证明**：
    - 此时 $g\in Ag = Bh，h\in Bh = Ag$（**核心**）
    - 故 $\exists a\in A，b\in B$，使得 $g = bh = bag$，即 $\exists ba = e$，同理 $ab = e$，此时 $Ag = Abh = Bh$，故 $A = Bb^{-1}$，从而 $A = B$
  <!-- - 若反设 $A\neq B$，则只能是 $g,h\notin A$ 且 $g,h\notin B$ -->
- **互补定理**：
  - 设 $H \leq K\leq G$
  - 若对任意 $N\leq G$ 都有 $H\cap N = K\cap N，HN = KN$，则 $H=K$
  - <font color='red'>本质是：乘积和交集是互补的
  - 乘积可看作平移。两子群与 $N$ 的交集是它们的平移不变量，两子群与 $N$ 的乘积是它们被 $N$ 平移后的结果，这两者组成整个子群</font>
  - **证明**：
    - 设 $k\in K-H$，则 $kN \subset HN$，但 $k\notin H$，这显然不可能

### 代表元系

- **代表元互逆定理**：同一子群的左右代表元系满足 $L = R^{-1}$
  - **证明**：
    - 只需 $G = \mathop{\bigsqcup}\limits_{x\in R^{-1}}xA$ 即可
    - **全集性**：对 $\forall z\in G$，存在 $z^{-1}\in Ag$，即 $z\in g^{-1}A$
    - **不交性**：反设 $z\in xA\cap yA$，则存在 $z = xa_1 = ya_2$，此时 $Ax^{-1}\cap Ay^{-1} = A(a_1z^{-1})\cap A(a_2z^{-1}) = Az^{-1} = \varnothing$
  - **理解**：简单运算即可，不难

### 生成并

- **生成并**：设 $H,K\leq G$，则 $\lang H\cup K \rang$ 称为生成并
  - <font color='red'>所有 $H,K$ 中元素的最小生成子集</font>
  - **简写符号**：$H\lor K$ 或 $\lang H,K \rang$ 
  - **子群性**
    - **证明**：
  - **极小性**：若 $H,K\leq L$，则 $\lang H\cup K \rang \leq L$
    - **证明**：
  - **指数公式**：$[H\lor K:H] \geq [K:H\cap K]$
