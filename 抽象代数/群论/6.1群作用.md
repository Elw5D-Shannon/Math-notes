# 群与对称性

## 群作用

- **群作用**：
  - 设 $S$ 是集合，$G$ 是群
  - 若映射 $G\times S\to S，(g,x)\mapsto gx$ 满足
    - **含幺性**：$ex=x$
    - **结合律**：$(g_1g_2)x = g_1(g_2x)$
    - <font color='red'>容易发现，这个定义已经蕴含了封闭性 $gx\in S$ 和可逆性 $g(g^{-1}x) = x$</font>
  - 则该映射称为群 $G$ 在集合上 $S$ 的作用
- **平凡作用**：群的任意元素都是幺元
- **正则作用**：群作用于自身
  - **左乘正则作用**
  - **右乘正则作用**
- **实例**：
  - **置换作用**：对称群 $S_n$ 作用于字典 $I_n = \{1,\cdots,n\}$
    - <font color='chartreuse'>群元素本身是映射：</font>$(\sigma,x)\mapsto \sigma(x)$
  - **左平移**：群 $H<G$，$H$ 作用于 $G$
    - <font color='chartreuse'>集合本身是群元素（左乘映射）：</font>$(h,x)\mapsto hx$
  - **左陪集平移**：群 $H,K<G$，$S$ 是 $K$ 的所有左陪集，$H$ 作用于 $S$
    - <font color='chartreuse'>集合本身是群子集（左乘映射）：</font>$(h,xK)\mapsto hxK$
  - **共轭**：正规子群 $H\lhd G$，$H$ 作用于 $G$
    - <font color='chartreuse'>集合本身是群元素（共轭映射）：</font>$(h,x)\mapsto hxh^{-1}$

### 群作用的置换意义

- **（定理4.5）作用同态**：若 $G$ 作用于 $S$，则存在 $G$ 到 $S$ 的置换群的同态 $\tau:G\to A(S)$
  - **证明**：
    - 任取 $g\in G$，设 $\tau_g : S\to S，x\mapsto gx$
      - 易得其为双射，从而是置换
      - 再由 $g$ 可逆性、结合律、含幺性，所有不同的 $\tau_g$ 可组成置换群 $A(S)$
    - 再易得 $\tau_{gg'} = \tau_g\tau_{g'}$，即 $\tau:g\mapsto \tau_g$ 是同态
  - **本质**：
    - 群作用都可看作对称作用的子集（$S$ 的置换群）
    - 稳定子群即为不包含元素 $x$ 的置换，作用同态的核为 $\mathop{\bigcap}\limits_{x\in S} G_x$
    - <font color='red'>有些在置换意义下等价的元素，在群作用意义下不等价。此时作用同态不为单射（比如后面的[共轭作用同态]()）</font>
- **作用同态**：
  - 设 $\tau_g:S\to S，x\mapsto gx$
  - 则 $\tau:G\to A(S)，g\mapsto \tau_g$ 称为作用同态
  - <font color='red'>（群元素）到（其作用）的映射</font>
- **（推论4.6）Cayley定理**：任意群均存在一个作用单同态
  - <font color='red'>群都同构于自身的某个置换群</font>
  - **证明**：
    - **存在性**：设 $G$ 左平移作用于其本身，由作用同态定理，存在 $\tau:G\to A(G)$
    - **单射性**：若 $\tau(g) = \tau_g = 1_G$，易得只能是 $g=e$，从而是单同态
  - **推论（有限群的置换表示）**：有限群均作用同构于 $S_n\pad (n = |G|)$
  - **反例（作用同态不一定是单的）**：详见后面的左陪集平移
    - <font color='red'>Cayley定理本质是说明：群自身的左平移作用总是单的</font>

### 轨道等价

- **轨道等价**：若 $\exists g\in G$ 使得 $gx = x'$，则称 $x\sim x'$
  - **证明**：
    - **自反性**：$ex = x$
    - **对称性**：$g^{-1}x' = x$
    - **传递性**
- **轨道**：$x$ 在轨道等价下的等价类 $\ol x$
- **不动点集**：$S_0 = \hkh{x\in S\mid \forall g\in G，gx = x}$
    - <font color='red'>不会被任何作用改变的集合元素（轨道是单点集）</font>
- **稳定子群**：设 $x\in S$，则 $G_x = \{g\in G\mid gx = x\}$ 是 $G$ 的子群
  - <font color='red'>$x$ 的稳定子群是所有使 $x$ 不动的置换</font>
  - **证明**：
    - **封闭性**：$(g_1g_2)x = g_1x = x$，故 $g_1g_2\in G_x$
    - **可逆性**：$gx = x$，两边左乘 $g^{-1}$ 得 $x = g^{-1}x$，故 $g^{-1}\in G_x$
    - **结合律**：由于 $G$ 是群，满足结合律，故 $g_1(g_2g_3) = (g_1g_2)g_3$，即 $G_x$ 也满足结合律
  - **几何意义**：<font color='chartreuse'>它也叫做迷向子群或各向同性子群
    - 当 $S$ 是空间几何体（球面、欧氏空间）时，$G$ 就是几何体的置换（旋转、平移）
    - 轨道 $\ol x$ 就是点 $x$ 可被转移到的所有点
    - 稳定子群 $G_x$ 就是保持 $x$ 不动，但转动 $x$ 周围方向的置换
      - 它使得 $x$ 处所有方向等价，所以叫迷向子群
    - 比如李群 $SO(3)$ 作用在球面时，北极点的稳定子群是绕北极轴的旋转</font>
- **（定理4.2）轨道定理**：
  - **第一公式**：$|G_x| = \cfrac{|G|}{|Gx|}$
    - **证明**：
      - $|Gx|$ 就是 $x$ 被所有 $g$ 作用的结果数量。稳定子的结果都相同，其它结果都不同，故结论易得
  - **第二公式**：$|\ol x| = [G:G_x]$
    - **证明1**：显然 $|Gx|$ 就是 $|\ol x|$
    - **证明2**：
      - 设 $g,h\in G$
      - 由于 $gx = hx \LR g^{-1}h\in G_x$
      - 设 $R$ 是 $G_x$ 的左陪集，则 $R\to \bar x，gG_x \mapsto gx$ 是双射，从而两边基数相等
  - <font color='red'>轨道等价类 $\ol x$ 是集合 $S$ 上的，它不是子群。要想用群论去研究，需要构造其在 $G$ 上的对应（$G_x$ 的陪集分解）
  - 此时轨道等价类可看作商群，稳定子群即为等价集，从而轨道阶数即为稳定子群指数</font>
- **实例**：
  - 置换作用中，$S = \set{1,2,3,4}$
    - 稳定子群 $G_3 = \{(1),(12),(14),(24),(124),(142)\}$
    - $3$ 的轨道等价类为 $\{G_3\cdot x\mid x\in S\} = \{G_3\cdot 1234\}$
    - 不动点集为空
    - <font color='chartreuse'>稳定子群是所有不含该元素的置换，阶为 $(n-1)!$，故轨道等价类基数为 $n$</font>
  - 左平移作用中
    - 稳定子群 $G_x = \lang e \rang\quad (\forall x\in G)$
    - 轨道等价类为 $G$ 全体
    - 不动点集为空
    - <font color='chartreuse'>由于幺元唯一性，稳定子群是平凡的</font>
  - 左陪集作用中
    - 稳定子群 $\forall x = aK\pad (a\in G)，H_x = (aKa^{-1})\cap H$
    - 轨道等价类为 $S$ 全体（$G$ 中所有陪集）
    - 不动点集为 $N_G(K)$ 中的 $K$ 陪集
    - <font color='chartreuse'>稳定子群只需令陪集不变即可，故为陪集本身
    - 而</font>
  - 共轭作用中
    - 稳定子群 $G_x = C_G(G) = C(G)\quad (\forall x\in G)$
    - 轨道等价类为共轭类
    - 不动点集为 $C(G)$
    - <font color='chartreuse'>稳定子群是中心</font>
- **核定理**：群作用的核 $\LR$ 作用同态的核 $\LR \mathop{\bigcap}\limits_{x\in S} G_x$
  - <font color='red'>所有恒稳定的作用</font>
  - **证明**：显然

### 群作用的物理意义

- **可迁作用（传递作用）**：$G$ 在 $S$ 上只有一个轨道
  - <font color='red'>任意元素 $x,y$ 都可互相转换</font>
  - **共轭定义法**：$\forall x,y\in S，\exists g\in G$ 使得 $gG_xg^{-1} = G_y$
    - **证明**：
  - **几何意义**：
  - **实例**：
    - 对称群 $S_n$ 在 $n$ 元集合的作用可迁
    - 二面体群 $D_n$ 在正 $n$ 边形的作用可迁
    - 左平移作用可迁
    - 左陪集平移作用可迁
  - **反例**：
    - 平凡作用在 $|S| \geq 2$ 时不可迁
    - 共轭作用一般不可迁
- **忠实作用**：核为 $\lang e \rang$ 的群作用
  - <font color='red'>作用同态是单射</font>
  - **实例**：
    - 群在自身上的左平移作用忠实（Cayley定理）
    - 对称群 $S_n$ 在 $n$ 元集合的作用忠实
    - 交错群 $A_n$ 在 $n$ 元集合的作用忠实
  - **反例**：
    - 平凡作用（所有 $g$ 都映为 $(1)$，丢失全部原群信息）
  - <font color='red'>群作用实际上是研究 $G$ 的工具，所以通过适当选取 $S$ 来得到置换群才是我们的目的
  - 实际工作中不一定要求作用是忠实的，而是倾向于选择"更能结合几何或物理实际"的 $S$ 和相应作用
    - 比如研究 $\R^3$ 中原点在旋转下的对称性时，取 $G$ 是旋转群，群作用为向量旋转，$S = \{0\}$</font>
- **群作用的不唯一性**：已知被作用集 $S$ 和作用时，不能唯一确定群 $G$
  - **证明**：
    - 确定作用时，只能确定作用同态的像，不能确定它的核
    - 由于 $G = \Im\tau \times \ker \tau$，故当核改变时，群 $G$ 改变
  - **反例**：
    - $S = \{1,2\}$，取平凡作用
      - 可取 $G = \lang e \rang$
      - 可取 $G = \Z_2$
      - 可取 $G = \Z$
- **忠实唯一定理**：已知被作用集 $S$ 和忠实作用时，可唯一确定群 $G$


### 共轭轨道

- **共轭类**：$\bar x = \{gxg^{-1}\mid g\in G\}$
  - **本质**：一种特殊的轨道
  - **实例（交换群）**：左平移作用下，$\Z_p$ 中的同余类
    - <font color='chartreuse'>此时就是一般的等价类。（可以看作置换后再逆置换，本质没有变化）</font>
  - **实例（非交换群）**：左平移作用下，$GL_n(R)$ 中同一相似标准型下的所有矩阵
    - <font color='chartreuse'>此时是两个方向的互逆置换，可能不处在同一个单向轨道</font>
- **（推论4.4）共轭轨道公式**：详见前面


### 小总结

- 不动点 $S_0$ 是 $S$ 的子集，表示任何 $g$（或 $\tau_g$）作用下均不动的点
- 稳定子群 $G_x$ 是 $G$ 的子群，表示某个 $x$ 上不动的 $g$（或 $\tau_g$）
- 作用核 $\ker\tau = \bigcap G_x$ 是 $G$ 的正规子群，表示任何 $x$ 上均不动的作用
- 显然三者完全不是一回事，要分开讨论

## 习题

- **轨道共轭性**：$\forall a,b\in S$，若 $ga = b$，则 $G_a = g^{-1}G_bg$
  - **证明**：
    - 由定义 $G_aa = a$，且 $(g^{-1}G_bg)a = g^{-1}b = a$，由消去律即得结论
- 设 $G$ 在 $S$ 上的作用可迁，$N\lhd G$，则（$N$ 的陪集作用的轨道）基数均相等
  - **证明**：
    - 设 $S = Ga$
    - 任取 $x = ga$，若 $nga = ga$，则 $(g^{-1}ng)a = a$
      - 从而 $N_x = \set{n\in N\mid g^{-1}ng\in G_a} = N\cap (gG_ag^{-1})$
    - 再由[等距平移](./1.群的概念.md) 得 $N_x = g(N\cap G_a)g^{-1} = gN_ag^{-1}$
    - 故 $|Nx| = \dfrac{|N|}{|N_x|} = \dfrac{|N|}{|N_a|} = |Na|$，由 $x$ 任意性即得结论

### 稳定点集

- **稳定点集**：$S^g = \{s\in S\mid gs = s\}$
  - <font color='red'>它和稳定子群是对偶概念，但没那么重要</font>
- **稳定对**：$A = \hkh{(g,s)\in G\times S\mid gs = s}$
  - <font color='red'>显然稳定对和稳定子群、稳定点集应当存在阶数关系</font>
- **平面公式**：$\sum\limits_{s\in S}|G_s| = \sum\limits_{g\in G} |S^g| = |A|$
  - **证明**：
- **BurnSide公式**：群作用的轨道数量 $t = \dfrac{|A|}{|G|}$
  - **证明**：$$|A| = \sum\limits_{x\in S} |G_x| = \sum\limits_{x\in S} \frac{|G|}{|Gx|} = \sum\limits^t_{i=1}\sum\limits_{x\in Gx_i} \frac{|G|}{|G_x|} = \sum\limits^t_{i=1} |G| = t|G|$$


### 例子

- **正 $n$ 边形的置换群**：二面体群 $D_n$
  - 绕中心旋转：$a = (123...n)$，其中 $a^k$ 为 $\dfrac{2k\pi}{n}$ 度旋转
  - 对称变换：$b = \prod\limits_{2\leq i\leq ?} (i, n-i+2)$
- **正多面体的置换群**：三维空间中，仅有 $4,6,8,12,20$ 五种正多面体
  - **顶点、棱、面数量** $(v,e,f)$ 分别为 $(4,6,4),(8,12,6),(6,12,8),(20,30,12),(12,30,20)$
    - **证明**：欧拉公式
  - **轨道公式**：设 $V$ 是顶点集，$G_1$ 是顶点 $1$ 的稳定子群，则 $|G| = |V|\cdot|G_1|$
  - **旋转群**：
    - **正四面体**：$G_1$ 是正三角形旋转群
    - **正六面体**：$G_1$ 是正三角形旋转群
    - **正八面体**：$G_1$ 是正方形旋转群
    - **正十二面体**：$G_1$ 是正三角形旋转群
    - **正二十面体**：$G_1$ 是正五边形旋转群
  - **对称群**：
    - **正四面体**：$G_1$ 是正三角形对称群
    - **正六面体**：$G_1$ 是正三角形对称群
    - **正八面体**：$G_1$ 是正方形对称群
    - **正十二面体**：$G_1$ 是正三角形对称群
    - **正二十面体**：$G_1$ 是正五边形对称群
