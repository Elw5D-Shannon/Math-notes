# 小阶群的结构

## 半直积

### 内半直积

- **内半直积**：
  - 设 $G$ 是群，若存在 $N\lhd G，H\leq G$ 满足
    - **生成并性**：$G = NH$
    - **不交性**：$N\cap H = \lang e \rang$
  - 则称 $G$ 是 $N$ 被 $H$ 扩张的内半直积 $N\rtimes H$
  - <font color='red'>内直积是两个都正规，内半直积是只有一个正规</font>
- **可裂群**：可写成半直积的群
  - **实例**：
    - **对称群 $S_3$**：
      - 正规子群 $N = \lang (123) \rang \cong \Z_3$
      - 作用群 $H = \lang (12) \rang\cong \Z_2$
      - 即 $S_3 = \Z_3 \rtimes \Z_2$
  - **反例**：
    - 四元数群 $Q_8$ 不能写成半直积

### 半循环群

- **半循环群**：若 $G$ 存在循环正规子群 $C$，且商群 $G/C$ 也是循环群，则称 $G$ 是半循环群
- **二元半循环定理**：二元生成群中，若存在循环正规子群，则是半循环群
  - **写法1**：$$\ip{a,b\Bigm| a^m = b^n = e，bab^{-1} = a^r}$$
    - <font color='red'>由于只有两个生成元，故给定阶数后，再写出彼此之间的乘法交换公式 $ab = ba^r$ 即可表达出所有元素</font>
  - **写法2**：$$\hspace{-2.5em}\ip{a,b\Bigm | a^n = e，b^m = a^k，bab^{-1} = a^r}$$
    - <font color='red'>这个写法更直观一些。因为 $m$ 实际上就是商群 $\lang \ol b \rang = G/\lang a \rang$ 的阶</font>
- **可裂半循环定理**：可裂半循环群 $\LR$ 循环群的半直积
  - **证明**：显然
  - **实例**：
    - 二面体群 $D_n = \Z_n\rtimes \Z_2$，其中 $\lang a \rang$ 是正规子群，商群 $\lang \ol b \rang$ 满足不交性、生成并性
  - **反例（半循环群不可裂）**：
    - 四元数群 $Q_8 = \lang a,b \rang$，其中 $\lang a \rang$ 是正规子群，但商群 $\lang \ol b \rang$ 存在交集元素 $b^2 = a^2$


### 半直积的作用性

- **半直积作用定理**：半直积可视为 $H$ 在 $N$ 上的共轭作用 $\p_h:N\to N，n\mapsto hnh^{-1}$
  - <font color='red'>它不是 $N$ 的内部自同构，而是 $NH$ 的一种特殊的内部自同构</font>
  - **证明**：
    - **良定义（一一对应性）**：
      - 由正规性得对任意 $n,h$ 都存在唯一的 $n'$ 使得 $hn = n'h$
      - 取 $n' = \p_h(n)$ 即可
    - **含幺性**：易得
    - **封闭性**：易得
- **半直积作用同态**：$\p:H\to \Aut N，h\mapsto \p_h$
  - **单射性**：由一一对应性得它是单射

### 外半直积

- **外半直积**：
  - 设 $N,H$ 是群，$\p$ 是半直积作用同态
  - 定义运算 $(n_1,h_1)\cdot (n_2,h_2) = \Big( n_1\p_{h_1}(n_2)， h_1h_2\Big)$
  - 则其构成一个群 $N\times_\p H$
  - **证明**：
    - **含幺性**：幺元 $(e_N,e_H)$
    - **可逆性**：$(n,h)^{-1} = \Big( \p_{h^{-1}}(n^{-1})， h^{-1} \Big)$
- **同构定理**：外半直积和内半直积同构
  - **证明**：
    - 设映射 $f:(N\rtimes H)\to (N\times_\p H)，nh\mapsto (n,h)$
      - **双射性**：显然
      - **同构式**：$n_1h_1n_2h_2 = n_1(h_1n_2h_1^{-1})(h_1h_2) = (n_1,h_1)\cdot(n_2,h_2)$
  - **实例**：
    - 可裂半循环群中若 $bab^{-1} = a^r$，则可写为 $\Z_n\rtimes_r \Z_m$
- **群扩张**：若 $G\geq N$，则称 $G$ 是 $N$ 的扩张
- **分裂扩张**：若 $G\cong N\times_\p H$，则称 $G$ 是 $N$ 的分裂扩张
- **分裂引理**：详见[模论部分的正合列](./8.模.md)

### 半直积的多种分解

- **元素唯一分解定理**：给定 $G,N,H$ 时，任意元素 $g$ 存在唯一分解 $nh$
  - **证明**：显然
- **半直积无唯一分解性**：若给定 $G,N,H$ 中的两个，则剩下的一个可能不唯一
  - <font color='red'>如果是直积，则唯一。这是半直积和直积的核心区别</font>
  - **反例（给定 $N,H$ 时 $G$ 不唯一）**：
    - $N = \Z_4，H = \Z_2$
      - 若取 $bab^{-1} = a$（平凡作用），则满足交换性，$G = \Z_4\oplus \Z_2$
      - 若取 $bab^{-1} = a^{-1}$（非平凡作用），计算易得 $G = D_4$
  - **反例（给定 $G,N$ 时 $H$ 不唯一）**：
    - $G = \Z_2\times D_3$，取正规子群 $N = \{0\}\times D_3$
      - 若取 $H = \lang (1,e) \rang$，则 $(x,e)(0,g)(x,e)^{-1} = (0,g)$，平凡作用（直积）
      - 若取 $H = \lang (1,b) \rang$，则 $(1,b)(0,a)(1,b)^{-1} = (0,a^{-1})$，非平凡作用
    <!-- - $G = \Z_4\rtimes (\Z_2\oplus \Z_2)$，取正规子群 $N = \Z_4$
      - 若取 $H = \lang (1,0) \rang$，则 $hnh^{-1} = $，非平凡作用
      - 若取 $H = \lang (1,1) \rang$，则 $hnh^{-1} = n$，平凡作用（直积） -->
  - **特殊例（只存在平凡作用，故分解唯一）**：
    - $S_3 = \Z_3\rtimes \Z_2$，取正规子群 $N = \lang (123) \rang$
      - 若取 $H = \lang (12) \rang$，则 $hnh^{-1} = n$
      - 若取 $H = \lang (13) \rang$，则 $hnh^{-1} = n$
      - 若取 $H = \lang (23) \rang$，则 $hnh^{-1} = n$
      - <font color='red'>$\p$ 均相同，此时 $H$ 彼此同构，分解只有一种</font>
- **半直积唯一分解定理**：若给定 $G,N,H$ 中的两个，以及作用同态 $\p$，则剩下的一个唯一
  - **证明**：
    - 半直积同构定理的直接推论
  - <font color='red'>这也是为什么我们定义外半直积时必须加上作用同态 $\p$</font>
  - <font color='red'>给定 $N,H$ 时，计算 $\Aut N$ 中的合法 $\p_h$ 即可对 $G = N\rtimes H$ 进行分类</font>
- **可裂半循环群分类定理**：
  - 设可裂半循环群 $\Z_n \rtimes \Z_m$ 中 $bab^{-1} = a^r$
  - 则作用同态 $\p:\Z_m\to \Aut\Z_n$ 的个数等于满足 $r^m\equiv 1\pmod n$ 的 $r$ 个数
  - **证明**：
    - 显然此时 $\p$ 与 $r$ 一一对应
    - 再由同态运算法则，$\p(b)^m = \p(b^m) = e$，即 $b^mab^{-m} = a^{r^m} = e$，从而得到结论
  - **推论**：若 $r_1^m\equiv r_2^m \pmod n$，则它们对应的半直积同构（但反之不成立）

## 特定阶数的群分类


### $pq$ 阶群

- **（命题6.1）$pq$ 阶群**：设素数 $p>q$
  - 若 $q\nmid p-1$，则 $pq$ 阶群均同构于 $\Z_{pq}$
  - 若 $q\mid p-1$，则 $pq$ 阶群只有两种：
    - 循环群 $\Z_{pq}$
    - 可裂半循环群 $K = \Z_p\rtimes \Z_q$
      - **生成元系**：$p,q$ 阶元素 $a,b$
      - **运算法则**：$bab^{-1} = a^r$，其中 $\begin{cases} r\not\equiv 1\pmod p & 非交换性 \\ r^q\equiv 1 \pmod p & a,b的阶数 \end{cases}$
  - **证明**：
    - 由柯西阶定理，存在 $|a| = p，|b| = q$
    - 由最大子群正规性，$S = \lang a \rang$ 是正规子群
      - 再由 $|G/S| = q$ 得其只能是循环群，即 $G/S = \lang bS \rang$，即 $G$ 是二元生成的半循环群
    - 由第三西罗定理，西罗 $q$ 子群的数量 $(kq+1)\mid p$。由素数得只能是 $1$ 或 $p$
      - 若为 $1$ ，则 $q\nmid p-1$
        - 此时 $\lang b \rang$ 也是正规子群，构成直积，从而 $G\cong \Z_{pq}$
      - 若为 $p$，则 $q\mid p-1$
        - 由正规性，$bab^{-1} = a^r$
        - 此时西罗 $p$ 子群不唯一，得 $G$ 不是交换群，从而 $r\not\equiv 1\pmod p$
        - 此时 $b^qab^{-q} = a^{r^q} = a$，从而 $r^q\equiv 1\pmod p$

### $2p$ 阶群

- **（推论6.2）$2p$ 阶群**：若 $p$ 是奇素数，则 $2p$ 阶群同构于循环群 $\Z_{2p}$ 或二面体群 $D_p$
  - **证明**：
    - 此时 $q = 2$，故总有 $q\mid p-1$
    - 由上面结论，此时非交换群 $K$ 为 $\Z_p\rtimes \Z_2$，显然是二面体群

## 低阶群分类

- **思路**：
  - 找出正规西罗 $p$ 子群
  - 将群写成正规西罗子群的半直积，$G = N\rtimes H$
  - 计算 $\Aut N$，得出半直积的各种作用同态，写出相应群的运算法则
  - 讨论分类结果是否存在可同构合并的情况

### 8阶群

- **（命题6.3）$8$ 阶群**：八阶非交换群只有两种
  - 二面体群 $D_4$
  - 四元数群 $Q_8$
  <!-- - <font color='chartreuse'>根据 $2$ 阶元素的数量，易得 $D_4 \not\cong Q_8$</font> -->
- **阶的分解**：$8 = 2^3$
- **西罗子群**：
  - 由非交换性得 $G$ 中不可能只有 $2$ 阶元素
    - <font color='chartreuse'>否则 $abab = baba =  e$，即 $b^{-1} = aba = b$，即 $ba = ab$，从而是交换群</font>
  - 设 $G$ 包含 $4$ 阶元素 $a$，显然 $\lang a \rang$ 是正规西罗 $2$ 子群
- **运算法则**：
  - 易得 $G$ 是二元生成群，从而是半循环群
  - 由 $|G/\lang a \rang|=2$ 得必定有 $b^2\in \lang a \rang$
    - 若 $b^2 = a$，则 $bab^{-1} = b^2 = a$，平凡作用，即交换群 $\Z_8$
    - 若 $b^2 = a^3$，则 $b$ 的阶数至少为 $8$，矛盾
    - 若 $b^2 = a^2$，则 $G = \ip{ a,b\Bigm | a^4 = e，b^2 = a^2，bab^{-1} = a^{-1} }$，此时其为 $Q_8$
    - 若 $b^2 = e$，则 $G = \ip{ a,b\Bigm | a^4 = e，b^2 = e，bab^{-1} = a^{-1} }$，此时其为 $D_4$

### 12阶群

- **（命题6.4）$12$ 阶群**：非交换12阶群只有三种：
  - 二面体群 $D_6$
  - 交错群 $A_4$
  - 可裂半循环群 $T = \Z_3\rtimes_5 \Z_4$，$\begin{cases} |a| = 3 \\ |b|=4 \\ bab^{-1} = a^5 \end{cases}$
- **阶的分解**：$12 = 2^2\times 3$
- **西罗子群**：
  - 设 $P$ 是西罗 $3$ 子群，则 $|P| = 3$，$[G:P] = 4$
  - 设 $P$ 的左陪集平移作用同态 $\p:G\to S_4$，则其核为 $K\leq P$，即 $K = P$ 或 $\lang e \rang$
    - 若 $K = \lang e \rang$，则 $\p$ 是单同态，从而 $G$ 同构于 $S_4$ 的子群，易得只能是 $A_4$
    - 若 $K = P$，则 $P$ 是正规西罗子群，从而唯一，即 $G$ 只有两个 $3$ 阶元素
- **运算法则**：
    - 设 $c$ 是 $3$ 阶元素，其共轭类数量为 $[G:C_G(c)] = 1$ 或 $2$，故 $C_G(c)$ 是 $6$ 阶或 $12$ 阶群，故由柯西阶定理，存在 $2$ 阶元素 $d\in C_G(c)$，
    - 设 $a = cd$，则 $|a| = 6$
      - 由指数 $2$ 子群正规性，$\lang a \rang\lhd G$ 且 $|G/\lang a \rang| = 2$，从而存在 $b^2 \in \lang a \rang$
      - 若 $b^2 = a^2$ 或 $a^4$，则 $b$ 的阶数存在矛盾
      - 若 $b^2 = a$ 或 $a^5$，则 $|b| = 12$，此时其为 $\Z_{12}$
      - 若 $b^2 = e$，则 $G = \ip{ a,b\Bigm| a^6 = b^2 = e，bab^{-1} = a^{-1} }$，此时其为 $D_6$
      - 若 $b^2 = a^3$，则 $G = \ip{ a,b\Bigm| a^6 = e，b^2 = a^3，bab^{-1} = a^{-1} }$，此时其为 $T$


### 20阶群

- **20阶非交换群的分类**：
  - 二面体群 $D_{10}$
  - 可裂半循环群 $\Z_5\rtimes_2 \Z_4$
  - 可裂半循环群 $\Z_5\rtimes_4 \Z_4$
- **阶的分解**：$20 = 2^2\cdot 5$
- **西罗子群**：
  - 西罗 $5$ 子群阶数为 $5$，个数为 $(5k+1)\mid 4$，解得为 $1$
  - 西罗 $2$ 子群阶数为 $4$，个数为 $(2k+1)\mid 5$，解得为 $1$ 或 $5$
- **运算法则**：
  - 若情况为 $\begin{cases} \Syl(5) = \Z_5 = \lang a \rang \\ \Syl(2) = \Z_4 = \lang b \rang \end{cases}$，则此时 $G = \Z_5\rtimes \Z_4$
    - 设 $bab^{-1} = a^r$
    - 若 $r=1$，则是平凡作用，此时其为 $\Z_{20}$
    - 若 $r=3$，由于 $3^4\equiv 2^4\pmod 5$，故和下面情况同构
    - 若 $r=2$，则 $G = \ip{ a,b\Bigm| a^5 = b^4 = 1，bab^{-1} = a^2 }$
      - <font color='red'>此时 $G$ 为 $S_5\times \Z_4$ 中由 $a = (12345)\times 1，b = (2345)\times \a$ 生成的子群</font>
    - 若 $r=4$，则 $G = \ip{ a,b\Bigm| a^5 = b^4 = 1，bab^{-1} = a^4 }$
      - <font color='red'>此时 $G$ 为 $S_5\times \Z_4$ 中由 $a = (12345)\times 1，b = (12)(35)\times \a$ 生成的子群</font>
  - 若情况为 $\begin{cases} \Syl(5) = \Z_5 = \lang a \rang  \\ \Syl(2) = \Z_2\oplus \Z_2 = \lang b \rang\oplus \lang c \rang \end{cases}$，则 $G = \lang a ,b,c \rang$
    - 设 $bab^{-1} = a^r$，则 $r^2\equiv 1\pmod 5$，解得为 $1$ 或 $4$
    - 若 $bab^{-1} = a，cac^{-1} = a^4$
      - 设 $x = ab$，则 $\begin{cases} |x| = 10 \\ x^6 = a \\ x^5 = b\end{cases}$
      - 此时 $G\cong D_{10} = \lang x,c\mid x^{10} = c^2 = 1，cx = x^9c \rang = D_5\times \Z_2$
    - 若 $bab^{-1} = a^4，cac^{-1} = a$
      - 易得和上面等价
    - 若 $bab^{-1} = a^4，cac^{-1} = a^4$
      - 此时 $bca(bc)^{-1} = ba^4b^{-1} = (a^4)^4 = a$，故也和上面等价
