# 群列

## 思想

- 前面学习了半直积和群扩张，通过它可分解将低阶群分解成西罗子群的半直积，再通过讨论作用同态得到全部半直积结构
- 对于高阶群，阶数的分解会非常复杂，传统西罗方法效率降低。但我们可以用群扩张组成的正规子群列，将群层层分解为"单群的单群的单群……"结构，并归类为"可解程度不同的群"，然后再用手法分类所有的有限单群，就能用群扩张理论反向拼回所有群

## 大纲

- 本章只对群的可解程度进行简单归类

### 有限群的三大分解条件

- **三个条件**：体现有限群 $G$ 能分解的程度
  - **幂零群条件**：$G$ 是西罗子群的直积
  - **超可解群条件**：对 $\forall m\Bigm| |G|$，$G$ 都存在 $m$ 阶子群
  - **Schur-Zassenhaus条件**：若 $G = mn，\gcd(m,n) = 1$，则 $G$ 存在 $m$ 阶子群
- **关系**：$1\subset 2\subset 3$
  - **证明**：若 $G = \prod\limits^n_{i=1} P_i$，设 $|P_i| = p_i^{\a_i}$，则 $|G| = \prod\limits^n_{i=1} p_i^{\a_i}$
    - $(1)\to (2)$：第一西罗定理直得条件2
    - $(1)\to (3)$：西罗子群定义直得条件3
    - $(2)\to (3)$：废话
  - **本质**：差不多在习题里被扒干净了。另外，它其实类似于有限阿贝尔群的推导
- **例子**：
  - $p$ 群、有限阿贝尔群满足1
  - $S_3$ 满足2，不满足1
  - $A_4$ 满足3，不满足2

### 可解性比较

- **可分裂程度**：幂零群 $\subset$ 超可解群 $\subset$ 可解群
  - **商全在中心**：中心列收敛 $\LR$ 幂零群
  - **商全循环**：超可解群
  - **商全交换**：导群列收敛 $\LR$ 可解群

## 换位子

### 换位子

- **全不可换元素**：$ga\neq ag\quad(\forall g\neq a)$
- **换位子（交换子）**：$[a,b] = (ab)(ba)^{-1} =  aba^{-1}b^{-1}$
  - <font color='red'>不可交换元素的一个配对</font>
  - **性质**：
    - **换位性**：$ab = [a,b]ba$
    - **逆封闭性**：$[a,b]^{-1} = [b,a]$
    - **幂封闭条件**：若 $[a,b]$ 和 $a,b$ 均可交换，则 $[a^m,b^n] = [a,b]^{mn}$
      - **证明**：由 $ab = [a,b]ba$ 得 $[a,b^n] = [a,b]^n$，再对 $m+n$ 用归纳法证明即可
    - **展开公式**： $[ab,c] = a[b,c]a^{-1}[a,c]$
    - **无乘积封闭性**：
      - **反例**：$[a,b][b,c] = aba^{-1}cb^{-1}c^{-1}$
- **换位子群（导群）**：群 $G$ 中全体换位子生成的群 $G'$ 或 $[G,G]$
  - <font color='red'>削去了 $G$ 生成元系中的（可换元素乘积 $ab = ba$）部分</font>
  - **应用**：$|G'|$ 的大小表示 $G$ 不可交换的程度<a id="换位子群本质"></a>
    - <font color='chartreuse'>$a,b$ 可换 $\Leftrightarrow$ $a$ 是 $b$ 的共轭不动点 $\Leftrightarrow$ 换位子 $[a,b] = e$
      - 小推论：若 $a,b$ 可换，则它们的逆也可换
    - 换位子 $[a,b]$ 与 $a$ 或 $b$ 可换 $\Leftrightarrow$ $a$ 是 $b^{-1}ab$ 或 $b$ 是 $a^{-1}ba$ 的共轭不动点
      - 也即 $a$ 与 $b^{-1}ab$ 可换
    - 也就是说，换位子不一定是全不可换元素，但不断在换位子群中再取换位子群，最终不变的子群一定是全不可换元素生成的（但未必是所有全不可换元素，参考 $S_n$）
    - 换位子 $[a,b] = [a,c] \Leftrightarrow a$ 是 $c^{-1}b$ 的共轭不动点
      - 也即 $a$ 与 $c^{-1}b$ 可换
    - 换位子 $[b,a]$ 与 $[a,c]$ 可换 $\Leftrightarrow [a^{-1},b]$ 和 $[b,a]$ 关于 $cac^{-1}$ 共轭等价（没用）</font>
  - **性质**：
    - **正规性**：$G'\lhd G$
      - **证明**：
        - 设 $f\in \Aut G$，则元素分析易得 $f(G') < G'$
        - 取 $f$ 是共轭，即得 $G'\lhd G$
    - **商群可换性**：$G/G'$ 是阿贝尔群
      - **证明**：
        - 由于 $(ab)(ba)^{-1} \in G'$，故 $abG' = baG'$，故 $G/G'$ 是阿贝尔群
    - <font color='red'>换位子群以外的都是次可交换元素，从而 $G'$ 在 $G-G'$ 眼中是可整体交换的</font>
- **导子群**：群 $G$ 的两个子群之间的换位子生成的群 $[G,X] = \ip{[g,x]\Bigm | g\in G,x\in X}$

### 导群的本质

- **交换邻接矩阵**：假设 $G$ 有 $n$ 个元素，其中第 $i,j$ 个元素是否可交换记为 $\d_{ij}$，则构成 $n\times n$ 对称矩阵（对角元均为1）
  - **指示性**：非零元数量就是换位子数量。它可以指示导群的大小
- **（定理7.8）导群定理**：若 $N\lhd G$，则 $G/N$ 是阿贝尔群 $\LR G'\leq N$
  - **证明**：
    - **必要性**：上述过程反过来即得 $G'<N$
    - **充分性**：易得
  - **本质**：<font color='red'>通过换位子群来实现群的极大交换化，是提出换位子的核心动机</font>
  - **实例**：$S_3/A_3 \cong \set{(1),(12)}$
    - <font color='chartreuse'>由于置换只与自身可换，故 $S_n$ 的换位子群只能是 $n/2$ 阶的，也即 $A_n$
    - 还可得到一个推论：$S_n$ 不可解</font>
  - **推论**：设 $f$ 是 $G$ 的同态，则 $\Im f$ 是阿贝尔群 $\LR G'\subset\ker f$
