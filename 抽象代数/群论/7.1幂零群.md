# 幂零群

## 有限群的分解

- **三个条件**：体现有限群 $G$ 能分解的程度
  - **西罗直积条件**：$G$ 是西罗子群的直积
  - **西罗分解条件**：对 $\forall m\Bigm| |G|$，$G$ 都存在 $m$ 阶子群
  - **西罗存在条件**：若 $G = mn\pad ((m,n) = 1)$，则 $G$ 存在 $m$ 阶子群
- **关系**：$1\subset 2\subset 3$
  - **证明**：若 $G = \prod\limits^n_{i=1} P_i$，设 $|P_i| = p_i^{\a_i}$，则 $|G| = \prod\limits^n_{i=1} p_i^{\a_i}$
    - $(1)\to (2)$：第一西罗定理直得条件2
    - $(1)\to (3)$：西罗子群定义直得条件3
    - $(2)\to (3)$：废话
  - **本质**：差不多在习题里被扒干净了。另外，它其实类似于有限阿贝尔群的推导
- **例子**：
  - $p$ 群、有限阿贝尔群满足1
  - $S_3$ 满足2，不满足1
  - $A_4$ 满足3，不满足2
  

## 幂零群

- **次中心**：设 $f_2:G\to G/C(G)$ 是同态，则 $C_2(G) = f_2^{-1}\Big(C(f_2(G))\Big)$ 称为次中心
  - **保正规性**：$C_2(G)\lhd G$
    - **证明**：由同态下正规性不变直得
  - **延拓性**：$C_2(G) \geq C(G)$
    - **证明**：
      - 易得 $g\in C_2(G) \LR gChC = ghC = hgC = hCgC$
      - 易得 $C$ 中元素符合条件，而 $G/C$ 中的其它可交换元素也符合条件，故只增不减
    - **理解**：$C_2(G)$ 和 $G$ 在商合并作用后是中心关系，$C(G)$ 和 $G$ 在原像是中心关系
    - **本质**：商映射保中心性 + 逆映射扩大性
  - **交换性**：$C_2(G)$ 是阿贝尔群
    - **证明**：只需证明中心以外的部分交换
      - 由定义，$gC$ 是 $G/C$ 的中心，故 $(g_1C)(g_2C) = (g_2C)(g_1C)$
      - 再由中心交换性，$(g_1C)(g_2C) = g_1g_2C = (g_2C)(g_1C) = g_2g_1C$，取 $e\in C$ 即得结论
    - **推论**：$C_{i+1}/C_{i}$ 是阿贝尔群

### 幂零群

- **幂零群**：$\exists n$ 使得 $C_n(G) = G$
  - **本质**：
  - **实例**：阿贝尔群均为幂零群：$G = C(G) = C_1(G) = \cdots$
  - **反例**：中心平凡的群均为非幂零群：$C(S_n) = \lang e \rang$
- **（定理7.2）$p$ 群幂零性**：有限 $p$ 群是幂零群
  - **证明**：
    - 由商群传递性，此时非平凡商群均为 $p$ 群，再由 $p$ 群中心非平凡性得总存在非平凡中心
    - 再由第一西罗定理，可设 $|G| = p^n，C(G) = p^k$，则 $|G/C(G)| = p^{n-k}$，同理可设 $|C\Big(G/C(G)\Big)| = p^{n-k-m}$，即 $|C_2(G)| = p^{n-m}$，从而可不断归纳
      - 也就是说，若 $G \neq C_i(G)$，则 $C_i(G) \gvertneqq C_{i+1}(G)$。再由 $G$ 有限即得结论
  - **本质**：只要保持性质不变，有限群总能被有限次剔除掉所有次交换元素，从而为幂零群
- **（定理7.3）有限直积传递性**：有限个幂零群的直积幂零
  - **证明**：只需证明2阶情况即可
    - 设 $G = H\times K$，则易得 $C_i(G) = C_i(H)\times C_i(K)$
    - 设 $\pi_H,\pi_K$ 是 $H\to H/C_i(H)$
      - 则满同态 $\p:G\to G/C_i(G)$ 具有以下分解：
      - $G\xto{\pi_H\times \pi_K} H/C_i\times K/C_i \xto{\psi} \cfrac{H\times K}{C_i(H)\times C_i(K)} = G/C_i(G)$
      - 从而 $C_{i+1}(G) = C_{i+1}(H)\times C_{i+1}(K)$
    - 易得幂零性可通过此式传递
  - **本质**：映射乘积性 + 商群直积互逆性
- **（引理7.4）幂零群的正规化子条件**：幂零群 $G$ 的真子群 $H \lvertneqq N_G(H)$
  - **证明**：
    - 若 $H = C_i$，则 $N_G(H) = G$，结论显然。否则由幂零性可得存在 $n$ 使得 $C_n < H < C_{n+1}$
    - 可设 $a\in C_{n+1}，a\notin H$
      - 任取 $h\in H$
      - 由于 $C_na$ 是 $G/C_{n}$ 的中心，得 $C_nah = hC_na$
      - 再由限制正规性，$C_n\lhd H$，故 $hC_na = C_nha$
    - 因此 $\exists h'\in C_n$ 使得 $ah = h'ha$，从而 $aha^{-1} = h'h\in H$，即 $a\in N_G(H)$
      - 再由 $a\notin H$，即得 $H$ 为真子群
  - **理解**：由幂零性，存在 $C_n < H < C_{n+1}$，而 $C_{n+1}\leq N_G(H)$
  - **本质**：幂零群中，所有元素都具有一定程度的交换性。若子群 $H$ 最多存在 $n+1$ 次可交换元素，则其显然在 $n+1$ 次中心 $C_{n+1}$ 中正规，从而正规化子必定包含 $C_{n+1}$
    - <font color='chartreuse'>当 $G$ 有限时，它被强化为[wieland定理](w定理)</font>
- **（命题7.5）西罗直积性**：有限群 $G$ 是幂零群 $\LR G$ 满足条件1
  - **充分性**：已知有限西罗 $p$ 群是幂零群，且幂零性可有限直积传递，故结论易得
  - **必要性**：设 $P$ 是西罗 $p$ 子群
    <!-- - 若 $P\lvertneqq G$，则由幂零正规化子引理，$P\lvertneqq N_G(P)$ -->
    - 若 $C_n \lvertneqq P\lvertneqq C_{n+1}$，则 $G = N_G(C_n) \leq N_G \Big( N_G(P) \Big) = N_G(P) \leq G$
      - 即 $N_G(P) = G$，即 $P$ 是正规子群
      - 再由第二西罗定理，$P$ 也是唯一的西罗子群。
    - 若 $P = C_i$，则显然 $N_G(P) = G$，也可得 $P$ 唯一性
    - 设 $|G| = \prod\limits^k_{i=1} p_i^{n_i}$，则可设 $P_i$ 是对应的西罗 $p_i$ 真子群，易得其不相交，从而积为直积
  - **理解**：[西罗正规不变性](5.有限群的结构.md)由轨道唯一性反推出，西罗正规性由次中心夹逼推出，西罗唯一性由共轭封闭性推出，从而有限幂零群的每个素因子 $p_i$ 对应的西罗 $p_i$ 子群均唯一，即可直积分解
  - **本质**：夹逼法，两边都正规，则中间的 $P$ 也正规
    - 某种意义上讲，直积和交换性密切相关。把群割成独立的小块，每部分越小，越可能交换，从而每部分均可交换（整体可交换）的概率越大
    - 而反过来的结论我们早就得到了，即越交换的群越可能是直积（有限阿贝尔群均可直积分解成交换群）
      - 幂零群是次一级的交换性，其只能分解成西罗子群的直积
      - 而再次一级，没有交换性，只要满足正规链条件，也能分解成不可分群的直积
- **（推论7.6）西罗存在性**：有限幂零群满足条件3
  - **证明**：易得

## 习题

### 传递性

- **子群传递性**：幂零群的子群幂零
  - **证明**：设 $K<G$，只需 $C_i(K) =  K\cap C_i(G)$ 即可
    - 易得 $C(K) = K\cap C(G)$，我们将 $K$ 分为 $\begin{cases} K_1 = C_1 \\ K_2 = C_2-C_1 \\ K_3 = K-C_2 \end{cases}$ 三个部分
    - 易得 $G_2\cap K = K_2$，<font color='chartreuse'>$G_3\cap K = K_3$
    - 再易得 $K\cap \Big(G/C(G)\Big) = (K_2\cup K_3)C(K)$
    - 再由 $C_2(G) = f_2^{-1}(C(G/C_1))$，</font>即得 $C_2(K) = K\cap C_2(G)$
    - 再将 $K_3$ 分为三个部分，归纳即可
  - **理解**：看着乱，核心思想就是取限制，将性质传递，然后归纳着做。下面的动机远远比上面的手段重要
  - **本质**：幂零性本质上已经将 $G$ 通过次交换性分解，若再从里面取限制 $K$，显然只需要对每个分解取限制即可
    - 若 $K$ 最高只有 $n-k$ 级次交换元素，则最低 $C_{n-k}(K) = K$。幂零性比 $G$ 更强
- **同态传递性**：幂零群的同态像幂零
  - **证明**：设 $F:G\to H$ 是同态
    - 中心同态传递性得 $F(C_1(G)) = C_1(H)$
    - 商群同态传递性得 $F(G/C_1(G)) = H/C_1(H)$
    - 中心同态传递性得 $F(C(G/C_1(G))) = C(H/C_1(H))$
    - 乘积同态传递性得 $F(C_2(G)) = C_2(H)$
      - <font color='chartreuse'>反商群传递性的本质就是乘积传递性
      - 即 $\pi^{-1}(gC_1) = \set{gx\mid g\in L_{C(G)}，x\in C(G)}$
      - 从而 $F(\pi^{-1}(C(G/C_1(G)))) = \set{F(g)x\mid x\in C(H)} = C_2(H)$</font>
    - 归纳即可
  - **本质**：同态能传递充分性质，但不能传递等价性质。只有同构才能传递所有性质
    - 比如中心和商就是充分性质，是由 $G$ 的全局得到的局部性质，而不是由局部反推出全局性质
- **商群传递性**：幂零群的商群幂零
  - **证明**：商映射是同态，或商群同构于商代表元系（G的子群），都可得到结论
  - <font color='chartreuse'>错啦！这样岂不是（子群传递性）可直接导出（商群传递性）了吗，但这显然不可能。实际上，同构并不能传递所有性质</font>
- **正规逆传递条件**：设 $N\leq C_1$ 幂零，若 $G/N$ 幂零，则 $G$ 幂零
  - **证明**：易得 $C_n(G/N) = C_n(G)/N$
    - 再由商群幂零性得 $C_n(G/N) = G/N$，综合即得结论
  - **反例**：若改为和可解群一样的 $N\lhd G$，则结论不成立
    - 如 $A_3\lhd S_3$，但非幂零

### 其它

- **阶与可换性**：有限幂零群中，若 $a^m = b^n = 1，(m,n) = 1$，则 $a,b$ 可换
  - **证明**：已知幂零群可分解为西罗子群直积
    - 易得此时 $(|a|,|b|) = 1$，故 $a,b$ 属于两个不同的西罗子群，从而独立，从而可换
- **分解视角下的正规子群**：有限幂零群 $G$ 中，任意非平凡正规子群 $N$ 满足 $N\cap C_1\neq \lang e \rang$
  - **证明**：由有限幂零性，可设 $C_n = G$，即 $N\cap C_n = N$
    - $\forall n\in N$，其共轭类 $\ol n\in C_n/C_{n-1} = C(G/C_{n-1})$，从而 $\forall g\in G，[n,g]\in N\cap C_{n-1}$
    - 反设 $N\cap C_{n-1} = \{e\}$，则 $n\in C(G)$
      - 由 $n$ 任意性即得 $N\leq C(G)$，从而 $N\cap C_{n-1} = N \neq \lang e \rang$
    - 反复进行即得结论
  - **推论**：设有限非交换幂零群 $G$ 中，$A$ 是最大正规子群。若 $A$ 是阿贝尔群，则 $C_G(A) = A$<a id="分解视角下的正规子群"></a>
    - **证明**：由 $A$ 在自身中可交换，得 $A\leq C_G(A)$
      - 要证 $C_G(A)\leq A$，首先证明其正规性
      - 任取 $g\in G，b\in C_G(A)，a\in A$
        - 由 $A$ 正规性，$g^{-1}ag\in A$，从而与 $b$ 可换
        - 则 $gbg^{-1}x = xgbg^{-1}$，从而 $b$ 在 $g$ 共轭下不变，由任意性即得 $C_G(A)\lhd G$
      - 由 $A$ 最大性，此时 $C_G(A) = G$ 或 $A$
        - 反设为 $G$，则 $A\leq C(G)$，由中心正规性，只能是 $C(G) = A$ 或 $G$。由 $G$ 非交换性得只能是 $A = C(G)$
        - 再由幂零性，$C_1<C_2$。再由 $C_2$ 正规性即得只能是 $C_2 = G$
      - 此时 $G/A = C_2/C_1$ 是阿贝尔群，由 $A$ 极大性得无非平凡子群，只能是素数阶循环群。但此时 $G = A\times Z_p$，是阿贝尔群，矛盾
- **Wieland定理**：有限群 $G$ 幂零 $\LR$ 最大真子群均正规<a id = "w定理"></a>
  - **证明**：
    - **必要性**：有限幂零性得 $G$ 是西罗子群的直积，故最大真子群均是西罗子群，直积因子当然是正规的
    - **充分性**：
    - 由[正规不变性](5.有限群的结构)的本质，若 $P$ 是 $G$ 的西罗 $p$ 子群，则 $N_G(P)$ 中每个子群的正规化子都是自身
  - **本质**：有限幂零群的最大真子群均为西罗子群（显然）
