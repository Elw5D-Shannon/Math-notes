# 一般的坐标系

- **预备知识**：
  - 至少看完古典微分几何的活动标架部分
  - 也可以看看抽象代数中模的张量积部分
- **符号约定**：
  - $[\bs g_1,\bs g_2,\bs g_2]$ 表示向量混合积
  - 采用爱因斯坦求和约定
    - $\d^\b_\a$ 是Kronecker记号
    - $[g^{ij}],[g_{ij}]$ 都可表示矩阵
    - 上标表示逆变，下标表示协变

## 二维斜角直线坐标系

- **平面斜角坐标系**：设 $x^1,x^2$ 是平面内两条夹角为 $\p$ 的直线，基矢为 $\bs g_1,\bs g_2$，单位矢量为 $\bs i_1,\bs i_2$
  - <font color='red'>在实际物理问题中，基矢不取单位向量会更方便
    - 比如计算晶格时，基矢取相邻原子的位矢
    - 存在仿射变换时，会出现拉伸，故基矢不可能总是单位向量</font>
- **矢量分量**：$P^1,P^2$ 表示矢量 $\bs P$ 的两个分量
- **矢量分解公式**：$\bs P = P^\a\bs g_\a$
  - <font color='red'>因为不是标准正交基，故矢量分解不是基矢投影</font>
- **矢量投影公式**：$$ \begin{cases}  \bs P\cdot \bs i_1 = P^1|\bs g_1| + \bs P^2|\bs g_2|\cos\p \\\\  \bs P\cdot \bs i_2 = P^1|\bs g_1|\cos\p + \bs P^2|\bs g_2| \end{cases} $$
  - <font color='red'>如果想要用 $\bs P\cdot\bs i_\a$ 表示 $P^\a$，则需要解线性方程组，太麻烦
  - 为此提出协变和逆变的概念</font>


### 协变和逆变

- **协变和逆变**：
  - 取 $\bs g^1,\bs g^2$ 使得 $\bs g^\b\bs g_\a = \d^\b_\a$，即 $$\begin{cases} \bs g^1\cdot\bs g_2 = \bs g^2\cdot\bs g_1 = 0 & 正交性 \\\\ \bs g^1\cdot\bs g_1 = \bs g^2\cdot\bs g_2  = 1 & 单位性 \end{cases}$$
  - 则 $\bs g_1,\bs g_2$ 称为**协变基矢**，$\bs g^1,\bs g^2$ 称为**逆变基矢**
  - <font color='red'>以后下标是协变指标，上标是逆变指标</font>
- **长度转换公式**：$|\bs g^\a| = \cfrac{1}{|\bs g_\a|\sin\p}$
  - **证明**：定义易得
- **逆变分量**：$P^\a = \bs P\cdot \bs g^\a$
  - <font color='red'>$P^\a$ 是 $P$ 关于协变基矢的分量，但它却叫做逆变分量。因为可被逆变基矢简单表出</font>
- **协变分量**：$P_\a = \bs P\cdot \bs g_\a$
  - <font color='red'>对偶地，协变分量 $P_\a$ 实际是 $P$ 关于逆变基矢的分量，但可被协变基矢简单表出</font>

### 坐标变换

- **关系不变定理**：线性变换下，对偶关系 $\bs g^\b\bs g_\a = \d^\b_\a$ 始终不变
  - **证明**：定义易得
- **协变定理**：设 $A$ 是坐标变换的矩阵，则 $\begin{cases} \bs g'_i = \sum\limits_{j} A^j_i \bs g_j & 协变基矢 \\\\ P'^j = \sum\limits_j \dkh{A^{-1}}^i_j P^j & 逆变分量 \end{cases}$
  - <font color='red'>坐标变换时，$\bs P$ 不变，即分量不变
  - 再由分解公式，协变基矢被拉长时，逆变分量就得同步变小
  - 写成数学形式就是逆矩阵</font>
- **逆变定理**：设 $A$ 是坐标变换的矩阵，则 $\begin{cases} \bs g'^i = \sum\limits_{j} \dkh{A^{-1}}^j_i \bs g_j & 逆变基矢 \\\\ P'^j = \sum\limits_j A^i_j P^j & 协变分量 \end{cases}$


## 三维斜角直线坐标系

- **空间斜角直线坐标系**：设 $x^1,x^2,x^3$ 是空间内三条相交直线
  - <font color='red'>协变基矢和逆变基矢定义同上</font>

### 协变和逆变

- **长度转换公式**：设 $\bs g^i$ 与 $\bs g_i$ 夹角为 $\p$，则 $|\bs g^i| = \cfrac{1}{|\bs g_j|\cos\p}$
- **逆变梯度公式**：$\bs g^i = \grad x^i = \nabla x^i$
  - **证明**：由协变基矢是位矢偏导数易得结论
- **外积计算公式**：设混合积 $[\bs g_1,\bs g_2,\bs g_3] = g$，则 $$ \begin{cases} \bs g^1 = \cfrac{1}{\sqrt{g}}(\bs g_2\times \bs g_3) \\\\ \bs g^2 = \cfrac{1}{\sqrt{g}}(\bs g_3\times \bs g_1) \\\\ \bs g^3 = \cfrac{1}{\sqrt{g}}(\bs g_1\times \bs g_2) \end{cases} $$
- **矩阵计算公式**：
  - 设协变基分解为 $\bs g^i = g^{ij}\bs g_j$，则 $\bs g^i\cdot \bs g^j = g^{ij}$
  - 设逆变基分解为 $\bs g_i = g_{ij}\bs g^j$，则 $\bs g_i\cdot \bs g_j = g_{ij}$
  - 矩阵 $[g^{ij}]$ 和 $[g_{ij}]^{-1}$ 互逆，且都是对称矩阵
- **指标升降公式**：
  - 设 $\bs P$ 是矢量
  - **矢量分解**：$\bs P = P^i\bs g_i = P_j\bs g^j$
  - **矢量投影**：$$\begin{cases} P^i = \bs P\cdot \bs g^i = P_k\bs g^k\cdot \bs g^i = P_k\bs g^{ki} \\\\ P_j = \bs P\cdot \bs g_j = P^k\bs g_k\cdot \bs g_j = P^k\bs g_{kj} \end{cases}$$


## 曲线坐标系

- **曲线坐标系**：设 $\bs r$ 是位矢，则三维向量值函数 $\bs r(x^i,x^j,x^k)$ 构成一个曲线坐标系
  - <font color='red'>仅一个坐标 $x^i$ 不变时，图像是曲面
  - 有两个坐标 $x^i,x^j$ 不变时，图像是曲线</font>
- **实例**：球坐标系 $x^1 = r，x^2 = \t，x^3 = \p$ $$\bs r = x^1\sin x^2\cos x^3\bs i + \bs r = x^1\sin x^2\sin x^3\bs i + \bs r = x^1\cos x^2\bs k$$


### 局部基矢

- **协变基矢**：设 $d\bs r = \bs g_idx^i$，则 $\bs g_i  = \cpfrac{\bs r}{x^i}$ 称为协变基矢
  - <font color='red'>此时位矢和曲线坐标不是线性关系，故协变基矢不断变化，从而也称为局部基</font>
- **切标架**：协变基矢构成的活动标架
- **逆变基矢**：若 $\bs g^i$ 与协变基矢 $\bs g_i$ 满足对偶关系 $\bs g^j\cdot \bs g_i = \d^j_i$，则称为逆变基矢


### 正交曲线坐标系

- **正交曲线坐标系**：三个坐标线处处正交的曲线坐标系
  - **度量矩阵**：对角矩阵，其中对角元称为拉梅常数
- **拉梅常数**：$A_1 = \sqrt{g_{11}}，A_2 = \sqrt{g_{22}}，A_3 = \sqrt{g_{33}}$
